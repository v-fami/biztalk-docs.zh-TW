# <a name="about-metadata-search-and-browse-with-your-wcf-lob-adapter-sdk-adapter"></a>關於中繼資料搜尋和瀏覽 WCF LOB 配接器 SDK 介面卡
不同於使用建置靜態服務[!INCLUDE[nextref_btsWinCommFoundation](../../includes/nextref-btswincommfoundation-md.md)]，使用所建置的配接器[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]可以提供的型別和作業的特定業務系統所提供的動態資訊。 開發人員可支援下列作業：  
  
- **中繼資料瀏覽**，讓使用者檢閱所有作業的清單，以及類型的特定業務系統所提供。  
  
- **中繼資料搜尋**，讓使用者搜尋作業使用不同的準則，通常是根據商務系統的一行。 比方說，使用者可以搜尋"* CUST\*」 來擷取其名稱中的任何地方含有"CUST"的所有作業。  
  
- **中繼資料擷取**，提供支援的作業或資料類型的特定清單的相關資訊。  
  
  如果商務系統的一行包含數百或數千個作業，以便能夠針對特定作業搜尋或瀏覽特定群組的資訊可以增強使用者體驗。
