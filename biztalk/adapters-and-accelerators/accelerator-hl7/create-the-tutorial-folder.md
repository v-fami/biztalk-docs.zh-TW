# <a name="create-the-tutorial-folder"></a>建立教學課程的資料夾
建立下列資料夾，請使用[!INCLUDE[btsWinNoVersion](../../includes/btswinnoversion-md.md)]總管：  
  
- \<*磁碟機*:\>\Program Files\Microsoft BizTalk\<版本\>Accelerator for HL7\SDK\Interrogative 教學課程  
  
   您可以使用此資料夾來儲存金鑰、 查詢、 回應和通知檔案供您在此案例中。  
  
  請繼續進行[建立 QRY^Q01.txt 檔案](../../adapters-and-accelerators/accelerator-hl7/create-the-qry-q01-txt-file.md)。
