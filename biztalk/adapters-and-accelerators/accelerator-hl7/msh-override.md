# <a name="msh-override"></a>MSH 覆寫
之前[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Accelerator for HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 將輸出 HL7 訊息路由至傳送埠，您可能想要取代 MSH 區段中的某些值。 若要這樣做，當訊息的訂用帳戶包含多個傳送埠，而且每個傳送埠應該會看到接收應用程式欄位不同的值。
