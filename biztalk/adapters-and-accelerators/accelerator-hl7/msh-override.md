# <a name="msh-override"></a>MSH 覆寫
Microsoft BizTalk Accelerator for HL7 之前 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 將輸出 HL7 訊息路由至傳送埠，您可能想要取代 MSH 區段中的某些值。 若要這樣做，當訊息的訂用帳戶包含多個傳送埠，以及每個傳送埠應該會看到不同的值，在接收應用程式 欄位。
