# <a name="a-role-based-product"></a>以角色為基礎的產品
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Server 和[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Accelerator for HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 提供的功能和使用者角色所適合之環境中的工具，如下列所述：  
  
-   開發人員工具已整合至[!INCLUDE[btsVStudio2008](../../includes/btsvstudio2008-md.md)]。 開發人員可以管理自己的解決方案和專案，在 方案總管，並可以立即存取 BizTalk 設計階段工具。  
  
-   [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] Management Console (MMC) 裝載 IT 專業人員的工具。 介面分析師、 IT 實作者以及 IT 操作員管理其[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]透過 MMC 架構的 BizTalk 管理主控台，BizTalk 總管 中，系統[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]組態總管 中，與其他執行的階段工具。  
  
-   商務分析師及其他資訊工作人員透過本身的一組 [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] 工具來分析商務狀態並存取商務資料。
