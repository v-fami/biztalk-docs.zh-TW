# <a name="party-aliases-tab"></a>合作對象別名 索引標籤
您使用**合作對象別名**索引標籤來檢視選取的合作對象的相關資訊。  
  
> [!NOTE]
>  您可以使用 BizTalk 總管 中建立合作對象。  
  
 如需有關如何建立合作對象的詳細資訊，請參閱 「 管理合作對象使用 BizTalk 總管 」 中[!INCLUDE[btsBizTalkServer2006r3](../../includes/btsbiztalkserver2006r3-md.md)]幫助。  
  
 在**BTAHL7 Configuration 總管**對話方塊**合作對象別名**索引標籤上，執行下列動作：  
  
|使用|動作|  
|--------------|----------------|  
|**名稱**|檢視合作對象公司的名稱。|  
|**值**|檢視合作對象識別碼值。|
