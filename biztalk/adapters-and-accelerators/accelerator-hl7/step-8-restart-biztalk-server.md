# <a name="step-8-restart-biztalk-server"></a>步驟 8： 重新啟動 BizTalk Server
在此步驟中，您重新啟動[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]為了讓您在本教學課程做才會生效的變更。  
  
### <a name="to-restart-biztalk-server"></a>若要重新啟動 BizTalk Server  
  
1.  啟動**BizTalk Server 管理**。  
  
2.  在[!INCLUDE[btsCoName](../../includes/btsconame-md.md)][!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]系統管理主控台中，展開**BizTalk Server 管理**，依序展開**BizTalk 群組**，然後展開**平台設定**. 按一下**主機**，然後按一下  **BizTalkServerApplication**。  
  
3.  您的伺服器，以滑鼠右鍵按一下，然後按一下 **重新啟動**。  
  
 若要繼續[步驟 9： 驗證分散的傳入批次案例](../../adapters-and-accelerators/accelerator-hl7/step-9-verify-the-fragmented-inbound-batch-scenario.md)。
