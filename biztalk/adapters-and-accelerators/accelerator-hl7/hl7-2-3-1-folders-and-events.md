# <a name="hl7-231-folders-and-events"></a>HL7 2.3.1 資料夾和事件
下表列出 HL7 編碼訊息的 HL7 版本 2.3.1 資料夾內的安裝程式精靈所建立的子資料夾。 這些子資料夾包含使用的結構描述[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Accelerator for HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 來驗證、 剖析和序列化此資料表的事件資料行中列出的事件。 子資料夾名稱會描述這些結構描述支援的事件類型。  
  
|子資料夾|事件|  
|---------------|------------|  
|病患的系統管理|A01 A51|  
|觀察報告|C01 C12，P06 P09，R01 R06，W01 W02|  
|病患的轉介|I01 I15|  
|主版檔案|M01 M11 MFA|  
|訂單項目|O01 O02、 Q06、 R0R、 RAR、 RDR、 RER、 RGR、 V01 V04|  
|財務管理|P01 P06|  
|病患照護|PC1-PCJ PCL PCH|  
|查詢|CNQ，Q01 Q05，Q07 Q09，R07 R09|  
|排程|S01 S26|  
|醫療的記錄資訊管理|T01 T12|
