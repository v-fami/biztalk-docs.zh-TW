# <a name="using-hl7-2x-schemas"></a>使用 HL7 2.X 結構描述
本章節將討論 2.X 版所支援的健全狀況層級七 (HL7) 標準[!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[HL7_CurrentVersion_FirstRef](../../includes/hl7-currentversion-firstref-md.md)]。  
  
> [!IMPORTANT]
>  您可能需要更新安裝 BTAHL7 以符合標準 HL7 結構描述。 若要這樣做，請參閱[解決資料庫錯誤](../../adapters-and-accelerators/accelerator-hl7/resolving-database-errors.md)。  
  
> [!NOTE]
>  BTAHL7 引擎無法處理具有模稜兩可的結構描述結構的 HL7 結構描述不符合的訊息執行個體。 模稜兩可的結構描述結構是一種標準 HL7 未完全定義。 這類結構描述包含的訊息類型 CSU 以及 SUR.
