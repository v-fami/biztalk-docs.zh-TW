# <a name="about-business-data-catalog-with-the-siebel-adapter"></a>有關 Siebel 介面卡的商務資料目錄
商務資料目錄是在 SharePoint 中的整合元件。 商務資料目錄是共用的服務和後端應用程式透過資料庫和 Web 服務，而不需要撰寫任何程式碼從表面商務資料。 它是定義商務實體，例如客戶、 發票和訂單的中繼資料儲存機制。 定義好之後，用戶端可以使用這些實體。  
  
 商務資料目錄提供商務資料目錄定義編輯器來協助您撰寫的商務資料目錄，在 SharePoint 中的應用程式定義檔。 您可以使用商務資料目錄定義編輯器，建立及編輯商業應用程式的中繼資料。
