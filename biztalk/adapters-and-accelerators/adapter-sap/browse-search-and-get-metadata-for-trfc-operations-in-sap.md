# <a name="browse-search-and-get-metadata-for-trfc-operations-in-sap"></a>瀏覽、 搜尋和 tRFC 作業 SAP 中取得中繼資料
tRFCs 並不是個別成品的 SAP 系統。 這些分類的不同節點所下[!INCLUDE[adaptersap_short](../../includes/adaptersap-short-md.md)]因為它們的中繼資料的特性是不同的 Rfc。 不過，瀏覽的經驗，搜尋和擷取中繼資料 tRFCs 是相同的 Rfc。 請參閱[瀏覽、 搜尋和 sap RFC 作業的 get 中繼資料](../../adapters-and-accelerators/adapter-sap/browse-search-and-get-metadata-for-rfc-operations-in-sap.md)有關使用[!INCLUDE[addadapterservrefshort](../../includes/addadapterservrefshort-md.md)]或[!INCLUDE[consumeadapterservshort](../../includes/consumeadapterservshort-md.md)]瀏覽，搜尋，然後從 SAP 系統擷取 tRFCs 中繼資料。  
  
 請注意，[!INCLUDE[adaptersap_short](../../includes/adaptersap-short-md.md)]呈現特殊作業**RfcConfirmTransID**下**TRFC**節點。 SAP 配接器會認可對 SAP 伺服器的 tRFC 呼叫使用這項作業。 如需有關這項作業的詳細資訊，請參閱[tRFCs SAP 中的作業](../../adapters-and-accelerators/adapter-sap/operations-on-trfcs-in-sap.md)。
