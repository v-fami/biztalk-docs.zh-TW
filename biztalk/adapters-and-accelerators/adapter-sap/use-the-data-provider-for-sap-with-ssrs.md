# <a name="use-the-data-provider-for-sap-with-ssrs"></a>使用適用於 SAP ssrs 資料提供者
您可以使用[!INCLUDE[adoprovidersaplong](../../includes/adoprovidersaplong-md.md)]以及 SQL Server Reporting Services (SSRS) 建立的 SAP 資料為基礎的報表。 如需 SSRS 的詳細資訊，請參閱[http://go.microsoft.com/fwlink/?LinkId=125922](http://go.microsoft.com/fwlink/?LinkId=125922)。  
  
 若要建立的 SAP 資料為基礎的報表，您必須使用[!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)]建立報表伺服器專案，並使用[!INCLUDE[adoprovidersapshort](../../includes/adoprovidersapshort-md.md)]從 SAP 系統擷取資料。  
  
 本節中的主題提供有關如何建立報表伺服器專案來產生報表根據 SAP 資料的資訊。
