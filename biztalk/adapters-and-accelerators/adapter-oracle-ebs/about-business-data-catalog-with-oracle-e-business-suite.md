# <a name="about-business-data-catalog-with-oracle-e-business-suite"></a>關於 Oracle E-business Suite 與商務資料目錄
商務資料目錄是一個新的整合元件在 Microsoft Office SharePoint Server 2007 和更新版本。 商務資料目錄是共用的服務和後端應用程式透過資料庫和 Web 服務，而不需要撰寫任何程式碼從表面商務資料。 它是定義商務實體，例如客戶、 發票和訂單的中繼資料儲存機制。 定義好之後，用戶端可以使用這些實體。  
  
 商務資料目錄提供商務資料目錄定義編輯器來協助您撰寫的商務資料目錄，在 Microsoft Office SharePoint Server 2007 和更新版本的應用程式定義檔。 您可以使用商務資料目錄定義編輯器，建立及編輯商業應用程式的中繼資料。  
  
 商務資料目錄的詳細資訊，請參閱 「 商務資料目錄 」，網址[http://go.microsoft.com/fwlink/?LinkId=103882](http://go.microsoft.com/fwlink/?LinkId=103882)。
