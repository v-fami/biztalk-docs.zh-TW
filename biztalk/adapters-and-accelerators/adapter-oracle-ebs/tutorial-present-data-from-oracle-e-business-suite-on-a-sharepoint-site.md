# <a name="tutorial-present-data-from-oracle-e-business-suite-on-a-sharepoint-site"></a>教學課程： 從 SharePoint 網站上的 Oracle E-business Suite 中呈現的資料
本教學課程提供詳細的指示使用[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]與 Microsoft Office SharePoint Server 上的 SharePoint 入口網站，從 Oracle E-business Suite 呈現資料。 它也會示範如何設定可讓您從 SharePoint 入口網站上設定的 Oracle E-business Suite 成品執行全文檢索搜尋 Microsoft Office SharePoint Server 中的搜尋應用程式。  
  
 若要示範如何執行這項操作，請考慮儲存員工的相關資訊的 Oracle E-business Suite 中的範例介面資料表。 在本教學課程中，會從根據搜尋字串使用 Oracle E-business Suite 擷取一份客戶的 Microsoft Office SharePoint Server 中建立的應用程式：  
  
-   商務資料清單網頁組件中 Microsoft Office SharePoint Server  
  
-   Microsoft Office SharePoint Server 中的搜尋功能  
  
> [!NOTE]
>  本教學課程之前，確定您已安裝使用的所有必要條件[!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)]與 Microsoft Office SharePoint Server。 必要條件的詳細資訊，請參閱[!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)]安裝指南 》，通常安裝在 C:\Program Files\Microsoft BizTalk 配接器 Pack\Documents。
