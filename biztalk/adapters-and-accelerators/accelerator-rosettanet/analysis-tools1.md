# <a name="analysis-tools"></a>分析工具
Microsoft BizTalk Server 會提供一組分析工具，為商務分析師和資訊工作者。 如需有關這些工具的詳細資訊，請參閱 MicrosoftBizTalk Server 說明。  
  
## <a name="business-activity-monitoring"></a>商務活動監控  
 BizTalk「商務活動監控」(BAM) 提供商務使用者不同性質商務程序的即時檢視。 它提供制定商務決策所需的資料。  
  
## <a name="human-workflow-services"></a>人力工作流程服務  
 BizTalk「人力工作流程服務」(HWS) 讓資訊工作者可透過應用程式來建構、修改、參與及追蹤以人力為主的工作流程程序的進度。 您可以執行工作流程活動、 監視工作流程參與者在日常的工作流程的一部分執行和追蹤的商務程序狀態的可見度的工作流程活動的狀態。
