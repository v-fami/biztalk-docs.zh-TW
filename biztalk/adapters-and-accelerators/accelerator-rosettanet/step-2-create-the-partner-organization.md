# <a name="step-2-create-the-partner-organization"></a>步驟 2： 建立交易夥伴組織
在此步驟中，您會使用 [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] 管理主控台來建立夥伴組織。  
  
### <a name="to-create-the-partner-organization"></a>若要建立夥伴組織  
  
1.  在 **[!INCLUDE[BTARN_CurrentVersion_abbrev](../../includes/btarn-currentversion-abbrev-md.md)]** 管理主控台中，展開 **BizTalk\<版本\>Accelerator for RosettaNet**，以滑鼠右鍵按一下**夥伴**按一下 **新增**，然後按一下 **夥伴**。  
  
2.  在**新交易夥伴屬性**對話方塊方塊中，執行下列動作：  
  
    |使用|動作|  
    |--------------|----------------|  
    |**名稱**|型別**夥伴**。|  
    |**GBI**|型別**987654321**。|  
  
3.  在**新交易夥伴屬性**對話方塊**連絡人屬性**索引標籤上，執行下列動作：  
  
    |使用|動作|  
    |--------------|----------------|  
    |**連絡人名稱**|型別**Jane Doe**。|  
    |**E-mail Address**|型別 **jdoe@fabrikam.com** 。|  
    |**電話號碼**|型別**555-555-5555**。|  
    |**傳真號碼**|型別**555-555-5555**。|  
    |**供應鏈代碼**|型別**Electronic Components**。|  
  
4.  按一下**套用**，然後**確定**。  
  
> [!NOTE]
>  如果您擁有加密憑證，請在這個地方設定。 設定加密憑證的相關資訊，請參閱[管理憑證](../../adapters-and-accelerators/accelerator-rosettanet/managing-certificates1.md)。
