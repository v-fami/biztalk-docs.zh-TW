# <a name="3a2-request-to-3a2-response-map-sample"></a>3A2 要求至 3A2 回應的對應範例
_3A2RequestMessageTo3A2ResponseMessage.btm 範例示範您如何能將 3A2 要求訊息，對應至 3A2 回應訊息。  
  
 根據預設，Microsoft® BizTalk Server 安裝程式將範例安裝在\<*磁碟機*\>: \Program Files\Microsoft BizTalk\<版本\>Accelerator for RosettaNet\SDK\PIPAutomation\DoubleAction。  
  
## <a name="sample-contents"></a>範例內容  
 這個範例會示範如何將 3A2 要求訊息的欄位對應至 3A2 回應訊息。 您可以將這個範例與「雙向動作 PIPAutomation 協調流程」範例一起使用。 「雙向動作 PIPAutomation 協調流程」範例示範如何實作協調流程，以自動產生雙向動作 0C2、0C4、3A2 和 3A4 PIP 的回應。
