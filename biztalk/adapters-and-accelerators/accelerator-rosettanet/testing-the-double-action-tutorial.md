# <a name="testing-the-double-action-tutorial"></a>測試雙向動作教學課程
在本節中，您將使用在前面章節中建立的 Contoso 與 Fabrikam 解決方案，以測試四種不同的「夥伴介面程序」(PIP)：0C2、0C4、3A2 和 3A4。 您將使用 Fabrikam 電腦上的 LOBWebApplication 來產生要求。 系統將使用增強的通訊管道，傳送要求到 Contoso 電腦。 「雙向動作」協調流程將根據您使用的 PIP 產生適當的回應。
