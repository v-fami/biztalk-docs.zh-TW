# <a name="0c4-request-to-0c4-response-map-sample"></a>0c4 要求至 0c 4 的回應對應範例
_0C4RequestMessageTo0C4ResponseMessage.btm 範例示範如何將 0C4 要求訊息對應到 0C4 回應訊息。  
  
 根據預設，Microsoft® BizTalk Server 安裝程式將範例安裝在\<*磁碟機*\>: \Program Files\\Microsoft BizTalk\<版本\>加速器for rosettanet\sdk\pipautomation\doubleaction。  
  
## <a name="sample-contents"></a>範例內容  
 這個範例會示範如何將 0C4 要求訊息的欄位對應至 0C4 回應訊息。 您可以將這個範例與「雙向動作 PIPAutomation 協調流程」範例一起使用。 「雙向動作 PIPAutomation 協調流程」範例示範如何實作協調流程，以自動產生雙向動作 0C2、0C4、3A2 和 3A4 PIP 的回應。
