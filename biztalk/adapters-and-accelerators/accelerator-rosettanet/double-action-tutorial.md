# <a name="double-action-tutorial"></a>雙向動作教學課程
本教學課程涵蓋了使用 [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] 的端點對端點解決方案。 其中詳細描述必須遵循的步驟，透過在兩家虛構公司之間建立交易夥伴實例，藉此實作 RosettaNet 相容解決方案：賣方組織 Contoso 與買方組織 Fabrikam。  
  
 本教學課程使用四種不同的交易夥伴介面程序 (PIP)：  
  
-   0C2 - 非同步測試要求  
  
-   0C4 - 同步測試查詢  
  
-   3A2 - 要求報價以及可用性  
  
-   3A4 - 訂單要求  
  
 本教學課程涵蓋 RosettaNet 架構解決方案的幾個不同領域，包括使用 [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] 管理主控台、建立商務營運系統 (LOB) 應用程式，以及建立可用來整合「企業資源規劃」(Enterprise Resource Planning，ERP) 系統的自訂協調流程。
