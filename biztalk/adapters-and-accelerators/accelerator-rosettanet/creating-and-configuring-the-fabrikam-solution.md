# <a name="creating-and-configuring-the-fabrikam-solution"></a>建立與設定 Fabrikam 解決方案
本節的教學課程詳細描述您應該在 Fabrikam 組織中遵循的步驟。 Fabrikam 組織是一個購買者，也因此作為啟動者，啟動所有交易夥伴介面程序 (PIP) 的要求。 在此教學課程中，您將要建立四個 PIP 的交易夥伴協議，建立並部署商務營運系統 (LOB) ASP[!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)] 應用程式以根據特定 PIP 類型傳送要求至 Contoso 電腦。
