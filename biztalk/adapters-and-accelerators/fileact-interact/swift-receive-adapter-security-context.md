# <a name="swift-receive-adapter-security-context"></a>SWIFT 接收配接器的安全性內容
接收器配接器，不同於傳送配接器，是從 SWIFTNet 連結 (SNL/RA) 的命令提示字元 （SWIFTNet 開始） 啟動。 SNL/RA 組態檔案 (paramconfig) 中設定接收配接器可執行檔。 在啟動配接器初始化 SNL 文件庫根據命令列參數。 組態值會快取供稍後使用。
