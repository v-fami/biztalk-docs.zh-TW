# <a name="uninstall-or-repair-the-fileact-and-interact-adapter"></a>解除安裝或修復 FileAct 和 InterAct 配接器
解除安裝或修復 BizTalk FileAct 和 InterAct 配接器的步驟。  
  
## <a name="uninstall-the-fileact-and-interact-adapter"></a>解除安裝 FileAct 和 InterAct 配接器  
  
1.  在**控制台**，選取**解除安裝程式**。  
  
2.  選取**BizTalk FileAct 和互動的介面卡**，然後選取**解除安裝**或**變更**。  
  
3.  選取**移除**選項，然後再選取**下一步**。  
  
4.  在 [摘要] 選取**解除安裝**。  
  
5.  選取**完成**時完成。

## <a name="repair-the-fileact-and-interact-adapter-installation"></a>修復 FileAct 和互動配接器安裝  
  
1.  在**控制台**，選取**解除安裝程式**。  
  
2.  選取**BizTalk FileAct 和互動的介面卡**，然後選取**變更**。  
3.  在**程式維護**頁面上，選取**修復**選項，然後再選取**下一步**。  
  
4.  在 [摘要] 選取**解除安裝**。  
  
5.  選取**完成**時完成。
