# <a name="swift-send-adapter-uri"></a>SWIFT 傳送配接器 URI
傳送配接器的 URI 是建構根據回應者名稱 (SWIFT: / /\<回應名稱\>)。 回應者的名稱是 SWIFT 所定義的辨別的名稱。 它會指出需要傳送訊息給誰。 SWIFT 會使用此值將訊息路由至收件者。
