# <a name="biztalk-fileact-and-interact-adapters-end-to-end-tutorial"></a>BizTalk FileAct 和互動配接器的端對端教學課程
The Microsoft®[!INCLUDE[swift_adapter](../../includes/swift-adapter-md.md)]端對端教學課程提供您可以使用的特定資訊[!INCLUDE[swift_adapter](../../includes/swift-adapter-md.md)]和[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]來設定即時和儲存和轉送訊息交換案例。  
  
 [!INCLUDE[swift_adapter](../../includes/swift-adapter-md.md)]端對端教學課程提供四個個別案例的教學課程，每種方案類型的形式包含詳細的步驟。 開始這些教學課程之前，您應該了解周圍的 BizTalk Server 的基本概念並熟悉 SWIFT 聯盟閘道 (SAG) 文件。  
  
 A4SWIFT 的端對端教學課程提供您詳細的步驟來設定即時與 FileAct 和 InterAct 配接器使用的儲存和轉送案例 A4Swift。 針對每個案例中，您將會執行下列作業：  
  
-   設定 SWIFT 配接器  
  
-   設定 SWIFTNet paramfile  
  
-   建立傳送埠和接收埠  
  
-   測試案例  
  
 在本教學課程中，您將會進行兩個角色： 寄件者和接收者。 您將建立傳送訊息及接收它們的連接埠。
