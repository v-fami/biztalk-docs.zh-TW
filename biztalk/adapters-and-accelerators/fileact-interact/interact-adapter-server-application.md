# <a name="interact-adapter-server-application"></a>InterAct 配接器伺服器應用程式
此章節提供的撰寫伺服器應用程式要求訊息的一般描述。 這些是從伺服器端 SNL 傳遞至伺服器應用程式的 XML 文件。 因此，這些訊息會運用在伺服器端的 SWIFTNet 原始物件的第一個部分的特性。
