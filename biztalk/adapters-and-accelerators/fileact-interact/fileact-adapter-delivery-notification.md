# <a name="fileact-adapter-delivery-notification"></a>FileAct 配接器傳遞通知
確認後完成 （包括簽章和摘要式驗證） 接收的訊息要求中所示，FileAct 配接器會傳輸給傳送者適當的傳遞通知。  
  
 同樣地，當 FileAct 配接器接收收到傳遞通知時，它會傳送適當的回應。
