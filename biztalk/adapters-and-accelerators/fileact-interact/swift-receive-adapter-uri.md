# <a name="swift-receive-adapter-uri"></a>SWIFT 接收配接器 URI
URI 的接收配接器會建構根據 Tuxedo 結束點的資訊 (SWIFT: / /\<SwTuxSrv\>) SWIFTNet 連結 (SNL)，以及訊息夥伴 (SWIFT: / /\<SagMessagePartner\>)資訊 SWIFTAlliance 閘道接收配接器 (SAG RA) 部署。  
  
 這表示為**應用程式名稱**接收配接器屬性頁上的屬性。
