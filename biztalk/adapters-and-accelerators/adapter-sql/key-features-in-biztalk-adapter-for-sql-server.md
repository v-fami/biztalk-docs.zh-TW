# <a name="key-features-in-biztalk-adapter-for-sql-server"></a>BizTalk Adapter for SQL Server 中的主要功能
此區段會列出中的功能[!INCLUDE[adaptersql](../../includes/adaptersql-md.md)]。  
  
> [!NOTE]
>  本主題已使用的舊版[!INCLUDE[adaptersql](../../includes/adaptersql-md.md)]幫助。  
  
## <a name="features-in-the-sql-adapter"></a>在 SQL 配接器的功能  
 以下是在前一版中引進的功能[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]。  
  
### <a name="technology-related-features"></a>技術相關功能  
  
|                           功能                           |                                                                                                                                                                                                                                                        註解                                                                                                                                                                                                                                                         |
|-------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|        Windows Communication Foundation (WCF) 的使用        | [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]建置最上層的[!INCLUDE[afproductnamelong](../../includes/afproductnamelong-md.md)] ([!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)])。 接著，[!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]以 WCF 為基礎。 配接器會公開為 WCF 配接器用戶端通道。 這可讓連線、 中繼資料交換，以及與外部系統的商務資料交換。 |
| WCF 通道模型和 WCF 服務模型的支援 |                                                            在 WCF 通道模型中，配接器用戶端可取用[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]藉由直接傳送和接收 XML 訊息。 在 WCF 服務模型中，配接器用戶端可以產生.NET proxy 類別，從 Web 服務描述語言 (WSDL) 使用取得[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]。                                                            |
|                64 位元平台支援。                 |                                                                                                                                                                                                      [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]現在是適用於 64 位元平台。                                                                                                                                                                                                      |
  
### <a name="metadata-related-features"></a>中繼資料相關的功能  
  
|                               功能                                |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               註解                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|----------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                瀏覽、 搜尋及擷取中繼資料                 |                                                                                                                                                                                                                                                                                                                                                                          配接器用戶端可以瀏覽，並在批次中搜尋中繼資料，藉由指定批次大小。 程式設計到配接器而不是透過使用配接器服務 BizTalk 專案增益集時，才使用這項功能。 中繼資料搜尋支援資料表、 檢視、 程序、 純量函數和資料表值函式層級。 搜尋字串使用直接 SQL 陳述式。                                                                                                                                                                                                                                                                                                                                                                           |
| 叫用不同的資料庫中相同名稱的成品的支援 | 在  [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]，包含只有結構描述名稱，以及在某些情況下，物件名稱的 XML 結構描述定義 (XSD) 檔案中的命名空間。 不過，如果應用程式想要在不同資料庫中執行具有不同的中繼資料的相同具名成品的作業，就會發生衝突產生的中繼資料。 區分中繼資料的唯一方式是使用 XSD 命名空間中的資料庫名稱。<br /><br /> 最新版[!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]可讓您指定 XSD 命名空間中的資料庫名稱，藉由設定的值**UseDatabaseNameInXsdNamespace**繫結屬性為 TRUE。 繫結屬性的預設值為 false，這表示 XSD 命名空間將不會包含資料庫名稱。 如需詳細資訊**UseDatabaseNameInXsdNamespace**繫結屬性，請參閱[了解 BizTalk Adapter for SQL Server 配接器繫結屬性](../../adapters-and-accelerators/adapter-sql/read-about-the-biztalk-adapter-for-sql-server-adapter-binding-properties.md)。 |

| 執行預存程序使用或不使用 FOR XML 子句的支援 |                                                                                              [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)]可讓您執行具有或不含 FOR XML 子句的 SELECT 陳述式的預存程序。 舊版配接器的支援只有這些預存程序具有 FOR XML 子句，SELECT 陳述式中。 執行預存程序的相關資訊，請參閱[使用 SQL 配接器的 SQL Server 中執行預存程序](../../adapters-and-accelerators/adapter-sql/execute-stored-procedures-in-sql-server-using-the-sql-adapter.md)。                                                                                               |
