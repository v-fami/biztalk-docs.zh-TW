# <a name="lesson-5-deploy-the-solution"></a>第 5 課： 部署方案
在這一課，您會部署在 BizTalk 方案中使用[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理主控台。 在 BizTalk 協調流程設計工具中，您可以建立邏輯連接埠。 現在請建立實體連接埠並將邏輯連接埠對應到實體連接埠，以設定的 BizTalk 應用程式。  
  
 設定應用程式之後，您將會啟動，並測試應用程式。
