# <a name="module-1-creating-a-swift-solution"></a>模組 1： 建立 SWIFT 解決方案
在這個模組中，您可以建立空白的 BizTalk 方案。 在第一課，您必須建立新的解決方案藉由使用 Microsoft [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)] [!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)]。 您也會建立包含您的 BizTalk 解決方案元件，例如結構描述、 對應、 協調流程和管線元件的個別專案。  
  
 請注意，方案可能包含只有一個 BizTalk 專案，或 （如果解決方案包含多個專案，其中您獨立開發和更新版本，您必須整合），它可能包含許多的 BizTalk 專案。  
  
### <a name="to-create-a-blank-biztalk-solution"></a>若要建立空白的 BizTalk 解決方案  
  
1. 開始**Visual Studio**。  
  
2. 在 **[檔案]** 功能表上，指向 **[開新檔案]**，然後按一下 **[專案]**。  
  
3. 在 **專案類型**窗格中的 新增專案 對話方塊中，選取**BizTalk 專案**。 在 [範本] 窗格中，選取**空白 BizTalk Server 專案**。  
  
4. 在 **名稱**方塊中，輸入**SWIFTProject**做為方案名稱。  
  
5. 在 **位置**方塊中，輸入 **\<*磁碟機*\>: \labs**。  
  
6. 按一下 **確定**開啟新的專案。  
  
    [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)] 在 [方案總管] 和 新的空白方案中建立空白方案\<*磁碟機*:\>\labs 資料夾。  
  
   請繼續進行[模組 2： 加入新的結構描述專案](../../adapters-and-accelerators/accelerator-swift/module-2-adding-a-new-schemas-project.md)。
