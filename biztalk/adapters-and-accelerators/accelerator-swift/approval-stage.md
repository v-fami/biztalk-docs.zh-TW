# <a name="approval-stage"></a>核准階段
訊息會先經過簽署及所提交核准之後，核准鏈結中的每個核准者檢閱和副署訊息，他或她的數位簽章加入至堆疊。 核准者只能檢視和副署的訊息。 他們無法變更它，而不會中斷數位簽章堆疊。
