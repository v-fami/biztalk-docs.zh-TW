# <a name="backing-up-the-a4swift-database"></a>A4SWIFT 資料庫的備份
您經常應該備份資料庫，BizTalk Server 和 A4SWIFT 系統較低的嚴重失敗的風險中。 這些資料庫包括 BizTalk Server 來源系統，和 A4SWIFT 資料庫。 除了降低風險，這也可讓您清除 A4SWIFT 可以成長到極大的歷程記錄檔案。  
  
 如需 BizTalk Server 來源系統中備份資料庫的詳細資訊，請參閱 BizTalk Server 說明中的 「 備份和還原 BizTalk Server 資料庫 」 主題。 本主題描述您用來備份 BizTalk 資料庫 「 備份 BizTalk Server 」 工作。  
  
 如需備份 A4SWIFT 資料庫資訊，請參閱 < 如何以重新向上自訂資料庫 > 主題在 BizTalk Server 說明中。 本主題描述如何修改備份 BizTalk Server 作業，以包含自訂 A4SWIFT 資料庫。
