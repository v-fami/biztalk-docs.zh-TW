# <a name="module-5-adding-a-flat-file-receive-location-and-xml-send-port"></a>模組 5： 加入一般檔案接收位置和 XML 傳送埠
在此模組中，您可以加入一個一般檔案接收埠和位置，和一個 XML 傳送埠。 您使用的接收埠和位置來接收 MT103 一般檔案從檔案資料夾。 您可以使用傳送埠 MT103 訊息傳送至以 XML 格式的輸出檔案資料夾。
