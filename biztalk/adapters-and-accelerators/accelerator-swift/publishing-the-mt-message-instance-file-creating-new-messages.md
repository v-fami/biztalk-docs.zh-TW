# <a name="publishing-the-mt-message-instance-file-creating-new-messages"></a>發行 （建立新的訊息） 的 MT 訊息執行個體檔案
**若要發行的 MT 訊息執行個體檔案：**  
  
1.  移至輸出資料夾的 InfoPath 表單範本，在上一個步驟中產生**...\MTXXX\TemplateDS\InfoPath 表單範本**。  
  
2.  在 Internet Explorer 中，開啟**http://localhost/sites/BASSite/New%20SWIFT%20MT%20Messages**。  
  
3.  在新的 Swift MT 訊息視窗中，按一下 **上傳**。  
  
4.  在上傳文件視窗中，按一下 **瀏覽**。  
  
5.  在 [選擇檔案] 對話方塊中，移至輸出資料夾的 InfoPath 表單上一個步驟中產生**...\MTXXX\TemplateDS\InfoPath 表單範本**。 選取例如 MT103.xml MTXXX.xml 檔案，然後按一下**開啟**。  
  
6.  在上傳文件視窗中，按一下 **確定**。  
  
7.  在新的 SWIFT MT 郵件中： MTXXX 視窗中的，於命名空間中，輸入**http://schemas.microsoft.com/BizTalk/Solutions/FinancialServices/SWIFT/EnvelopeMTxxx**，然後按一下  **確定**。
