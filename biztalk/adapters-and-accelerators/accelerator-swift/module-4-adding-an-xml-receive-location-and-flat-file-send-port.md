# <a name="module-4-adding-an-xml-receive-location-and-flat-file-send-port"></a>模組 4： 加入 XML 接收位置和一般檔案傳送埠
在此模組中，您可以加入一個接收埠和位置，和一個傳送埠。 您可以使用的接收埠和位置來接收輸入的檔案資料夾中的 XML 格式 MT103 訊息。 您可以使用傳送埠將已驗證的 MT103 一般檔案訊息傳送至輸出檔案資料夾。
