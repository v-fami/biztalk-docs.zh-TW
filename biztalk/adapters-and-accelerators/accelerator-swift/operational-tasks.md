# <a name="operational-tasks"></a>操作工作
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)][!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]包含一個或多部伺服器和一系列工具企業應用程式整合 (EAI)，自動化財務商務程序，並加速使用 SWIFT FIN 標準的訊息交換。  
  
 作業會包括管理工作的[!INCLUDE[btsBizTalkServer2006r3](../../includes/btsbiztalkserver2006r3-md.md)]應用程式。 A4SWIFT 會使用下列工具和主控台執行這些作業：  
  
-   **BizTalk Server 管理主控台。** 您可以使用此工具來管理 BizTalk 群組，包括所有的 BizTalk 資料庫來追蹤商務程序的效能。  
  
-   **商務活動監控 (BAM) 架構。** 使用此工具來追蹤資料的項目和商務交易期間所收集的商務里程碑。
