# <a name="network-load-balancing"></a>網路負載平衡
分散式的 A4SWIFT 部署可能包含訊息、 協調流程中，主機 MRSR 站台或 SQL Server 資料庫的 SharePoint 伺服器的 BizTalk Server 電腦。 如果您使用設定檔和商務需求夠大，您可能需要設定網路負載平衡 (NLB) 兩個或多個 BizTalk HTTP 前端 （MRSR 站台） 或兩個或多個 BizTalk 訊息伺服器。 需要網路負載平衡的多部伺服器的範例，請參閱中的 「 範例的完整 A4SWIFT 部署 」 圖表[實體圖](../../adapters-and-accelerators/accelerator-swift/physical-diagram.md)。  
  
 如需網路負載平衡的詳細資訊，請參閱 BizTalk Server 部署指南 》，和[步驟 2： 設定伺服器上的 NLB](../../adapters-and-accelerators/accelerator-swift/step-2-configuring-nlb-on-the-servers.md)。
