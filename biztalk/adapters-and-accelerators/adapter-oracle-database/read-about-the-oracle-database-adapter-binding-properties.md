# <a name="read-about-the-oracle-database-adapter-binding-properties"></a>了解 Oracle 資料庫配接器繫結屬性
[!INCLUDE[adapteroracle](../../includes/adapteroracle-md.md)]會呈現數個繫結屬性。 藉由設定這些屬性，您可以控制某些配接器的行為。 本章節描述[!INCLUDE[adapteroracle_short](../../includes/adapteroracle-short-md.md)]繫結屬性。 它也會示範如何存取它們使用的.NET 程式設計或設定屬性[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]實體連接埠繫結。
