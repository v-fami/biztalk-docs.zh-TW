# <a name="update-center-for-host-integration-server-and-ole-db-provider-for-db2"></a>Host Integration Server 和 OLE DB Provider for DB2 的更新中心

尋找並管理更新 Host Integration Server (HIS) 和 Microsoft OLE DB Provider for DB2 （隨附於 Microsoft SQL Server Feature Pack）。

## <a name="overview"></a>概觀
此頁面會列出最新的更新，針對不同版本的 Host Integration Server 和 Microsoft OLE DB Provider for DB2。 

> [!NOTE]
> [Microsoft 產品生命週期頁面](http://support.microsoft.com/lifecycle/)列出不同 HIS 版本的支援的狀態。

## <a name="biztalk-adapters-for-host-systems"></a>BizTalk Adapters for Host Systems
有四個 BizTalk Adapters for Host Systems 隨附於他：

* BizTalk Adapter for DB2
* BizTalk Adapter for 主應用程式
* BizTalk Adapter for 主機檔案
* BizTalk Adapter for MQ 系列用戶端 (MQSC)

這些介面卡是他的更新累計更新。
