# <a name="installing-the-exception-management-samples-using-install-scripts"></a>安裝使用安裝指令碼的例外狀況管理範例
本章節描述如何安裝例外狀況管理範例，以及在隨附的安裝指令碼從[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]。  
  
 **若要從安裝指令碼安裝 ESB 例外狀況管理範例**  
  
1.  在 **[開始]** 功能表上，按一下 **[執行]**。  
  
2.  在**執行**] 對話方塊中，輸入**cmd**，然後按 ENTER 開啟 [命令提示字元。  
  
3.  執行下列命令，取代*\<路徑 >*參數與您想要安裝的.cmd 檔案的完整路徑 （在此版本中的名稱與預設路徑是 \Source\Samples\Exception Handling\Install\Scripts\\):  
  
    ```  
    <path>\Setup_bin.cmd  
    ```
