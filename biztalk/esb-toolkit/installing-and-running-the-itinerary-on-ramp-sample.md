# <a name="installing-and-running-the-itinerary-on-ramp-sample"></a>安裝及執行路線上手範例
行程上手範例將示範如何設定 ESB 路線屬性執行單向和要求-回應傳訊的案例，如下所示：  
  
-   執行會使用訊息服務行程  
  
-   執行使用協調流程路線  
  
-   執行路線使用傳訊服務和協調流程的步驟組合  
  
 行程上手範例會使用 Windows Form 測試用戶端應用程式可執行的行程 Web 服務方法並顯示傳回的訊息。 您可以開啟測試用戶端專案來執行它，並檢查以查看如何使用您自己的服務導向架構 (SOA) 和 ESB 應用程式中的行程 Web 服務的程式碼。
