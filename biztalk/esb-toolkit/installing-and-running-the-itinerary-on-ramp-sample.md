# <a name="installing-and-running-the-itinerary-on-ramp-sample"></a>安裝和執行路線入口範例
路線入口範例將示範如何設定 ESB 路線屬性執行單向和要求-回應傳訊案例，如下所示：  

- 執行路線，會使用傳訊服務  

- 執行路線，使用協調流程  

- 執行路線使用傳訊的服務和協調流程的步驟組合  

  路線入口範例會使用 Windows Form 測試用戶端應用程式來執行路線 Web 服務的方法，並顯示傳回的訊息。 您可以開啟測試用戶端專案來執行它，以及檢查程式碼，以了解如何使用您自己的服務導向架構 (SOA) 和 ESB 應用程式中的行程 Web 服務。
