# <a name="amberpoint-biztalk-nano-agent"></a>AmberPoint BizTalk Nano 代理程式
AmberPoint SOA 管理系統 (SMS) 提供了詳盡的視覺效果與分析工具來解決執行階段控管的挑戰。 [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]和 Microsoft BizTalk Server 整合的 SOA 商務應用程式和服務比較簡單，但分析結果的複合應用程式和協調流程會更具有挑戰性。 AmberPoint 的 BizTalk Server 整合，透過其 BizTalk Nano 代理程式會擴充現有的執行階段治理功能，可包含 BizTalk Server 接收位置和傳送埠，不論其基礎的配接器繫結。
