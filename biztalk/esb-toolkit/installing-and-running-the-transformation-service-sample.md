# <a name="installing-and-running-the-transformation-service-sample"></a>安裝及執行 「 轉換服務 」 範例
轉換 Web 服務直接公開透過 Web 服務，因此消除了 BizTalk Messagebox 資料庫往返的訊息在 Microsoft BizTalk 部署的對應。 轉換 Web 服務可讓外部應用程式提交至 Microsoft.Practices.ESB 應用程式的文件，並讓它使用部署在 BizTalk 對應轉換。
