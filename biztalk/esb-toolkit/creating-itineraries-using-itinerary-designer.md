# <a name="creating-itineraries-using-itinerary-designer"></a>建立行程使用路線設計工具
您可以使用[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]路線設計工具來建立 ESB 中繼原則，名為行程，在執行階段的行程為基礎的路由使用。 此工具會提供數個圖形或對應至不同的動作，您可能會設計路線時執行的模型項目。  
  
 本節提供概念資訊和指導方針來協助您了解如何開發使用路線設計工具的行程。 它包含下列主題：  
  
-   [有關行程](../esb-toolkit/about-itineraries.md)  
  
-   [在路線設計工具中工作](../esb-toolkit/working-in-itinerary-designer.md)  
  
-   [行程中實作的設計模式](../esb-toolkit/implementing-design-patterns-in-itineraries.md)  
  
-   [開發活動](../esb-toolkit/development-activities.md)
