# <a name="install-the-namespace-component-sample-using-the-install-scripts"></a>安裝使用安裝指令碼的命名空間 」 元件範例
本章節描述如何安裝 「 命名空間元件 」 範例隨附的 Windows Installer 檔案從[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]。  
  
 **若要從安裝指令碼安裝命名空間 」 元件範例**  
  
1.  在 **[開始]** 功能表上，按一下 **[執行]**。  
  
2.  在**執行**] 對話方塊中，輸入**cmd**，然後按 ENTER 開啟 [命令提示字元。  
  
3.  執行下列命令，取代*\<路徑\>*參數與您想要安裝的.cmd 檔案的完整路徑 （在此版本中的預設路徑是 \Source\Samples\Namepace\Install\Scripts\\):  
  
    ```  
    <path>\Setup_bin.cmd  
    ```
