# <a name="installing-the-multiple-web-services-sample"></a>安裝多個 Web 服務範例
本節描述安裝多個 Web 服務範例的程序。 您必須安裝中的範例[安裝及執行路線上手範例](../esb-toolkit/installing-and-running-the-itinerary-on-ramp-sample.md)和[安裝及執行動態解析範例](../esb-toolkit/installing-and-running-the-dynamic-resolution-sample.md)您安裝及使用這個範例。  
  
 **若要安裝多個 Web 服務範例**  
  
1.  在 Windows 檔案總管 中，開啟 \Source\Samples\MultipleWebServices\Install\Scripts 資料夾。  
  
2.  按兩下名為 Setup_bin.cmd 的指令碼檔案。
