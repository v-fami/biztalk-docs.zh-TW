# <a name="new-registry-entry-page"></a>[新的登錄項目] 頁面
圖 1 顯示 ESB Management 入口網站新登錄項目頁面，讓您可以檢視現有端點的清單中指定的 Microsoft BizTalk Server ESB 應用程式。  
  
 ![新的登錄項目頁面](../esb-toolkit/media/ch8-newregistryentrypage.gif "Ch8 NewRegistryEntryPage")  
  
 **圖 1**  
  
 **ESB Management 入口網站新登錄項目頁面**  
  
 下列清單說明如何使用 ESB Management 入口網站新登錄項目頁面的功能：  
  
-   使用三個下拉式清單頂端附近的頁面以指定類型、 狀態和 BizTalk 應用程式包含您想要檢視的端點：  
  
    -   在第一個 （端點型別） 下拉式清單中，您可以選取**接收位置、 傳送埠**或**所有端點**。  
  
    -   在第二個 （狀態） 下拉式清單中，您可以選取**已註冊、 NotRegistered，暫止，** 或**所有**。  
  
    -   在第三個 （BizTalk 應用程式） 下拉式清單中，您可以選取其中一個目前已安裝的 BizTalk 應用程式。  
  
-   指定的應用程式與端點的準則後, 按一下 BizTalk 應用程式下拉式清單，以顯示相符的端點清單旁邊的箭頭圖示。  
  
-   端點的清單包含比對端點的詳細的資料。 每個不是動態的端點，此清單會包含一個連結，可讓您檢視和編輯發佈此端點的擱置中要求的詳細資料。  
  
-   按一下**檢視詳細資料**連結以開啟清單的任何資料列中[登錄詳細資料頁面](../esb-toolkit/registry-details-page.md)從中您可以修改要求，並將發行端點。
