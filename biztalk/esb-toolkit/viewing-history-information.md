# <a name="viewing-history-information"></a>檢視歷程記錄資訊
ESB 管理入口網站會維護一份記錄的通用描述、 探索與整合 (UDDI) 登錄的所有要求。 系統管理員可以檢視在入口網站中的這份清單。  
  
### <a name="to-view-uddi-request-history"></a>若要檢視 「 UDDI 要求歷程記錄  
  
1.  請確定您在入口網站使用 （入口網站管理員的自動成員） 的 Microsoft BizTalk Server 系統管理員帳戶群組成員的帳戶登入。  
  
2.  指向**登錄**入口網站的主功能表上的索引標籤，然後按一下 **管理暫止要求**開啟入口網站[管理擱置的要求頁面](../esb-toolkit/manage-pending-requests-page.md)。  
  
3.  按一下**檢視要求記錄**連結。
