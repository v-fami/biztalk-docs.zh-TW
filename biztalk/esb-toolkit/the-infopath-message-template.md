# <a name="the-infopath-message-template"></a>InfoPath 訊息範本
若要檢視 ESB 管理入口網站中的 ESB 錯誤訊息或者，使用者可以利用名為 ESB 例外狀況訊息檢視器中，所提供的 Microsoft InfoPath 訊息範本[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]。  
  
 ESB 例外狀況管理架構可以保存的序列化格式，以及 [ESB 例外狀況訊息檢視器] 範本中，將會顯示錯誤訊息和所包含的原始訊息的數個檢視中的訊息。 ESB 例外狀況訊息檢視器會提供下列檢視：  
  
- 一般檢視  
  
- 例外狀況物件的檢視  
  
- Messages view (訊息檢視)
