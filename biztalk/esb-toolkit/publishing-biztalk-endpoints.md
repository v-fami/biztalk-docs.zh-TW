# <a name="publishing-biztalk-endpoints"></a>發佈 BizTalk 端點
您可以使用 ESB 管理入口網站建立和發行項目變成目前設定的通用描述、 探索與整合 (UDDI) 伺服器。  
  
### <a name="to-publish-a-microsoft-biztalk-server-endpoint-into-the-current-uddi-server"></a>若要將 Microsoft BizTalk Server 端點發行至目前的 UDDI 伺服器  
  
1.  指向**登錄**入口網站的主功能表上的索引標籤，然後按一下 **新登錄項目**開啟[新登錄項目頁面](../esb-toolkit/new-registry-entry-page.md)。  
  
2.  使用頁面上的下拉式清單，做為篩選條件的端點類型、 狀態，以及應用程式端點所在的位置。  
  
3.  按一下您要建立的 UDDI 登錄要求發行的端點旁邊的箭頭圖示。  
  
4.  如果系統管理員已啟用自動發行的端點，在入口網站會自動發佈的 UDDI 伺服器到新的項目。  
  
5.  如果系統管理員尚未啟用自動發行的端點，將要求保留在佇列中直到系統管理員核准或拒絕要求。
