# <a name="portal-admin-pages"></a>入口網站管理頁面
當您將滑鼠移**Admin**索引標籤上，快顯功能表提供入口網站的系統管理工作的清單。 **Admin**  索引標籤，只會顯示擁有 （Microsoft BizTalk Server 系統管理員帳戶群組的成員） 的系統管理權限的使用者。 在快顯功能表上，選項如下：  
  
-   **管理稽核記錄檔**。 此功能表項目會開啟[稽核記錄檔 頁面](../esb-toolkit/audit-log-page.md)包含稽核事件的清單。 您可以從這個頁面上，檢視與稽核事件相關聯的訊息詳細資料。  
  
-   **錯誤設定**。 此功能表項目會開啟[錯誤設定 頁面](../esb-toolkit/fault-settings-page.md)其中您可以指定一系列的系統管理設定針對稽核、 警示的服務佇列和警示的電子郵件服務。  
  
-   **管理警示**（系統管理 檢視）。 此功能表項目會開啟[入口網站 [警示] 頁面](../esb-toolkit/portal-alerts-page.md)管理 檢視中。 系統管理員可以使用此頁面來檢視和編輯所有警示和系統內的訂閱。 它們也可以檢視和編輯與特定警示相關聯的訂用帳戶。  
  
-   **登錄設定**。 此功能表項目會開啟[登錄設定 頁面](../esb-toolkit/registry-settings-page.md)您可以在其中指定一系列的系統管理範本、 電子郵件通知與登錄的整合設定。
