 **若要安裝 「 轉換服務 」 範例的安裝指令碼**  
  
1.  在 **[開始]** 功能表上，按一下 **[執行]**。

3.  執行下列命令，取代*\<路徑 >*參數與您想要安裝的.cmd 檔案的完整路徑 （在此版本中的預設路徑是 \Source\Samples\TransformServices\Install\Scripts\\):  
  
    ```  
    <path>\TransformServices_Install.cmd
    ```
