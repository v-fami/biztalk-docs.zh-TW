 **若要安裝 「 轉換服務 」 範例的安裝指令碼**  
  
1.  在 **[開始]** 功能表上，按一下 **[執行]**。  
  
2.  在**執行**] 對話方塊中，輸入**cmd**，然後按 ENTER 開啟 [命令提示字元。  
  
3.  執行下列命令，取代*\<路徑\>* 參數與您想要安裝的.cmd 檔案的完整路徑 （在此版本中的預設路徑是 \Source\Samples\TransformServices\Install\指令碼\\):  
  
    ```  
    <path>\TransformServices_Install.cmd  
    ```
