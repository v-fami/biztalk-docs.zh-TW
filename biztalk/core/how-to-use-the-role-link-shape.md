# <a name="how-to-use-the-role-link-shape"></a>如何使用角色連結圖形
A**角色連結**圖形包含供實作角色提供者角色和使用角色或取用者角色的預留位置。 它可以包含上述其中一種角色，或兩者皆包含。  
  
 您可以直接將連接埠類型**角色連結**圖形，使用現有角色或新的角色，以及現有或新的連接埠類型。  
  
> [!NOTE]
>  建議您最好在建立角色連結之前先建立角色連結類型及關聯的連接埠類型，好讓您可以使用現有的角色連結類型來定義角色連結。 如需詳細資訊，請參閱[如何在協調流程中建立角色連結](../core/how-to-create-role-links-in-orchestrations.md)。  
  
### <a name="to-add-a-role-link"></a>新增角色連結  
  
-   以滑鼠右鍵按一下 連接埠介面，然後按一下 **新角色連結**。  
  
     -或者-  
  
     從**BizTalk 協調流程** 索引標籤的 工具箱 拖曳**角色連結**圖形到連接埠介面。  
  
### <a name="to-remove-a-role-link"></a>移除角色連結  
  
-   以滑鼠右鍵按一下**角色連結**以移除，然後按一下**刪除**。
