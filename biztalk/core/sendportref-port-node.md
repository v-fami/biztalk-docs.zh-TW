# <a name="sendportref-port-node"></a>SendPortRef (連接埠節點)
繫結檔案 [連接埠] 節點的 [SendPortRef] 節點包含與服務所參考之傳送埠有關的資訊。  
  
## <a name="nodes-in-the-sendportref-node"></a>SendPortRef 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|名稱|Attribute|xs:string|指定服務所參考之傳送埠的名稱。|不需要|預設值：空白|
