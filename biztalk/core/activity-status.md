# <a name="activity-status"></a>活動狀態
結果詳細資料的 [活動狀態] 區域，可讓您在單一頁面中檢視結果項目的所有詳細資料。  
  
 [活動狀態] 區域包含兩個子區域：[里程碑] 和 [資料]。  
  
## <a name="milestones"></a>里程碑  
 [里程碑] 區域會顯示為選定的活動所定義的所有里程碑。 里程碑資料會顯示達到里程碑的日期和時間。 有些里程碑是互斥的；例如已核准和已拒絕之訂單的里程碑。 在互斥里程碑的情況下，其中一個會顯示里程碑的日期和時間資料，另一個則是空白，表示該里程碑沒有資料。  
  
## <a name="data"></a>資料  
 [資料] 區域會顯示為活動定義的所有資料項目。 資料項目是依字母順序顯示。
