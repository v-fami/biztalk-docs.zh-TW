# <a name="using-design-time-xml-tools"></a>使用設計階段 XML 工具
EDI XML 工具可讓您針對 EDI 結構描述和對應執行設計階段作業。 在 Visual Studio 的 [方案總管] 窗格中，以滑鼠右鍵按一下結構描述或對應，即可從顯示的功能表中執行這些工具。
