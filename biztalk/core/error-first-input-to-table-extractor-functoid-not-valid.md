# <a name="error---first-input-to-table-extractor-functoid-not-valid"></a>錯誤-無效的表格抽選程式運算質的第一個輸入
**錯誤碼**  
  
 btm1019  
  
 **說明**  
  
 第一個輸入的參數所指定**表格抽選程式**運算質不可以從連結**表格迴圈**運算質，視需要。  
  
 **使用者動作**  
  
 指定之間建立連結**表格抽選程式**運算質與適當**表格迴圈**運算質拖曳到另一個其中一個。 在地圖格線頁上，前一個運算質必須位於後一個運算質右邊。 此外，使用**設定\<運算質 > 運算質**對話方塊方塊中，確認新建立的連結是第一個輸入的參數所指定**表格抽選程式**運算質。
