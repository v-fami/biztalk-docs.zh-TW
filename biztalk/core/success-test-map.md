# <a name="success---test-map"></a>成功-測試對應
**錯誤碼**  
  
 btm1053  
  
 **說明**  
  
 指定對應的「測試對應」作業成功，並已產生指定的輸出執行個體訊息檔案。  
  
 **使用者動作**  
  
 視需要檢查指定的輸出執行個體訊息檔案，此檔案由「測試對應」作業所產生。
