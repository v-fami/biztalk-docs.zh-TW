# <a name="how-to-add-a-component-to-a-pipeline"></a>如何新增元件至管線
從工具箱拖曳元件至設計介面以新增元件至管線。  
  
### <a name="to-add-a-component-to-a-pipeline"></a>新增元件至管線  
  
-   在 工具箱 拖曳 管線元件拖曳至**放置到此處 ！** 在設計介面上的方塊。  
  
 以下圖例顯示管線設計介面如何說明管線。 此管線有兩個階段：「組合」階段和「編碼」階段。 XML 組合器管線元件已加入至 「 組合 」 階段，但 「 編碼 」 階段仍為空白，因為它仍顯示**放置到此處 ！** 表示管線元件，可以新增至階段。  
  
 ![階段與 BizTalk 管線中的元件](../core/media/ebiz-pipe-stages02.gif "ebiz_pipe_stages02")  
說明 BizTalk 管線中的階段和元件。  
  
> [!NOTE]
>  管線元件只能放置在設計介面上的特定位置。 管線元件只能放置在與它有階段相關性的階段中。 指標旁邊若出現有線條劃過的圓圈將無法放置管線元件。 管線元件可以放置在出現箭號且設計介面反白的部分。
