# <a name="resend-status-details-report"></a>重新傳送狀態詳細資料報表
此狀態報表嘗試重試合作對象當做 AS2 接收者的合作對象屬性設定為重新傳送 AS2 訊息，如果未收到 MDN 時顯示資訊。  
  
 這個報表顯示 [AS2/MDN 狀態] 報告中的訊息上按一下滑鼠右鍵，然後按一下**檢視可靠傳訊狀態**。 重新傳送狀態詳細資料頁面會再顯示此訊息重新傳送嘗試資訊。  
  
 此報表才可使用您已選取**重新傳送 AS2 訊息，若未收到 MDN**的合作對象做為相關合作對象的 AS2 訊息接收者屬性。  
  
 報表會顯示在下列頁面上的訊息重試次數的資訊：  
  
|頁面|顯示資料|  
|----------|--------------------|  
|**一般**|目前傳送嘗試，以及重新傳送組態的索引。|  
|**重新傳送歷程記錄**|訊息的傳送嘗試的歷程記錄。|
