# <a name="how-to-define-durations"></a>如何定義持續時間
持續時間可以指定時間量 (由特定活動項目的開始和結束里程碑所定義)。 例如，訂單的有效日數。  
  
 使用「BAM 檢視精靈」來設定或修改持續時間。 在設定持續時間之前，請先使用「BAM 活動精靈」來建立商務活動及項目。 如需有關如何使用精靈的詳細資訊，請參閱[定義商務活動和在 Excel 中的檢視](../core/defining-business-activities-and-views-in-excel.md)。  
  
### <a name="to-define-durations"></a>定義持續時間  
  
1.  在 BAM 檢視精靈中，按一下 **下一步**直到您看到第二個**編輯 BAM 檢視： 檢視項目**頁面，然後再按一下**新持續時間**。  
  
    > [!NOTE]
    >  您必須要有至少兩個里程碑或群組，才可以建立持續時間。  
  
2.  輸入的名稱**持續時間名稱**。  
  
3.  從下拉式清單選取**開始商務里程碑**，**結束商務里程碑**，**時間解析**，然後按一下 **確定**。
