# <a name="error---cannot-resolve-schemalocation-attribute"></a>錯誤-無法解析 schemaLocation 屬性
**錯誤碼**  
  
 BEC8001  
  
 **說明**  
  
 BizTalk 編輯器無法找到由您在開啟的結構描述所匯入的結構描述。  
  
 **使用者動作**  
  
 按一下**結構描述**節點。 在 [屬性] 視窗中，按一下**匯入**屬性。 省略符號 (**...**) 按鈕變成無法使用。 按一下按鈕 。 **匯入**對話方塊隨即出現。 請在對話方塊中編輯要匯入結構描述的路徑。
