# <a name="how-to-display-the-sso-server-using-the-client-utility"></a>如何顯示 SSO 伺服器使用用戶端公用程式
使用此命令以顯示使用者目前指向的「單一登入伺服器」。  
  
### <a name="to-display-the-sso-server-using-the-client-utility"></a>使用用戶端公用程式顯示 SSO 伺服器  
  
1.  在 **啟動** ] 功能表上，按一下 [ **執行**, ，然後輸入 **cmd**。  
  
2.  在命令列，移至「企業單一登入」安裝目錄。 預設安裝目錄是*\<磁碟機\>*: \Program Files\Common Files\Enterprise Single Sign-on。  
  
3.  型別 **ssoclient – showserver**。  
  
    > [!NOTE]
    >  在支援使用者帳戶控制 (UAC) 的系統上，您可能需要使用系統管理權限來執行工具。
