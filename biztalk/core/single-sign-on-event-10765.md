## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10765|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_NO_CREDENTIALS|  
|訊息文字|對應已設定無認證。|  
  
## <a name="explanation"></a>說明  
 您已要求在對應上的 GetCredentials 和指定的對應具有無認證。  
  
## <a name="user-action"></a>使用者動作  
 使用命令列或 MMC 嵌入式管理單元來設定對應的認證。
