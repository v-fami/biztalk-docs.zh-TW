# <a name="btadhostclass"></a>BTAD_HostClass
BTAD_HostClass 指出作業是在 BizTalk 管理資料庫中，還是在本機電腦上執行。  
  
## <a name="remarks"></a>備註  
 下表描述 BTAD_HostClass 的可能值。  
  
|Value|說明|  
|-----------|-----------------|  
|ConfigurationDb|指定要在群組的 BizTalk 管理資料庫中執行匯入作業。|  
|BizTalkHostInstance|指定要在本機電腦上執行安裝或解除安裝作業。|
