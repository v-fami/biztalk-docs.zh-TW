# <a name="facts"></a>事實
「事實」 (Fact) 是與世界相關的不連續資訊。 事實可以來自於許多來源 （事件系統，在商務應用程式、 資料庫資料表等物件），而且必須送入中的商務規則引擎[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]使用其中一種下列項目：  
  
-   .NET 物件 (方法、屬性和欄位)  
  
-   XML 文件 (項目、屬性和文件子區段)  
  
-   資料庫資料列集 (資料表資料行的值)  
  
 在商務規則編輯器中，您可以使用 [事實總管] 來瀏覽各種來源中規則內的資料，也可以將資料放入規則中。
