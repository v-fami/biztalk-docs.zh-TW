# <a name="windows-sharepoint-services-adapter-walkthroughs"></a>Windows SharePoint Services 配接器逐步解說
本節包含三個逐步解說。 第一個顯示如何使用 Windows SharePoint Services 配接器與以內容為基礎的路由 (CBR) 來設定 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 傳送與接收訊息。 第二個顯示如何整合 BizTalk Server 與 Microsoft Office。 第三個顯示如何在執行階段存取內送訊息的 Windows SharePoint Services 內容屬性，然後使用協調流程中的動態連接埠，根據屬性決定訊息的目的地。
