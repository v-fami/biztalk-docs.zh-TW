# <a name="managing-bam-definitions"></a>管理 BAM 定義
BAM 定義是 BAM 基礎結構的一部分。 其內容定義所要追蹤和彙總的資料，以及追蹤資料的商務使用者檢視。 本節主題說明用以管理 BAM 定義之要素的各項程序，這些要素包括活動、檢視、成品和警示。
