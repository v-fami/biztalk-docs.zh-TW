6. <span data-ttu-id="2981a-121">如果您想要覆寫的指令碼檔案已存在於應用程式，選取**覆寫所有**核取方塊。</span><span class="sxs-lookup"><span data-stu-id="2981a-121">If you want to overwrite a script file that already exists in the application, select the **Overwrite all** check box.</span></span> <span data-ttu-id="2981a-122">指令碼檔案必須與新增檔案同名，才會被覆寫。</span><span class="sxs-lookup"><span data-stu-id="2981a-122">The script file must have the same file name as the one being added to be overwritten.</span></span> <span data-ttu-id="2981a-123">否則，加入新指令碼時，應用程式中的現有指令碼會維持不變。</span><span class="sxs-lookup"><span data-stu-id="2981a-123">Otherwise the new script will be added, and the existing script will remain in the application unchanged.</span></span>

>  <span data-ttu-id="2981a-127">如果您沒有提供這個路徑，在安裝時，就不會將指令碼複製到本機檔案系統。</span><span class="sxs-lookup"><span data-stu-id="2981a-127">If you do not provide this path, the script will not be copied to the local file system on installation.</span></span> <span data-ttu-id="2981a-128">如果這個指令碼應該會在應用程式解除安裝時執行，請務必提供這個路徑；否則，指令碼不會出現在本機檔案系統中，並且無法在解除安裝期間執行。</span><span class="sxs-lookup"><span data-stu-id="2981a-128">If the script should run when the application is uninstalled, be sure to provide this path; otherwise the script will not be present on the local file system and will not run during uninstallation.</span></span>
