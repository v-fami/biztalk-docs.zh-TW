# <a name="error---node-not-valid-for-property-field-promotion"></a>錯誤-節點不適用於屬性欄位升級
**錯誤碼**  
  
 BEC2001  
  
 **說明**  
  
 指定的節點無法升級為「屬性欄位」，因為它無法包含必要的文字內容。 只有**欄位項目**節點**欄位屬性**節點，或**記錄**帶有混合或簡單內容的節點，請升級為屬性欄位。  
  
 **使用者動作**  
  
 使用**屬性欄位**索引標籤中**升級屬性**對話方塊來移除不允許的屬性升級。
