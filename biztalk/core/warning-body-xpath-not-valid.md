# <a name="warning---body-xpath-not-valid"></a>警告-Body XPath 不正確
**錯誤碼**  
  
 BEC1009  
  
 **說明**  
  
 **Body XPath**此信封結構描述中指定的根節點屬性參考不是節點**記錄**節點，視需要。  
  
 **使用者動作**  
  
 選取指定的根節點，然後選取的值**Body XPath**屬性，以正確識別最上層**記錄**相關聯的訊息本文的節點。
