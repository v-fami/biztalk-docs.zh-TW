# <a name="developing-applications"></a>開發應用程式
本節包含專供想要建立 BizTalk Server 應用程式並使用 BizTalk Server 配接器的開發人員使用的資訊。 這些應用程式建立之後，通常由系統管理員，管理使用資訊中找到[管理 BizTalk Adapter for JD Edwards OneWorld](../core/administering-biztalk-adapter-for-jd-edwards-oneworld.md) > 一節。  
  
> [!NOTE]
>  對於 Microsoft Windows 平台上安裝的 JD Edward 系統，您必須使用 BizTalk Server 檔案配接器；對於 IBM AS400 系統上安裝的系統，您必須使用 JD Edwards 接收。
