# <a name="single-sign-on-event-10776"></a>單一登入： 事件 10776
## <a name="details"></a>詳細資料  
  
|                 |                                                                                                                                                       |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
|  產品名稱   |                                                               企業單一登入                                                               |
| 產品版本 |                                              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                               |
|    事件識別碼     |                                                                         10776                                                                         |
|  事件來源   |                                                                        ENTSSO                                                                         |
|    元件    |                                                                          不適用                                                                          |
|  符號名稱  |                                                            ENTSSO_E_INVALID_ACCOUNT_FORMAT                                                            |
|  訊息文字   | 帳戶名稱的格式不正確。 網域帳戶必須包含網域名稱。 本機帳戶不能包含網域或電腦的名稱。 |
  
## <a name="explanation"></a>說明  
 帳戶名稱的格式不正確。 網域帳戶必須包含網域名稱。 本機帳戶不能包含網域或電腦的名稱。  
  
## <a name="user-action"></a>使用者動作  
 請指定不同的帳戶名稱。
