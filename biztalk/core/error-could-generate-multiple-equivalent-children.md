# <a name="error---could-generate-multiple-equivalent-children"></a>錯誤-可能產生多個 Equivalent 子系
**錯誤碼**  
  
 btm1026  
  
 **說明**  
  
 目的結構描述的連結，不適當地啟用多個節點**相等**要產生的群組節點。  
  
 **使用者動作**  
  
 重新設定連結至目的結構描述，可能需要使用邏輯運算質，因此，只有單一節點內**相等**對應期間產生群組節點。
