# <a name="how-to-edit-link-properties"></a>如何編輯連結屬性
連結具有數個屬性，會出現在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]**屬性**視窗中的，如下所示：  
  
- **標籤**屬性提供連結的簡短描述，並在表格驅動迴圈特別有用。 如需**標籤**屬性，請參閱[如何管理現有連結](../core/how-to-manage-existing-links.md)。  
  
- **來源連結**屬性會決定此連結代表的是文字值的項目或項目的名稱。  
  
- **目標連結**屬性可讓您控制對應中處理連結的順序。  
  
  ![螢幕擷取畫面沒有圖說文字](../core/media/properties-grid.gif "Properties_Grid")  
  
  如需來源與目標結構描述的一般資訊，請參閱[編譯器指示詞和連結](../core/compiler-directives-and-links.md)。 如需設定和使用資訊**來源連結**並**目標連結**屬性，請參閱[如何設定節點階層比對](../core/how-to-configure-node-hierarchy-matching.md)並[如何設定來源連結編譯器值](../core/how-to-set-the-source-links-compiler-value.md)。
