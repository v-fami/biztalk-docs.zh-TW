# <a name="error---input-for-mass-copy-functoid-not-valid"></a>錯誤-無效的大量複製運算質的輸入
**錯誤碼**  
  
 btm1069  
  
 **說明**  
  
 相關的輸入的參數**大量複製**運算質不正確或是太多輸入的參數相關**大量複製**已指定運算質。 **大量複製**運算質必須只有一個輸入： 來自連結**記錄**來源結構描述中的節點。  
  
 **使用者動作**  
  
 重新設定相關**大量複製**運算質使其具有單一輸入的參數從**記錄**來源結構描述中的節點。
