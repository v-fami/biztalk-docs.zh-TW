# <a name="how-to-set-receive-pipelines-for-tibco-rendezvous"></a>如何設定 TIBCO Rendezvous 接收管線
Microsoft BizTalk Adapter for TIBCO Rendezvous 需要您設定接收處理常式和接收管線。  
  
### <a name="to-set-the-pipeline"></a>若要設定管線  
  
1.  設定**接收處理常式**至**BizTalkServerIsolatedHost**清單中。  
  
2.  設定**接收管線**至**XMLReceive**或任何同等的管線。  
  
3.  按一下 **[確定]**。
