# <a name="error-messages"></a>錯誤訊息
下表列出 PeopleSoft Enterprise 系統元件介面中的錯誤訊息及其說明，以及可能的解決方式。  
  
 **錯誤訊息**  
  
|錯誤識別碼|可能的原因 / 錯誤描述|可能的更正方式|  
|--------------|-----------------------------------------|-------------------------|  
|E-JNI0004|沒有 psjoa.jar。<br /><br /> 發生 Java 例外狀況。|確認 PeopleSoft psjoa.jar 檔案的位置。|  
|E-PSFT0030|沒有 psjoa.jar。<br /><br /> 無法產生元件介面 Bean。|確認 PeopleSoft psjoa.jar 檔案的位置。|  
|E-PSFT0019|錯誤的伺服器名稱。<br /><br /> 連線至 PeopleSoft Application Server 失敗。|確認 PeopleSoft 主控件與使用者參數。|  
|E-PSFT0024|使用者名稱與密碼有誤。<br /><br /> 連線失敗。 錯誤訊息： JavaClient 為無效的使用者名稱，或您輸入的密碼錯誤。|需要 PeopleSoft 使用者名稱與密碼，且需區分大小寫。 請確定您輸入大小寫格式正確的資訊。|
