# <a name="how-to-reconnect-to-a-group"></a>如何重新連接群組
您先前連線到您的 BizTalk server 群組可能有時會為 BizTalk Server 管理主控台中無法使用。 例如，當群組所在的電腦重新啟動時，該群組在管理主控台中可能會變成無法使用。 您可以藉由重新連線至無法使用的 BizTalk Server 群組**重新整理**則代表群組是無法使用。  
  
## <a name="prerequisites"></a>必要條件  
 若要執行此程序，您必須以「BizTalk Server 操作員」群組的成員登入，或以「BizTalk Server 系統管理員」群組的成員登入。  
  
### <a name="to-reconnect-to-a-group"></a>重新連接群組  
  
1.  按一下**啟動**，按一下 **所有程式**，按一下  [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然後按一下  **BizTalk Server 管理**。  
  
2.  在主控台樹狀結構中，按兩下 [[!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)]]。  
  
3.  以滑鼠右鍵按一下 **（無法使用）** 群組，然後再按一下**重新整理**。
