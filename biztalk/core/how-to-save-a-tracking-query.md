# <a name="how-to-save-a-tracking-query"></a>如何儲存追蹤查詢
您可以儲存查詢產生器中使用介面從 BizTalk Server 管理主控台的訊息或服務執行個體查詢。  
  
## <a name="prerequisites"></a>必要條件  
 您必須以「BizTalk Server 操作員」群組的成員身分登入，才能執行此程序。  
  
### <a name="to-save-a-tracking-query"></a>若要儲存追蹤查詢  
  
1. 按一下 **開始**，按一下**程式**，按一下  [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然後按一下**BizTalk Server 管理**。  
  
2. 在 **群組中樞**頁面上**新的查詢**索引標籤上，按一下 **另存新檔**儲存中的 SQL 命令**查詢運算式**區域。  
  
3. 在 [**另存查詢為**] 對話方塊中，瀏覽至您要儲存查詢的目的地目錄。 請務必包括查詢的適當檔案名稱。
