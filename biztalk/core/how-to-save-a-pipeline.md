# <a name="how-to-save-a-pipeline"></a>如何儲存管線
將管線組態的管線存放區資訊儲存至 .btp 檔案中，其中指定：  
  
-   管線名稱。  
  
-   作為依據的範本與原則 (傳送或接收)。  
  
-   屬於管線一部分的元件之順序，包含每個階段中使用的元件順序與集合。  
  
-   每個管線元件的組態資訊。  
  
### <a name="to-save-a-pipeline"></a>儲存管線  
  
1.  在 [檔案] 功能表上，按一下 [儲存]。  
  
     -或者-  
  
2.  按一下靠近畫面上方的索引標籤，在設計介面上顯示管線。  
  
3.  在**檔案**功能表上，按一下 **存**。  
  
4.  在**另存新檔**對話方塊中，於**檔案**名稱 欄位中，輸入管線的名稱。 已提供預設名稱。  
  
5.  按一下 **[確定]**。
