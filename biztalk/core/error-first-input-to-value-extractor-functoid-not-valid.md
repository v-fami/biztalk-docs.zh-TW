# <a name="error---first-input-to-value-extractor-functoid-not-valid"></a>錯誤-值擷取程式運算質不是有效的第一個輸入
**錯誤碼**  
  
 btm1002  
  
 **說明**  
  
 第一個輸入的參數所指定**值擷取程式 」**運算質必須是輸出 （ADO 資料錄集） 從**資料庫尋查**運算質。  
  
 **使用者動作**  
  
 請確認所有的第一個輸入**值擷取程式 」**運算質是從連結**資料庫尋查**其左邊的地圖格線頁中的運算質。
