# <a name="case-handling-in-flat-file-schemas"></a>在 一般檔案結構描述中處理的案例

## <a name="overview"></a>概觀
您可以使用**案例**屬性來從其對等的 XML 格式進行轉譯時執行的一般檔案資料的大小寫轉換。 當一般檔案組合器將 XML 訊息轉譯成其相等的一般檔案格式，而**案例**屬性會設為**大寫**或**小寫**，所有資料由對應的結構描述將會轉換成大寫或小寫，分別在轉譯過程。  
  
 當**案例**屬性設定為**（預設）**，任何大小寫轉換會執行。
