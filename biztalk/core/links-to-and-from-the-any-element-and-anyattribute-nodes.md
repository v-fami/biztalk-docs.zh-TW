# <a name="links-to-and-from-the-any-element-and-anyattribute-nodes"></a>Any 項目與 anyAttribute 節點的連結
您必須使用**大量複製**運算質至對應項目包含**任何**或**anyAttribute**節點。 BizTalk 對應工具不支援之間的直接連結**任何**或**anyAttribute**節點。 **任何**和**anyAttribute**節點，搭配**大量複製**運算質，最常用於對應 HTML 文件的區段。
