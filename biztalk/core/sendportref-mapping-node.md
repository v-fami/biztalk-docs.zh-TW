# <a name="sendportref-mapping-node"></a>SendPortRef (對應節點)
繫結檔案的 [對應] 節點的 SendPortRef 節點會列出父對應節點所參考的傳送埠的名稱。  
  
## <a name="nodes-in-the-sendportref-node"></a>SendPortRef 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|名稱|Attribute|xs:string|指定父對應節點所參考之傳送埠的名稱|不需要|預設值：空白|
