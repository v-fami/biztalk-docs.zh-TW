# <a name="managing-groups"></a>管理群組
本節提供管理 BizTalk 群組的程序。 您可使用 BizTalk Server 群組來代表組織的單位 (例如企業、部門或中樞)，這需要內含的 Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 實作。 BizTalk 管理資料庫 (BizTalkMgmtDb) 會儲存 BizTalk 群組及該群組中之伺服器的組態資訊。 如需 BizTalk 群組的詳細資訊，請參閱[BizTalk 群組](../core/biztalk-groups.md)。
