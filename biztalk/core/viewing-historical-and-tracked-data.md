# <a name="viewing-historical-and-tracked-data"></a>檢視歷程記錄和追蹤資料
本節提供使用如何在 BizTalk Server 管理主控台中檢視歷程記錄和追蹤資料，從 [群組中樞] 頁面的特定工作的資訊。  
  
-   顯示訊息所採取的處理步驟。 如需依訊息處理步驟的詳細資訊，請參閱[檢視訊息流程](../core/viewing-message-flow.md)。  
  
-   資料目前顯示在您的 BizTalk 追蹤資料庫以及從資料封存還原的資料。 詳細的資訊關於封存和追蹤資料，請參閱[如何選取即時或封存的資料來源](../core/how-to-select-a-live-or-archived-data-source.md)。  
  
-   限制資料存取的使用者擁有適當的權限。 如需資料存取的詳細資訊，請參閱[存取控制與資料安全性](../core/access-control-and-data-security.md)。  
  
-   可讓您執行即時偵錯您的協調流程。 如需詳細資訊即時偵錯，請參閱[使用協調流程偵錯工具檢視](../core/working-with-the-orchestration-debugger-view.md)。
