# <a name="this-scheduler-was-unable-to-schedule-the-batch"></a>這個排程器無法排程批次

|-|-|

|元件|批次處理引擎|  
|符號名稱|UnableToScheduleBatch|  
|訊息文字|這個排程器無法排程批次|  
  
## <a name="explanation"></a>說明  
 此錯誤表示排程器無法判斷下一個出現的批次釋放。  
  
## <a name="user-action"></a>使用者動作  
 若要解決這個錯誤，請確定排定的批次釋放是有效的。
