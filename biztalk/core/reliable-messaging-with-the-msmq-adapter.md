# <a name="reliable-messaging-with-the-msmq-adapter"></a>使用 MSMQ 配接器的可信賴傳訊
您可以使用 MSMQ 配接器，以特定組態設定和交易，來改善傳送和接收訊息的可靠度。  
  
> [!NOTE]
>  若您不使用交易，則在電腦當機時，可能會遺失叢集化組態中的訊息。
