# <a name="error---property-field-missing"></a>錯誤-屬性欄位遺失
**錯誤碼**  
  
 BEC2008  
  
 **說明**  
  
 此結構描述中的指定的節點正在升級至**欄位項目**對應不存在的屬性結構描述中節點。 會發生此狀況時**欄位項目**節點已移除或已使用做為目標的屬性升級之後，重新命名屬性結構描述中。  
  
 **使用者動作**  
  
 請修改屬性結構描述，使其包含遺漏**欄位項目**節點或使用**升級屬性**對話方塊來刪除或修改相關的屬性升級。
