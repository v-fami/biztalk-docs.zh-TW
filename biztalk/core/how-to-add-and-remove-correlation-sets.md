# <a name="how-to-add-and-remove-correlation-sets"></a>如何新增及移除相互關聯集
您可以使用 [協調流程檢視] 視窗，在協調流程中新增相互關聯集或是從中移除相互關聯集。 相互關聯集是相互關聯類型的執行個體。  
  
### <a name="to-add-and-configure-a-correlation-set"></a>新增及設定相互關聯集  
  
-   在協調流程檢視] 視窗中，以滑鼠右鍵按一下**相互關聯集**，然後按一下 [**新相互關聯集**。  
  
     **相互關聯屬性** 對話方塊隨即出現。 新增您要包含在相互關聯集內的屬性。 將會建立此相互關聯集以及對應的相互關聯類型。  
  
### <a name="to-remove-a-correlation-set"></a>移除相互關聯集  
  
-   在協調流程檢視] 視窗中，以滑鼠右鍵按一下您想要移除，然後按一下 [相互關聯集**刪除**。
