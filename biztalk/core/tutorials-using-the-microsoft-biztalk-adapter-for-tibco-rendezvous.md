# <a name="tutorials-using-the-microsoft-biztalk-adapter-for-tibco-rendezvous"></a>教學課程： 使用 Microsoft BizTalk Adapter for TIBCO Rendezvous
Microsoft BizTalk Adapter for TIBCO Rendezvous 是專為企業營運系統程序管理及自動化而設的開發與執行階段環境。 本節以兩個 SDK 範例來呈現設定與部署配接器的端對端練習，方便您快速入門。  
  
 這些教學課程包含詳細的逐步解說資訊，教導您如何新增配接器至 BizTalk Server、建立適當的傳送埠與接收埠、建置與部署範例，以及使用範例來處理測試資料。
