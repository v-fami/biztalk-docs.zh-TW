# <a name="how-to-view-an-individual-activity"></a>如何檢視個別活動
您可以在 BAM Web 入口網站的 [商務活動搜尋結果] 頁面中檢視活動。 如需搜尋活動的步驟，請參閱[如何搜尋活動](../core/how-to-search-for-activities.md)。  
  
### <a name="to-view-detailed-tracking-information"></a>檢視詳細的追蹤資訊  
  
1.  在 BAM Web 入口網站搜尋可用的活動後，請在 [商務活動搜尋結果] 頁面中按一下代表您要檢視之活動的傳回資料列。  
  
2.  [活動狀態] 頁面將會顯示關於所選取活動的詳細資訊。  
  
3.  在檢視一個商務活動後，您還可以檢視任何相關活動或相關文件。
