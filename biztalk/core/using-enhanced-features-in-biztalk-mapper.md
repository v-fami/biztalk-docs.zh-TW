# <a name="using-enhanced-features-in-biztalk-mapper"></a>在 BizTalk 對應工具中使用增強功能
BizTalk 對應工具引進一組使用性增強功能，可協助您處理複雜的對應。  
  
 雖然現有的 BizTalk 對應工具很適合用於處理小型與中型結構描述，但是對於大型與複雜的結構描述則效果不佳。 在此，全新的增強功能扮演很突出的角色。
