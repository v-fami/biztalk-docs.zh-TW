# <a name="xsd-resources-on-the-web"></a>在 Web 上的 XSD 資源
本文件並不提供 XML 結構描述定義 (XSD) 語言的詳細資訊，甚至是 XSD 的詳細說明，因其已超出本文件的範圍。 如需詳細資訊，直接移至 XSD 規格與 World Wide Web Consortium (W3C) 網站上維護的入門。 特別推薦您下列的教學課程，第一個是學習基本概念的快速方法，後面兩個則是關於 XSD 資訊之完整且可靠的來源。 如需 XSD 的教學課程，請參閱下列的 W3C 網頁：  
  
- XML 結構描述第 0 部分： Primer [ http://go.microsoft.com/fwlink/?LinkId=16787 ](http://go.microsoft.com/fwlink/?LinkId=16787)。  
  
- XML 結構描述第 1 部分： 結構[ http://go.microsoft.com/fwlink/?LinkId=16788 ](http://go.microsoft.com/fwlink/?LinkId=16788)。  
  
- XML Schema Part 2: Datatypes [ http://go.microsoft.com/fwlink/?LinkId=16789 ](http://go.microsoft.com/fwlink/?LinkId=16789)。  
  
  建議您在使用 BizTalk 編輯器來建構複雜的結構描述之前，先對 XSD 有深入的瞭解。
