# <a name="warning---body-xpath-not-a-descendent"></a>警告-Body XPath 不是子系
**錯誤碼**  
  
 BEC1004  
  
 **說明**  
  
 **Body XPath**此信封結構描述中指定的根節點的屬性參考不是該根節點，所需的子系的節點。 應該不會發生此錯誤除非**Body XPath**修改在 [BizTalk 編輯器] 以外的屬性。  
  
 **使用者動作**  
  
 選取指定的根節點，然後選取的值**Body XPath**正確識別關聯的訊息內文的最上層節點的屬性。
