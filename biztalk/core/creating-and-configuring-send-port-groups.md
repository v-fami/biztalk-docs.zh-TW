# <a name="creating-and-configuring-send-port-groups"></a>建立和設定傳送埠群組

## <a name="overview"></a>概觀
本節提供使用 BizTalk Server 管理主控台將傳送埠群組新增到 BizTalk 應用程式以及設定和刪除傳送埠群組的指示。 傳送埠群組是傳送埠的命名集合，可用來將相同的訊息傳送到多個目的地。 您可以將傳送埠群組繫結至協調流程連接埠，或是同單一傳送埠一樣，在以內容為基礎的路由選擇 (CBR) 中使用它們。 如需傳送埠群組的背景資訊，請參閱[傳送埠與傳送埠群組](../core/send-ports-and-send-port-groups.md)。  
  
> [!NOTE]
>  您可以使用 Microsoft Windows Management Instrumentation (WMI) 物件模型，建立和執行會自動化系統管理工作的指令碼。 如需使用 WMI 的詳細資訊，請參閱**WMI 類別參考** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
