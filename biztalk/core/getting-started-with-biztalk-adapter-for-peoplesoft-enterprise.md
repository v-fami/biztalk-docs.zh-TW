# <a name="getting-started-with-biztalk-adapter-for-peoplesoft-enterprise"></a>BizTalk Adapter for PeopleSoft Enterprise 入門
Microsoft BizTalk Adapter for PeopleSoft Enterprise 可讓您使用 PeopleSoft 物件。 本節討論如何設定配接器以存取 PeopleSoft 特定資訊。 同時還提供 PeopleSoft 環境的一般介紹。  
  
 PeopleSoft 是一套包含 PeopleSoft 元件的系統，並具備相關的資料表及資料格式可供特定工作使用，例如人力資源或庫存管理。 本文件提供了有關下列工作的資訊：  
  
-   設定 BizTalk Adapter for PeopleSoft Enterprise。  
  
-   自訂您的 PeopleSoft 系統以與配接器互動。  
  
-   存取元件介面，並在您的協調流程中使用它們。  
  
 BizTalk Adapter for PeopleSoft Enterprise 已經使用下列 PeopleSoft PeopleTools 版本測試過：  
  
 8.17.02、8.43 和 8.45 - 元件介面
