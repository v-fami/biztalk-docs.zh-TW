# <a name="x12-edi-character-set"></a>X12 EDI 字元集
使用及參與字元或抑音符號 （'） 時，指定下列項目：  


|                                                                   |                                  字元集                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------|
|             只有在 EDI 文件及參與字元              |                            使用擴充的字元集                            |
|           只有抑音符號 (\`) 中的 EDI 文件            |                              使用 UTF8 字元集                              |
| 及參與字元**並**抑音符號 (\`) 相同的文件中： | -輸入文件必須具有 UTF8 編碼方式<br />-使用 UTF8 字元集 |
