# <a name="net-object-facts"></a>.NET 物件事實
在「商務規則編輯器」中，您可以指定 .NET 組件做為資料來源。 然後從組件選取類別或類別成員，將它拖曳至詞彙定義或規則中。  
  
 判斷提示的 .NET 物件處理是最直接的處理類型。 引擎會將各個物件視為唯一執行個體而進行處理。 如需有關**Assert**函式，請參閱[引擎控制函式](../core/engine-control-functions.md)。
