# <a name="flat-file-disassembler-pipeline-component"></a>一般檔案解譯器管線元件
「一般檔案解譯器」元件會剖析分隔式與序數一般檔案格式訊息，並將它們轉換為 XML 表示法。 「一般檔案解譯器」也會移除一般檔案訊息的標頭與結尾結構，以及將訊息內的交換獨立成個別文件。 它還可以從文件與標頭升級屬性。  
  
 如需一般檔案的詳細資訊，請參閱[具有位置記錄的一般檔案訊息](../core/flat-file-messages-with-positional-records.md)。 另請參閱[具有分隔記錄的一般檔案訊息](../core/flat-file-messages-with-delimited-records.md)。  
  
 設定一般檔案解譯器管線元件的相關資訊，請參閱[如何設定一般檔案解譯器管線元件](../core/how-to-configure-the-flat-file-disassembler-pipeline-component.md)。
