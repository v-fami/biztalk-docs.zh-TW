|                 |                                                               |
|-----------------|---------------------------------------------------------------|
|  產品名稱   |                   企業單一登入                   |
| 產品版本 |  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    事件識別碼     |                             10771                             |

|    元件    |                              不適用                              |
|  符號名稱  |                      ENTSSO_E_NOT_ADMIN                       |
|  訊息文字   | 只有本機系統管理員可以執行此函式。 |
  
## <a name="explanation"></a>說明  
 某些函式只能由系統管理員在本機電腦上執行。

 有系統管理員協助您連絡這台電腦的本機系統管理員。
