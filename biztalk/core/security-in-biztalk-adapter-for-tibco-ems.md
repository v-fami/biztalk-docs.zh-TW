# <a name="security-in-biztalk-adapter-for-tibco-ems"></a>在 BizTalk Adapter for TIBCO EMS 中的安全性

## <a name="overview"></a>概觀
Microsoft BizTalk Adapter for TIBCO Enterprise Message Service 提供單一登入 (SSO) 支援。 由企業單一登入工具所建立的分支機構應用程式，代表像是 TIBCO Enterprise Message Service 的伺服器系統。  

本節提供在安全環境中部署 BizTalk Adapter for TIBCO Enterprise Message Service 的指導方針。  
  
> [!CAUTION]
>  強烈建議您限制只有，授權使用者才能使用 BizTalk Adapter for Enterprise Message Service 的特定業務應用程式在直接連接的用戶端檔案。
