# <a name="single-sign-on-event-11067"></a>單一登入： 事件 11067
## <a name="details"></a>詳細資料  
  
|                 |                                                                                     |
|-----------------|-------------------------------------------------------------------------------------|
|  產品名稱   |                              企業單一登入                              |
| 產品版本 |             [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]              |
|    事件識別碼     |                                        11067                                        |
|  事件來源   |                                       ENTSSO                                        |
|    元件    |                                         不適用                                         |
|  符號名稱  |                      SSO_ERROR_PASSWORD_SYNC_MIIS_START_FAILED                      |
|  訊息文字   | （來自 MIIS) 的 Windows 密碼同步處理無法 start.%r<br /><br /> 錯誤碼： %1 |
  
## <a name="explanation"></a>說明  
 這是資訊訊息，指出 Windows （來自 MIIS) 的密碼同步無法啟動。  
  
## <a name="user-action"></a>使用者動作  
 使用者不必採取任何動作。
