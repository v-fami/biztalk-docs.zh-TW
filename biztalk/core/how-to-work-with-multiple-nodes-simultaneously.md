# <a name="how-to-work-with-multiple-nodes-simultaneously"></a>如何同時使用多個節點
有些作業可以同時在多個節點上執行，但是必須選取所有相關的節點。 本主題提供如何同時使用多個節點的指示。  
  
### <a name="to-select-multiple-nodes-that-are-not-adjacent"></a>選取多個不相鄰的節點  
  
1.  在要同時選取的不相鄰節點集合中選取第一個節點。  
  
2.  按住 CTRL，然後在要同時選取的不相鄰節點集合中選取其餘的節點。  
  
     即可同時選取所有不相鄰的節點。  
  
### <a name="to-select-a-range-of-adjacent-nodes"></a>選取一定範圍的相鄰節點  
  
1.  選取位於要同時選取之相鄰節點集合的前端節點。  
  
2.  按住 SHIFT 鍵，然後選取要同時選取之節點集合的另一端節點。  
  
     如此即可同時選取步驟 1 與步驟 2 中選取之節點所定義的相鄰節點範圍。  
  
### <a name="to-modify-property-values-of-simultaneously-selected-nodes"></a>修改同時選取之節點的屬性值  
  
1.  < 若要選取多個不相鄰的節點 」 程序中所述，請選取多個節點。 稍早在這份文件。  
  
2.  修改中的相關屬性值**屬性**視窗。  
  
> [!NOTE]
>  只有所有選取節點共有的屬性才能同時變更。
