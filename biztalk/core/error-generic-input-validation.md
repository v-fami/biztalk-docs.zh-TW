# <a name="error---generic-input-validation"></a>錯誤-一般輸入驗證
**錯誤碼**  
  
 btm1045  
  
 **說明**  
  
 由於未指定的原因，無法驗證「測試對應」作業指定的輸入執行個體訊息檔案是否符合來源結構描述。  
  
 **使用者動作**  
  
 在 [BizTalk 編輯器] 中開啟來源結構描述，並使用**驗證結構描述**和**產生執行個體**可用的命令，當您以滑鼠右鍵按一下 [方案總管] 中，找出問題的結構描述。
