# <a name="edi-message-content-status-report"></a>EDI 訊息內容狀態報告

## <a name="overview"></a>概觀
此狀態報告會顯示交易集的標頭和內容。 這個報表顯示以滑鼠右鍵按一下 交易集內交易集詳細資料狀態報告中，然後按一下**檢視交易集內容**。  
  
 此報表是您已選取時，才可以使用**儲存交易集/內容 reporting**相關合作對象 [EDI 屬性] 對話方塊的 [一般] 頁面中的屬性。  
  
 此報告會提供兩個內容檢視：  
  
-   [文字格式] 檢視：以一般人可讀取的格式顯示交易集的內容，就像 EDI 文字檔案一樣。 這個檢視會顯示 ST 標頭、交易集內容和 SE 結尾，每個區段一行。  
  
-   [二進位格式] 檢視：以不分隔的文字格式顯示交易集的內容，交易集的每個 ASCII 字元一個十六進位表示的資料表。
