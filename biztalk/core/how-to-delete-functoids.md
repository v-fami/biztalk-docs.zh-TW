# <a name="how-to-delete-functoids"></a>如何刪除運算質
當開發對應時，您有時會需要刪除現有的運算質。 刪除運算質也會刪除關聯的輸入和輸出連結，以及任何其他輸入參數、指令碼規格和資料表格線組態。  
  
### <a name="to-delete-an-existing-functoid"></a>刪除現有的運算質  
  
1.  顯示包含您要刪除的運算質之格線頁。 如需使用格線頁，包括在格線頁之間切換的詳細資訊，請參閱[使用格線頁](../core/working-with-grid-pages.md)。  
  
2.  按一下以選取您要刪除的運算質。  
  
     您可以選取多個運算質，然後一次將它們刪除。 如需大量選取運算質和連結的詳細資訊，請參閱[如何選取多個連結和運算質](../core/how-to-select-multiple-links-and-functoids.md)。  
  
3.  在**編輯**功能表上，按一下 **刪除**。  
  
    > [!NOTE]
    >  您可以也按下 DELETE 鍵，或以滑鼠右鍵按一下相關格線頁中的所選取運算質，然後按一下**刪除**從內容功能表。  
  
    > [!NOTE]
    >  您無法復原刪除運算質或連結的動作。 如需有關如何復原或重做使用者作業的詳細資訊，請參閱[如何復原或重做使用者作業](../core/how-to-undo-or-redo-user-operations.md)。
