<span data-ttu-id="f003a-103">在開發 BizTalk 對應，您可能想要變更來源或目的結構描述樹狀檢視的大小或格線檢視的大小。</span><span class="sxs-lookup"><span data-stu-id="f003a-103">In the process of developing BizTalk maps, you may want to change the sizes of the source or destination schema tree views, or the size of the grid view.</span></span> <span data-ttu-id="f003a-104">有時候，您可能需要在先前關閉的檢視中開啟對應。</span><span class="sxs-lookup"><span data-stu-id="f003a-104">Sometimes, you may want to open your map in the same view you had closed earlier.</span></span> <span data-ttu-id="f003a-105">本主題提供這些工作的逐步解說。</span><span class="sxs-lookup"><span data-stu-id="f003a-105">This topic provides step-by-step instructions for these tasks.</span></span>  
  
## <a name="prerequisites"></a><span data-ttu-id="f003a-106">必要條件</span><span class="sxs-lookup"><span data-stu-id="f003a-106">Prerequisites</span></span>  
 <span data-ttu-id="f003a-107">這些指示需要執行中的 BizTalk 對應工具。</span><span class="sxs-lookup"><span data-stu-id="f003a-107">These instructions require that BizTalk Mapper is running.</span></span>  
  
### <a name="to-make-the-schema-tree-views-or-the-grid-view-taller-or-shorter"></a><span data-ttu-id="f003a-108">若要使結構描述樹狀檢視或格線檢視較高或較矮</span><span class="sxs-lookup"><span data-stu-id="f003a-108">To make the schema tree views or the grid view taller or shorter</span></span>  
  
1.  <span data-ttu-id="f003a-109">移動滑鼠游標至主要編輯視窗的下邊緣 (在格線檢視兩旁會顯示結構描述樹狀檢視)，直到游標變成標準的視窗垂直調整大小圖示。</span><span class="sxs-lookup"><span data-stu-id="f003a-109">Move the mouse pointer to the bottom edge of the main editing window (which displays the schema tree views on either side of the grid view) until the cursor changes to the standard window vertical resizing icon.</span></span>  
  
2.  <span data-ttu-id="f003a-110">按住滑鼠左鍵並上下拖曳視窗邊緣。</span><span class="sxs-lookup"><span data-stu-id="f003a-110">Click and hold the left mouse button, and then drag the window edge either up or down.</span></span>  
  
     <span data-ttu-id="f003a-111">透過對整個主要編輯視窗進行垂直大小的重新調整，您已重新垂直調整結構描述樹狀檢視和格線檢視的大小。</span><span class="sxs-lookup"><span data-stu-id="f003a-111">You have vertically resized the schema tree views and the grid view by vertically resizing the entire main editing window.</span></span>  
  
### <a name="to-make-the-schema-tree-views-or-the-grid-view-wider-or-more-narrow"></a><span data-ttu-id="f003a-112">若要使結構描述樹狀檢視或格線檢視較寬或較窄</span><span class="sxs-lookup"><span data-stu-id="f003a-112">To make the schema tree views or the grid view wider or more narrow</span></span>  
  
1.  <span data-ttu-id="f003a-113">移動滑鼠游標至主要編輯視窗中兩條窗格分割線 (用來分割結構描述樹狀檢視和格線檢視) 的其中之一，直到游標變成標準的視窗水平調整大小圖示。</span><span class="sxs-lookup"><span data-stu-id="f003a-113">Move the mouse pointer to one of the two pane dividers in the main editing window (which divides the schema tree views from the grid view between them) until the cursor changes to the standard window horizontal resizing icon.</span></span>  
  
2.  <span data-ttu-id="f003a-114">按住滑鼠左鍵並左右拖曳 (調整寬窄) 窗格邊緣。</span><span class="sxs-lookup"><span data-stu-id="f003a-114">Click and hold the left mouse button, and then drag the pane edge either left (narrower) or right (wider).</span></span>  
  
     <span data-ttu-id="f003a-115">透過變更主要編輯視窗分配給結構描述樹狀檢視的大小，您已重新水平調整結構描述樹狀檢視和格線檢視的大小。</span><span class="sxs-lookup"><span data-stu-id="f003a-115">You have horizontally resized one of the schema tree views and the grid view by changing the amount of the main editing window dedicated to each of these views.</span></span>  
  
     <span data-ttu-id="f003a-116">或者，您也可以透過對整個主要編輯視窗進行水平大小的重新調整，來使結構描述樹狀檢視和格線檢視變得較寬或較窄。</span><span class="sxs-lookup"><span data-stu-id="f003a-116">Alternatively, you can also make the schema tree views and the grid view wider or narrower by horizontally resizing the entire main editing window.</span></span>
