# <a name="appendix-a-sample-files"></a>附錄 a： 範例檔案
Microsoft BizTalk Adapter for JD Edwards EnterpriseOne 支援 jdeRequest 和 jdeResponse XML 結構，以便在 JD Edwards EnterpriseOne 中執行商務功能。  
  
 您可以使用 JD Edwards EnterpriseOne XML 執行下列動作：  
  
- 將商務功能呼叫彙總成單一物件。  
  
- 使用 JD Edwards EnterpriseOne ThinNet API。  
  
- 存取 Z 檔案和商務功能。  
  
  本節提供 jdeRequest 與 jdeResponse XML 結構的範例，以便在 JD Edwards EnterpriseOne 中執行商務功能。
