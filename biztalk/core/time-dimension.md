# <a name="time-dimension"></a>時間維度
時間維度可用來建立與時間相關的彙總。 例如，時間維度可以用來建立下列資料表：  
  
|Year|Month|Count|  
|----------|-----------|-----------|  
|2003|一月|120|  
||二月|230|  
||三月|350|  
||四月|280|  
  
 時間維度可以和任何其他維度結合。 例如，您可以在資料列使用時間維度，而在資料行使用資料維度，以建立下列資料表：  
  
|||網球拍|足球|  
|------|------|---------------------|------------------|  
||一月|50|70|  
||二月|120|110|  
||三月|300|50|  
||四月|220|60|
