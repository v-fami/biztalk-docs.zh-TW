# <a name="building-and-running-orchestrations"></a>建置和執行協調流程
在設計協調流程後，您可以編譯並執行它來測試完整性和正確性。  
  
 在您的協調流程編譯無誤後，您可以先在測試環境中部署，再進行任何必要的修正，最後再將它部署至生產環境中執行。
