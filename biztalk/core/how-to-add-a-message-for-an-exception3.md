# <a name="how-to-add-a-message-for-an-exception"></a>如何新增例外狀況訊息
請依照下列步驟來新增含有訊息的錯誤。  
  
### <a name="to-add-a-message-for-an-exception"></a>若要新增例外狀況的訊息  
  
1.  在**協調流程檢視**視窗中，以滑鼠右鍵按一下**訊息**選取**新訊息**。  
  
     Message_3 隨即建立，您可將它專門指派給該錯誤。  
  
2.  以滑鼠右鍵按一下**Message_3**選取**屬性**。  
  
3.  在**屬性** 對話方塊中，將**訊息類型**。  
  
     選取 **.NET 類別**，然後選取 **[systemstring]**。
