# <a name="wcf-adapters-property-schema-and-properties"></a>WCF 配接器屬性結構描述和屬性
下表包含 WCF 配接器屬性結構描述中的升級屬性。 WCF 配接器會為您可以在應用程式中使用的屬性指派值。 WCF 配接器也提供了一種將自訂屬性寫入但不升級至 BizTalk 訊息內容的機制，以及一種將自訂屬性升級至 BizTalk 訊息內容的機制。 如需詳細資訊，請參閱[SOAP 標頭與已發佈 WCF 服務](../core/soap-headers-with-published-wcf-services.md)。
