# <a name="what-is-a-real-time-aggregation"></a>什麼是即時彙總？
商務分析師可以將您的樞紐分析表定義為即時彙總 (RTA) 或排程彙總。 RTA 將提供您資料的最新檢視；例如，特定 PO 在出貨程序中的情況。 您可以重新整理畫面來更新全天的資料檢視。
