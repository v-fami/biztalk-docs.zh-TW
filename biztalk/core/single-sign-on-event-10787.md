# <a name="single-sign-on-event-10787"></a>單一登入： 事件 10787
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10787|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_FLAGS_NOT_VALID|  
|訊息文字|指定的旗標無效或互不相容。 請參閱事件記錄檔 （在電腦 '%1'），如需詳細資訊。|  
  
## <a name="explanation"></a>說明  
 指定的旗標無效或互不相容。  
  
## <a name="user-action"></a>使用者動作  
 請檢查事件記錄檔以查看此應用程式的有效和無效的旗標，指定的電腦上和相關的錯誤。
