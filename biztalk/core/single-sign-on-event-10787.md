|                 |                                                                                                                               |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------|
|  產品名稱   |                                                   企業單一登入                                                   |
| 產品版本 |                                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                   |
|    事件識別碼     |                                                             10787                                                             |
|  事件來源   |                                                            ENTSSO                                                             |
|    元件    |                                                              不適用                                                              |
|  符號名稱  |                                                   ENTSSO_E_FLAGS_NOT_VALID                                                    |
