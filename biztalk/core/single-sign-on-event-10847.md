|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10847|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_MAPPINGS_ONLY_FOR_DOMAIN_USER_ACCOUNTS|  
|訊息文字|個別的應用程式，建立對應的 Windows 網域僅限使用者 （非本機帳戶或群組帳戶）。|  
  
## <a name="explanation"></a>說明
