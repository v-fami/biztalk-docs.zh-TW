# <a name="tibco-rendezvous-adapter"></a>TIBCO Rendezvous 配接器

[開始](../core/getting-started-with-biztalk-adapter-for-tibco-rendezvous.md)描述不同的訊息和概念，並也提供安裝、 結構描述和限制的詳細資料。 也請逐步執行某些教學課程，可接收和傳送資料，使用此配接器。

## <a name="architecture"></a>Architecture
[架構](../core/architecture-of-biztalk-adapter-for-tibco-rendezvous.md)描述連線能力，以及如何傳遞訊息。
