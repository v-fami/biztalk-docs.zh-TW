# <a name="error---could-generate-multiple-choice-children"></a>錯誤-可能產生多個 Choice 子系
**錯誤碼**  
  
 btm1027  
  
 **說明**  
  
 目的結構描述的連結，不適當地啟用多個節點**選擇**要產生的群組節點。  
  
 **使用者動作**  
  
 重新設定連結至目的結構描述，因此，只有單一節點內**選擇**產生群組節點。
