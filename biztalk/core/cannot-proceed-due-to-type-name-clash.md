## <a name="explanation"></a>說明  
 此錯誤表示在相同定義的命名空間中的多個成品具有相同的名稱。  
  
## <a name="user-action"></a>使用者動作  
 重新命名相同的命名空間中的成品，或將它們放在不同的命名空間。
