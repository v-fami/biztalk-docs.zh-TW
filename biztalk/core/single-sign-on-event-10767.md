# <a name="single-sign-on-event-10767"></a>單一登入： 事件 10767
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10767|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_NO_SSO_SERVER|  
|訊息文字|無法連絡 SSO 伺服器 '%1'。 請檢查是否已設定 SSO，以及 SSO 服務是否在該伺服器上執行。|  
  
## <a name="explanation"></a>說明  
 ENTSSO 用戶端是無法連絡 ENTSSO 伺服器。  
  
## <a name="user-action"></a>使用者動作  
 請檢查網路連線，並確定伺服器名稱的拼字正確。
