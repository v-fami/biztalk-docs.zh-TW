# <a name="jd-edwards-oneworld-adapter"></a>JD Edwards OneWorld 配接器
Microsoft BizTalk Adapter for JD Edwards OneWorld 可讓您在 BizTalk Server 中使用 JD Edwards OneWorld 商務功能。 下列幾節討論設定 BizTalk Adapter for JD Edwards OneWorld 以存取 JD Edwards OneWorld 特定資訊。  
  
## <a name="get-started"></a>快速入門 
[開始](../core/getting-started-with-biztalk-adapter-for-jd-edwards-oneworld.md)列出安裝介面卡，並逐步執行教學課程的步驟。

## <a name="architecture"></a>Architecture
[架構](../core/planning-and-architecture17.md)詳細列出配接器的運作方式、 如何執行個體和函式會構成集區在設計階段和執行的階段，限制查詢及擷取清單，當使用多訂單項目，與此配接器。

## <a name="security"></a>安全性
[BizTalk Adapter for JD Edwards OneWorld 中的安全性](../core/security-in-biztalk-adapter-for-jd-edwards-oneworld.md)內 BizTalk 伺服器來保護您使用此配接器的應用程式牽涉到使用企業單一登入 (SSO)。

## <a name="create-the-artifacts"></a>建立的成品
[建立應用程式成品](../core/developing-applications3.md)示範如何在 BizTalk 管理中，然後在 Visual Studio 中，將成品新增。 它也會顯示如何在協調流程中使用訊息內容屬性。

## <a name="import-your-app"></a>匯入您的應用程式
[部署 BizTalk Adapter for JD Edwards OneWorld](../core/deploying-biztalk-adapter-for-jd-edwards-oneworld.md)討論如何將 BizTalk 管理中，匯入您的應用程式的繫結檔案並超過限制。 

## <a name="exception-handling"></a>例外狀況處理 
[BizTalk Server 例外狀況處理](../core/using-biztalk-server-exception-handling1.md)提供更新 jdearglist.txt 檔案，並將不同的圖形加入至您的協調流程處理的例外狀況的指引。
