# <a name="how-to-modify-general-settings"></a>如何修改一般設定
您可以使用「設定儀表板」來修改整個 BizTalk 群組中某個主控件的一般組態設定。 本主題提供執行此作業的逐步程序。  

## <a name="prerequisites"></a>必要條件  
 若要執行此作業，您必須以 BizTalk Server Administrators 群組的成員身分登入。  

### <a name="to-modify-the-general-settings-of-a-host"></a>若要修改主控件的一般設定  

1. 在  **BizTalk Server 管理主控台**，展開[!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)]，以滑鼠右鍵按一下**BizTalk 群組**，然後按一下**設定**。  

2. 在  **BizTalk 設定儀表板**對話方塊的 **主機**頁面上，按一下**一般** 索引標籤。
