# <a name="security-in-jd-edwards-oneworld"></a>JD Edwards OneWorld 中的安全性

## <a name="overview"></a>概觀
Microsoft BizTalk Adapter for JD Edwards OneWorld 提供單一登入 (SSO) 支援 。 由企業單一登入工具所建立的分支機構應用程式，代表像是 JD Edwards OneWorld 的伺服器系統。  

本節提供如何在安全環境中部署 Microsoft BizTalk Adapter for JD Edwards OneWorld 的指導方針。  
  
> [!CAUTION]
>  我們建議您限制只有，授權使用者才能使用 BizTalk Adapter for JD Edwards OneWorld 特定業務應用程式在直接連接的用戶端檔案。
