# <a name="how-to-open-a-pipeline"></a>如何開啟管線
開啟管線檔會顯示「管線設計師」，並在設計區域中顯示已定義的管線及其階段。 此外，若尚未開啟 [管線設計師工具箱與屬性] 視窗，則會顯示該視窗。  
  
### <a name="to-open-a-pipeline"></a>開啟管線  
  
1.  在 [方案總管] 中，按兩下您要開啟的管線。  
  
     -或者-  
  
2.  在 方案總管 中，選取管線，然後在**檢視**功能表上，按一下 **開啟**。
