# <a name="single-sign-on-event-10831"></a>單一登入： 事件 10831
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10831|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_SSO_DATABASE_EXISTS|  
|訊息文字|指定的 SSO 資料庫已經存在。|  
  
## <a name="explanation"></a>說明  
 在設定期間，您嘗試建立已存在的資料庫。  
  
## <a name="user-action"></a>使用者動作  
 請檢查您的系統，或提供您新的資料庫不同的名稱。
