# <a name="error---duplicate-property-field-promotion"></a>錯誤-重複的屬性欄位升級
**錯誤碼**  
  
 BEC2016  
  
 **說明**  
  
 此結構描述中的多個節點會做為屬性欄位升級為相同**欄位項目**相同的屬性結構描述中的節點。  
  
 **使用者動作**  
  
 使用**屬性欄位** 索引標籤**升級屬性**對話方塊中，透過存取**升級屬性**屬性**結構描述**節點，以刪除所有只保留一個具有相同相關聯的屬性欄位升級**欄位項目**屬性結構描述中的節點。 您可能想要加入新**欄位項目**屬性結構描述的節點，讓已刪除的升級可以重新升級為它們自己**欄位項目**節點。
