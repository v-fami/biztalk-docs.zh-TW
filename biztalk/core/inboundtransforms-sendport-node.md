# <a name="inboundtransforms-sendport-node"></a>InboundTransforms (SendPort 節點)
繫結檔案之 [SendPort] 節點的 [InboundTransforms ] 節點包含與該繫結檔案匯出之雙向傳送埠關聯的輸入轉換集合。  
  
## <a name="nodes-in-the-inboundtransforms-node"></a>InboundTransforms 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[轉換 （InboundTransforms 節點）](../core/transform-inboundtransforms-node.md)|記錄|Transform (ComplexType)|指定 BizTalk Server 對應 (或轉換)，這個項目代表來源結構描述和目的結構描述之間的對應。|不需要|預設值：無|
