# <a name="how-to-enable-soap-message-tracing"></a>如何啟用 SOAP 訊息追縱
您可以啟用 SOAP 訊息追蹤，協助您偵錯 Web 服務發佈應用程式。 您會在已發佈的 Web 服務中設定中斷點，藉以傳回詳細例外狀況給 Web 用戶端。 若要追蹤 SOAP 訊息，您必須使用 SOAP 延伸模組。 如需 SOAP 延伸模組的詳細資訊，請參閱 < 如何以:: 實作 SOAP 延伸模組 > 網址[http://go.microsoft.com/fwlink/?LinkId=62314](http://go.microsoft.com/fwlink/?LinkId=62314)。
