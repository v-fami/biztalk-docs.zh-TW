# <a name="how-to-manage-the-properties-window"></a>如何管理屬性視窗
BizTalk 編輯器使用 [Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 屬性] 視窗，以顯示結構描述樹狀結構檢視中與節點關聯的屬性值，並提供變更這些值的介面。 本主題針對管理 [屬性] 視窗的某些特性，提供逐步解說。  
  
### <a name="to-turn-property-descriptions-on-and-off"></a>開啟和關閉屬性描述  
  
-   在 屬性 視窗中任何屬性的名稱，以滑鼠右鍵按一下，然後按一下 **描述**。  
  
    > [!NOTE]
    >  使用此程序來切換開啟和關閉屬性的顯示。  
  
### <a name="to-arrange-properties-alphabetically-within-their-categories"></a>在其類別中依照字母順序排列屬性  
  
-   按一下**分類**[屬性] 視窗中的屬性清單上方的工具列按鈕。  
  
### <a name="to-arrange-properties-alphabetically-without-regard-to-their-categories"></a>不論類別，只依照字母順序排列屬性  
  
-   按一下**字母順序**[屬性] 視窗中的屬性清單上方的工具列按鈕。
