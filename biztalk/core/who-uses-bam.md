# <a name="who-uses-bam"></a>誰會使用 BAM？
有兩種類型的互動的使用者與 BAM 的資訊工作者功能： 商務分析師和商務使用者。  
  
 商務分析師使用 BAM Excel 增益集來定義他們想要監控的商務程序。 還會使用 BAM Excel 增益集來定義商務使用者查看 BAM 所收集資料的方式。 例如，商務分析師想要提供關於訂單的資料，並就訂單提出了以下有趣的問題：  
  
-   公司在這段期間 (年、月、週、日、小時等) 收到多少訂單？  
  
-   目前有效訂單的金額為何？  
  
-   訂單通過各訂單程序階段所需的平均時間為何？  
  
-   特定程序階段中有多少訂單？  
  
-   有多少客戶訂購特定產品？  
  
-   特定產品的訂單加總為何？  
  
-   地理區域之間的訂購數量是否不同？  
  
 商務分析師使用這些問題來定義活動，反映 BAM 應該從商務程序收集的資訊類型。  
  
 商務使用者使用 BAM 來檢視由商務分析師所定義而由 BAM 所收集的資料。 商務使用者可以使用 BAM 的方式如下：  
  
-   檢視程序的單一執行個體的資料。 例如，如果商務分析師建立活動來監控訂單程序，商務使用者就可以使用 BAM 來查看關於特定訂單的資料。  
  
-   檢視相關活動執行個體的資料。 例如，與特定訂單關聯的發票或出貨。  
  
-   檢視有關程序的即時或封存彙總資料。 彙總資料是經過一段時間的資料。 例如，使用者可能會檢視上一期間 (週、月、年等) 所收到金額小於 $500 的訂單件數，或今日到達的訂單平均金額。  
  
-   檢視進度資料，允許使用者查看程序各不同階段的資料，例如訂單從接收階段轉移到履行階段所花費的平均時間。
