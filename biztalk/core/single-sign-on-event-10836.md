# <a name="single-sign-on-event-10836"></a>單一登入： 事件 10836
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10836|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_FILE_PASSWORD_TOO_SHORT|  
|訊息文字|指定的密碼太短，它必須在至少 7 個字元。|  
  
## <a name="explanation"></a>說明  
 備份檔案的密碼必須至少七個字元。  
  
## <a name="user-action"></a>使用者動作  
 輸入密碼的長度至少七個字元。
