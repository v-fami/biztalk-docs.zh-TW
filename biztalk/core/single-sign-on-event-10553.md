# <a name="single-sign-on-event-10553"></a>單一登入： 事件 10553
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10553|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|SSO_ERROR_ADMIN_CALLBACK_ACCESS_DENIED|  
|訊息文字|管理伺服器存取 denied.%r|  
  
## <a name="explanation"></a>說明  
 呼叫已從用戶端對管理伺服器，但不是被接受。 原因可能是由數種不同原因，例如不正確的通訊協定或用戶端上沒有足夠的安全性權限。  
  
## <a name="user-action"></a>使用者動作  
 請注意在錯誤記錄檔的資訊，請連絡產品支援服務。
