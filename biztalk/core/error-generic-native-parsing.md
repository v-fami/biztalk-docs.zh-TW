# <a name="error---generic-native-parsing"></a>錯誤-一般原生剖析
**錯誤碼**  
  
 btm1042  
  
 **說明**  
  
 無法剖析針對「測試對應」作業指定的原生輸入執行個體訊息檔案，且發生一般嚴重剖析錯誤。  
  
 **使用者動作**  
  
 視需要更正指定檔案中，與對應關聯的來源結構描述或原生輸入執行個體訊息，或兩者。 考慮在 BizTalk 編輯器中工作以隔離問題。 **驗證結構描述**和**產生執行個體**可用的命令，當您以滑鼠右鍵按一下 [方案總管] 中的結構描述可用於尋找結構描述錯誤。
