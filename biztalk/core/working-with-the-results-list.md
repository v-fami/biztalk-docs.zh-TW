# <a name="working-with-the-results-list"></a>使用結果清單
當您在 [群組中樞] 頁面中執行查詢時，它會產生**查詢結果**包含搜尋結果中的，例如相關訊息、 管線和組件清單。  
  
> [!NOTE]
>  若要排序 [結果] 清單，請按一下任何欄標題。 您每按一下，清單的顯示方式就會在欄值字母/數值的遞增與遞減順序之間切換。
