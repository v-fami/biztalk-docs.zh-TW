# <a name="error---failure-writing-csharp-file-ltfile---0gt-error-1"></a>錯誤-寫入 CSharp 檔案時失敗 (&lt;file:---{0}&gt;)。 錯誤： {1}
**錯誤碼**  
  
 btm1060  
  
 **說明**  
  
 當對應工具編譯器無法在建置工作指定的位置中建立 csharp 檔案時，便會發生錯誤。  
  
 **使用者動作**  
  
 請檢查包含在訊息 (錯誤 {1}) 中的錯誤訊息。 如果這是未獲授權的例外狀況，就表示您可能沒有專案輸出資料夾的寫入權限。
