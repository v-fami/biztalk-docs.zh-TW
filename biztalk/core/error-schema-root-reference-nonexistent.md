# <a name="error---schema-root-reference-nonexistent"></a>錯誤-結構描述根參考不存在
**錯誤碼**  
  
 BEC2006  
  
 **說明**  
  
 **根參考**屬性**結構描述**節點參考不存在的根節點。 參考的根節點可能已刪除或重新命名之後已被設定為**根參考**屬性值。  
  
 **使用者動作**  
  
 設定**根參考**屬性**結構描述**節點一次。 屬性的下拉式清單包括目前適用於根節點的所有有效值。
