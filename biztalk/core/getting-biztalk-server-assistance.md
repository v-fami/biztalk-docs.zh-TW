# <a name="getting-biztalk-server-assistance"></a>取得 BizTalk Server 協助
Microsoft 提供三個有關 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 的主要資訊來源：  
  
-   隨 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 安裝的文件、教學課程以及範例。  
  
-   Microsoft Developer Network (MSDN) 與 TechNet 上的 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 網站。  
  
-   其他 Microsoft 網站上的 BizTalk Server 資訊。  
  
 您也可以從 BizTalk Server 社群或直接從 Microsoft 支援服務獲得協助。  
  
## <a name="biztalk-server-documentation-tutorials-and-samples"></a>BizTalk Server 文件、教學課程以及範例  
 當您安裝 BizTalk Server 時，您可以安裝的 SDK、 範例應用程式和教學課程。 核心文件、範例與教學課程涵蓋有效使用 BizTalk Server 所需的觀念與程序。 而在 BizTalk Server 說明中還包含使用 BizTalk Server 建置應用程式的語言與程式設計介面的參考資料。  
  
 如果您安裝 SDK 和教學課程使用[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]安裝程式，您可以存取這些兩種方式：  
  
-   透過 [開始] 功能表上的 [!INCLUDE[btsBizTalkServerNoVersion_md](../includes/btsbiztalkservernoversion-md.md)] 程式群組。  
  
-   透過 BizTalk Server 工具和公用程式媒體中您[!INCLUDE[btsBizTalkServerNoVersion_md](../includes/btsbiztalkservernoversion-md.md)]安裝資料夾。  
  
 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]文件集可在網站上。

 數個網站提供特定的對象，以取得與其工作最相關的 BizTalk Server 資訊連結。
