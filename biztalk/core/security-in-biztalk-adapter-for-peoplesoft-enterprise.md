# <a name="security-in-biztalk-adapter-for-peoplesoft-enterprise"></a>BizTalk Adapter for PeopleSoft Enterprise 中的安全性

## <a name="overview"></a>概觀
Microsoft BizTalk Adapter for PeopleSoft Enterprise 提供單一登入 (SSO) 支援 。 由企業單一登入工具所建立的分支機構應用程式，代表像是 PeopleSoft 的伺服器系統。 

本節提供在安全環境下部署 Microsoft BizTalk Server Adapter for PeopleSoft Enterprise 的指導方針。  
  
> [!CAUTION]
>  我們建議您限制只有，授權使用者才能使用 BizTalk Adapter for PeopleSoft Enterprise 的特定業務應用程式在直接連接的用戶端檔案。
