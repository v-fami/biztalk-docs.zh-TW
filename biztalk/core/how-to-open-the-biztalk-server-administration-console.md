# <a name="how-to-open-the-biztalk-server-administration-console"></a>如何開啟 BizTalk Server 管理主控台
您可以使用下列程序來開啟 BizTalk Server 管理主控台，以及排序或重新整理詳細資料窗格中顯示的資料。  
  
## <a name="prerequisites"></a>必要條件  
 若要執行此程序，您必須以「BizTalk Server 操作員」群組的成員登入，或以「BizTalk Server 系統管理員」群組的成員登入。  
  
### <a name="to-open-the-biztalk-server-administration-console"></a>開啟 BizTalk Server 管理主控台  
  
- 按一下 **開始**，按一下**程式**，按一下  [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然後按一下**BizTalk Server 管理**。  
  
   或  
  
- 按一下 **開始**，按一下**執行**，型別**BTSmmcLauncher.exe**，然後按一下 **確定**。  
  
### <a name="to-refresh-the-biztalk-server-administration-console"></a>重新整理 BizTalk Server 管理主控台  
  
-   以滑鼠右鍵按一下您想要重新整理，然後按一下主控台樹狀目錄中的節點**重新整理**。  
  
     這會重新整理該節點中的所有物件。  
  
### <a name="to-sort-columns-in-the-details-pane-of-the-biztalk-server-administration-console"></a>排序 BizTalk Server 管理主控台的詳細資料窗格中的資料行  
  
-   在主控台樹狀結構中，按一下項目，在詳細資料窗格中檢視其資訊。  
  
-   在詳細資料窗格中，按一下您要依照其排序的資料行標頭。 這會以遞增順序在標頭之下排序項目。  
  
-   再按一下資料行標頭，以遞減 (相反) 順序排序。  
  
### <a name="to-reorder-the-columns-in-the-details-pane-of-the-biztalk-server-administration-console"></a>重新排序 BizTalk Server 管理主控台的詳細資料窗格中的資料行  
  
-   在詳細資料窗格中，按一下並拖曳資料行，將資料行移至新的位置，以變更資料行出現在詳細資料窗格中的順序。
