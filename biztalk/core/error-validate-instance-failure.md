# <a name="error---validate-instance-failure"></a>錯誤-驗證執行個體失敗
**說明**  
  
 **驗證執行個體**命令無法驗證指定的執行個體訊息檔案是否符合指定的結構描述。  
  
 **使用者動作**  
  
 檢視伴隨的錯誤訊息，以判斷執行個體驗證失敗的原因。
