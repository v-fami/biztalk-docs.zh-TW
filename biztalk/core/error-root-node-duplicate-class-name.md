# <a name="error---root-node-duplicate-class-name"></a>錯誤-根節點重複類別名稱
**錯誤碼**  
  
 BEC2013  
  
 **說明**  
  
 **RootNode TypeName**必須是唯一的結構描述中根節點的所有屬性。 兩個或多個根節點，此結構描述中找到具有相同的值及其**RootNode TypeName**屬性。  
  
 **使用者動作**  
  
 選取相關根節點，如此可確保您為提供的值及其**RootNode TypeName**屬性是有效且唯一。
