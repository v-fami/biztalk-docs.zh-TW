# <a name="error---exception-caught"></a>錯誤-攔截到例外狀況
**錯誤碼**  
  
 btm1023  
  
 **說明**  
  
 發生指定的例外，表示驗證對應或測試時發生嚴重錯誤。  
  
 **使用者動作**  
  
 根據指定例外的性質，採取適當的更正動作。
