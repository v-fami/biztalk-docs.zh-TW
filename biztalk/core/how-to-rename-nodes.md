# <a name="how-to-rename-nodes"></a>如何重新命名的節點
您有時可能會想要重新命名現有的節點。 本主題描述此工作的兩個變化。  
  
### <a name="to-rename-a-single-node"></a>重新命名單一節點  
  
1.  選取**記錄** 節點，**欄位項目** 節點，或**欄位屬性**您想要重新命名的節點。  
  
2.  在**BizTalk**功能表上，按一下 **重新命名**。  
  
3.  輸入節點的新名稱，然後按 ENTER。  
  
    > [!NOTE]
    >  您可以藉由變更的值也重新命名節點及其**節點名稱**屬性。  
  
### <a name="to-rename-multiple-nodes-to-have-the-same-name"></a>重新命名多個節點為相同名稱  
  
1.  在要給予相同名稱的節點集之中選取第一個節點。  
  
2.  按住 CTRL，然後選取要給予相同名稱的其餘節點。  
  
3.  在 [屬性] 視窗中變更的值**節點名稱**屬性，然後按 ENTER 結束。  
  
     所有選取的節點都會在結構描述樹狀結構檢視中以新名稱更新。
