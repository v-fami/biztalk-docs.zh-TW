# <a name="creating-and-configuring-send-ports"></a>建立和設定傳送埠

## <a name="overview"></a>概觀
本節提供關於使用 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 管理主控台來建立和設定 BizTalk 應用程式之傳送埠的指示。 傳送埠是傳送訊息或接收訊息的目的或來源位置，並會以其唯一的名稱來識別。  
  
> [!NOTE]
>  您可以使用 Microsoft Windows Management Instrumentation (WMI) 物件模型，建立和執行會自動化系統管理工作的指令碼。 如需使用 WMI 的詳細資訊，請參閱**WMI 類別參考** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
