# <a name="nesting-orchestrations"></a>巢狀協調流程
您可以使用**呼叫協調流程**圖形和**啟動協調流程**圖形來叫用另一個協調流程。 您也可以將協調流程組成任意深度的巢狀：例如被呼叫的協調流程可以呼叫第三個協調流程，而此流程可呼叫第四個協調流程等等。
