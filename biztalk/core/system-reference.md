# <a name="system-reference"></a>System 參考
**系統**參考命名空間包含主要類別和基底類別，定義常用值和參考資料類型、 事件和事件處理常式、 介面、 屬性和處理例外狀況。 此預設參考是讓所有 BizTalk 專案正確運作的必要項目。 請勿刪除**系統**參考。  
  
 如需有關**系統**參考，請參閱 < System 命名空間 」，Visual Studio 組合集合中在[http://go.microsoft.com/fwlink/?LinkId=193568](http://go.microsoft.com/fwlink/?LinkId=193568)。
