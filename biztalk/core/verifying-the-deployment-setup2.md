# <a name="verifying-the-deployment-setup"></a>驗證部署設定
使用 BizTalk Server 匯入繫結檔案之前，請確認下列項目：  
  
-   在新電腦上回應的資料夾必須存在且相同，否則請編輯繫結檔案。  
  
-   TIBCO Enterprise Message Service 系統密碼 (如果出現在組態中) 必須以 ***** 格式儲存在繫結檔案中。 如需詳細資訊，請參閱[部署限制](../core/deployment-limitations1.md)。
