# <a name="error---native-serialization"></a>錯誤-原生序列化
**錯誤碼**  
  
 btm1048  
  
 **說明**  
  
 嘗試將對應所產生的 XML 輸出執行個體訊息轉換為目的結構描述所指定的原生格式時，發生指出的序列化錯誤。  
  
 **使用者動作**  
  
 根據指出的序列化錯誤，適當更正在對應中指定的轉換，或更正目的結構描述，或兩者。 可能會很有幫助，BizTalk 編輯器中開啟目的結構描述，並使用**驗證結構描述**，**驗證執行個體**，和**產生執行個體**可用的命令時以滑鼠右鍵按一下方案總管 中的結構描述。
