# <a name="implementing-message-security"></a>實作訊息安全性
您可以設定 Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 環境來使用憑證，以傳送及接收簽署和加密的訊息。 藉由在加密和數位簽章中使用憑證，BizTalk Server 便可以：  
  
- 傳送及接收可以信任的資料。  
  
- 確認它所處理的資料是安全的。  
  
- 確認授權的合作對象可以收到它的訊息。  
  
- 確認它可以從授權的合作對象收到訊息。  
  
  本章節提供的資訊是關於如何設定 BizTalk Server 管線、接收位置、連接埠和 BizTalk Server 環境來實作訊息安全性。  
  
  如需訊息安全性的詳細資訊，請參閱[規劃訊息安全性](../core/planning-message-security.md)。
