# <a name="replaying-actions"></a>重新執行動作
「協調流程偵錯工具」可以讓您逐步重新執行協調流程的動作。 當您開啟協調流程時，游標會放在第一個動作上，例如，初始化。 使用向上與向下箭號可在協調流程中前進或後退。 依照繪製協調流程在圖形間移動的進度執行。
