# <a name="how-to-define-milestone-groups"></a>如何定義里程碑群組
使用里程碑群組將相關的里程碑 (例如，定義訂單有效期長度的開始和結束里程碑) 放在一起， 然後您就可以將里程碑群組當做單一里程碑使用。  
  
 使用「BAM 檢視精靈」來建立新里程碑群組。 在建立里程碑或里程碑群組之前，請使用「BAM 活動精靈」來建立商務活動、項目和商務活動檢視。 如需有關如何使用精靈的詳細資訊，請參閱[定義商務活動和在 Excel 中的檢視](../core/defining-business-activities-and-views-in-excel.md)。  
  
 商務活動及項目是透過「BAM 活動精靈」所建立，而非「BAM 檢視精靈」。  
  
### <a name="to-define-milestone-groups"></a>定義里程碑群組  
  
1.  在 BAM 檢視精靈中，按一下 **下一步**直到您看到第二個**編輯 BAM 檢視： 檢視項目**頁面。 按一下**新群組**。  
  
2.  輸入商務里程碑別名。  
  
3.  在視窗下方 [別名] 方塊中，按一下您想要包含在群組中，然後按一下的里程碑**確定**。  
  
    > [!NOTE]
    >  如果您想要將所有的里程碑活動放在群組中，按一下**選取所有里程碑**。  
  
4.  當您完成新增別名、 持續時間和群組時，按一下**下一步**繼續對話方塊的 [彙總] 頁面。
