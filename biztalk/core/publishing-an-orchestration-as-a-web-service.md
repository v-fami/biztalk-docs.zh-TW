# <a name="publishing-an-orchestration-as-a-web-service"></a>協調流程發佈為 Web 服務
您可以使用 [BizTalk Web 服務發佈精靈] 將協調流程發佈為 Web 服務。 精靈會根據 BizTalk 組件中的協調流程來建立 Web 服務。 使用此精靈，即可選取協調流程和接收埠來發佈 Web 服務。 您可以定義目標命名空間、SOAP 標頭需求，以及精靈所產生 Web 服務專案的位置。  
  
 在執行精靈之前，您必須編譯其中包含要發佈為 Web 服務之協調流程和結構描述的 BizTalk 組件。  
  
 您必須先啟用 Web 服務，才能執行 [BizTalk Web 服務發佈精靈]。 如需有關啟用 Web 服務系統的詳細資訊，請參閱[啟用 Web 服務](../core/enabling-web-services.md)。  
  
> [!NOTE]
>  您必須先安裝 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，才能夠將協調流程發佈為 Web 服務。 您不需要安裝 Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]。
