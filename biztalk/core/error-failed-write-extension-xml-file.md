# <a name="error---failed-write-extension-xml-file"></a>錯誤-無法寫入延伸模組 XML 檔案
**錯誤碼**  
  
 btm1058  
  
 **說明**  
  
 嘗試將此對應產生的延伸模組物件 XML 寫入指定的檔案時，發生指定的錯誤。  
  
 **使用者動作**  
  
 根據指出的錯誤，採取適當的動作以更正失敗原因。 例如，釋放您的硬碟空間。
