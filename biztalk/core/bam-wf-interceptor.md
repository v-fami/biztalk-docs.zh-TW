# <a name="bam-wf-interceptor"></a>BAM WF 攔截器
BAM WF 攔截器提供了在 WF 應用程式內追蹤資料的完整支援。  
  
 您可以使用 BAM WF 攔截器執行下列作業：  
  
-   直接將工作流程應用程式資料傳送到 BAM，而不需要重新編譯您的 WF 應用程式。  
  
-   鎖定工作流程內的特定活動，以選擇性地將工作流程應用程式資料傳送到 BAM。  
  
-   接受工作流程的交易語意。 只有在主控交易認可時，才會將資料傳送到 BAM。
