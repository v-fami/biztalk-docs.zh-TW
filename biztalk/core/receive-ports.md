# <a name="receive-ports"></a>接收埠
接收埠可當作單向連接埠或雙向連接埠使用。 單向接收埠只負責接收訊息。 要求-回應 (或雙向) 連接埠則可接收和傳送訊息。 接收埠是接收位置的邏輯群組。 例如，您建立多個接收位置，而且要藉由特定協調流程來處理在這些位置接收的訊息。
