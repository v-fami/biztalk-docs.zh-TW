# <a name="how-to-list-bam-views"></a>如何列出 BAM 檢視
系統管理員可以使用**取得檢視**命令來列出執行命令的電腦上的 BAM 主要匯入資料庫中所部署的 BAM 檢視。  
  
### <a name="to-list-bam-views"></a>列出 BAM 檢視  
  
1. 開啟命令提示字元，如下所示： 按一下**開始**，按一下**執行**，型別**cmd**，然後按一下 **確定**。  
  
2. 瀏覽至 [!INCLUDE[btsBiztalkServerPath](../includes/btsbiztalkserverpath-md.md)]。  
  
3. 型別**bm get 檢視**。  
  
4. 按 ENTER 鍵。
