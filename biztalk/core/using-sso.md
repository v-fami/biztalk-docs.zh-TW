# <a name="using-sso"></a>使用 SSO
您可以使用 MMC 嵌入式管理單元或是命令列管理公用程式 (ssomanage)，以管理 SSO 系統。 這包括像是更新 SSO 資料庫、新增、刪除和管理應用程式以及管理使用者對應等活動。  
  
> [!NOTE]
>  在系統上可支援使用者帳戶控制 (UAC)，您可能需要以系統管理權限執行 SSO 命令列工具。  
  
 只有「單一登入」系統管理員可以執行這些工作。
