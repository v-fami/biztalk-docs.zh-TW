|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10769|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_APP_INCOMPLETE_FIELDS|  
|訊息文字|無法啟用應用程式，因為欄位不完整，此應用程式。|  
  
## <a name="explanation"></a>說明  
 無法啟用應用程式，因為欄位不完整，此應用程式。  
  
## <a name="user-action"></a>使用者動作  
 一旦您已經指定給定應用程式的欄位數目，您必須建立並填入才能啟用應用程式。
