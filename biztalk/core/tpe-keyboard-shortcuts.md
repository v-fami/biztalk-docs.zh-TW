# <a name="tpe-keyboard-shortcuts"></a>TPE 鍵盤快速鍵
在「追蹤設定檔編輯器」(TPE) 中，您可以使用鍵盤快速鍵來執行與使用滑鼠相同的巡覽和編輯功能。 您可以使用這些快速鍵來執行用於 TPE 中的所有功能。 本節描述鍵盤快速鍵並提供在 TPE 中使用鍵盤巡覽的秘訣。  
  
## <a name="tpe-tabs"></a>TPE 索引標籤  
 當您一開始決定建立新的設定檔或在 TPE 中開啟現有的設定檔時，索引標籤焦點會出現在 [事件來源檢視] 的右窗格上。  
  
 當您退出 [活動檢視] 時，索引標籤焦點會移到右邊窗格的 [事件來源檢視]。 當您在 [事件來源檢視] 中時，可以使用向上鍵和向下鍵來巡覽框架內容和反白顯示特定協調流程圖形或結構描述項目。  
  
## <a name="tpe-keyboard-shortcuts"></a>TPE 鍵盤快速鍵  
 TPE 提供鍵盤快速鍵巡覽，而不需使用滑鼠。 下表列出 TPE 支援的鍵盤快速鍵。  
  
|動作|鍵盤快速鍵|  
|------------|-----------------------|  
|建立新設定檔|ALT + F + N|  
|開啟現有的追蹤設定檔|ALT + F + O|  
|儲存追蹤設定檔|ALT + F + S|  
|另存新檔|ALT + F + A|  
|匯入活動定義|ALT + F + I|  
|刪除快顯功能表項目|DELETE|  
|重新命名快顯功能表項目|F2|  
|開啟追蹤設定檔編輯器說明|F1|  
|套用追蹤設定檔|ALT + T + A|  
|移除追蹤設定檔|ALT + T + R|  
|設定管理資料庫|ALT + T + M|  
|訊息內容結構描述|SHIFT + F10 + M|  
|結束，關閉 TPE|ALT + F + X|  
|選取協調流程排程|ALT + S + O|  
|選取傳訊內容|ALT + S + M|  
|選取內容屬性|ALT + S + C|  
|選取傳訊屬性|ALT + S + P|  
|內容屬性結構描述|SHIFT + F10 + M|  
|訊息屬性結構描述|Shift + F10 + P|
