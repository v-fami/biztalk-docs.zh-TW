# <a name="single-sign-on-event-10833"></a>單一登入： 事件 10833
## <a name="details"></a>詳細資料  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  產品名稱   |                 企業單一登入                  |
| 產品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件識別碼     |                           10833                            |
|  事件來源   |                           ENTSSO                           |
|    元件    |                            不適用                             |
|  符號名稱  |            ENTSSO_E_SSO_DATABASE_NOT_CONFIGURED            |
|  訊息文字   |       未設定指定的 SSO 資料庫。        |
  
## <a name="explanation"></a>說明  
 系統找不到您指定的 SSO 資料庫。  
  
## <a name="user-action"></a>使用者動作  
 請確認您指定的 SSO 資料庫和 SQL Server 的正確名稱。
