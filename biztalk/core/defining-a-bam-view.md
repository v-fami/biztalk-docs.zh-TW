# <a name="defining-a-bam-view"></a>定義 BAM 檢視
有時候您並不想要向所有商務使用者公開所有的商務資料。 您可以建立不同的 BAM 檢視來公開不同類型的資訊。  
  
> [!IMPORTANT]
>  所有的別名、持續時間、群組、維度和量值名稱都必須是唯一的，因為這些項目都是相同檢視中的一部分。  
  
> [!NOTE]
>  若要定義 BAM 檢視，您先前必須已經建立了至少一個 BAM 活動。 如需有關如何建立活動的詳細資訊，請參閱[如何定義商務活動](../core/how-to-define-a-business-activity.md)。  
  
> [!NOTE]
>  BAM 檢視會在 Excel 活頁簿中顯示為工作表。 您可以為 BAM 活動建立許多檢視。 使用 Excel 應用程式視窗下方的工作表索引標籤來選取要編輯的 BAM 檢視。
