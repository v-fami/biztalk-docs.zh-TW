# <a name="error---schema-file-format-not-valid"></a>錯誤-無效的結構描述檔案格式
**說明**  
  
 結構描述檔案不包含有效的 XML 結構描述定義語言 (XSD) 語法。  
  
 **使用者動作**  
  
 驗證結構描述檔案的來源，以判斷該問題的原因。
