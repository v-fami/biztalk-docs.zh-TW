# <a name="create-your-application"></a>建立您的應用程式
本節包含的開發人員想要建立 BizTalk Server 應用程式，並使用 BizTalk Adapter for JD Edwards EnterpriseOne 的資訊。 一旦建立這些應用程式，然後您就可以[加入例外狀況處理](../core/using-biztalk-server-exception-handling3.md)協調流程。 
  
> [!NOTE]
>  對於 Microsoft Windows 平台上安裝的 JD Edward 系統，您必須使用 BizTalk Server 檔案配接器；對於 IBM AS400 系統上安裝的系統，您必須使用 JD Edwards 接收。
