# <a name="schemas-node"></a>[結構描述] 節點
使用**結構描述**節點以顯示部署在 BizTalk 中可用的目前應用程式與系統結構描述的結構描述。 以滑鼠右鍵按一下**結構描述**節點以顯示下列命令：  
  
-   **隱藏屬性結構描述**。 選取時，會篩選出用於將執行個體訊息中的特定值升級為訊息內容的結構描述。 當您選取此功能表項目時，則會取代在**結構描述**節點功能表**顯示屬性結構描述**。  
  
-   **顯示屬性結構描述**。 選取時會篩選出屬性結構描述。 當您選取此功能表項目時，則會取代在**結構描述**節點功能表**隱藏屬性結構描述**。
