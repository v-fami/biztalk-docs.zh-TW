# <a name="adapter-framework-configuration-schema-extensions"></a>配接器架構組態結構描述延伸模組
BizTalk 配接器架構支援根據 XSD 定義動態產生使用者介面。 配接器會提供必要的 XSD，而配接器架構則建立可讓使用者輸入值的屬性頁。  
  
 配接器架構會提供數個延伸模組，以便建立自訂使用者介面。 若要使用這些延伸模組，您必須匯入配接器架構結構描述 (BizTalkAdapterFramework.xsd)。 藉由匯入結構描述，您便可以存取和使用配接器組態結構描述中的裝飾和特定類型。  
  
 使用本節所討論的裝飾標記和內建類型，以自訂配接器的屬性方格。
