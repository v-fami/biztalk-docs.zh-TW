# <a name="adapter-samples---development"></a>配接器範例 - 開發
本節針對各配接器範例示範的功能、建置和執行範例的指示以及預期的結果，提供相關資訊。  
  
## <a name="in-this-section"></a>本節內容  
  
-   [File 配接器 （BizTalk Server 範例）](../core/file-adapter-biztalk-server-sample.md)。 示範如何撰寫。以網路為基礎 File 配接器在 Microsoft Visual C#。  
  
-   [HTTP 配接器 （BizTalk Server 範例）](../core/http-adapter-biztalk-server-sample.md)。 示範如何實作要用於 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 的「要求/回應」和「請求/回應」通訊範例。  
  
-   [SubmitDirect （BizTalk Server 範例）](../core/submitdirect-biztalk-server-sample.md)。 示範可從使用配接器之 .NET 為基礎的應用程式，將訊息提交給 BizTalk Server 的兩種方式。  
  
-   [交易式配接器 （BizTalk Server 範例）](../core/transactional-adapter-biztalk-server-sample.md)。 示範如何建立及使用明確 Microsoft 分散式交易協調器 (MSDTC) 交易對資料庫在處理期間的 BizTalk Server 訊息。
