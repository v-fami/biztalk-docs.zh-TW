# <a name="how-the-tpe-works"></a>TPE 如何運作
例如，當商務分析師要針對特定客戶訂單進行疑難排解，或是分析出貨流程中特定商務事件和里程碑的趨勢時，他可以指定想要看到的資料檢視。 為了擷取商務分析師所需的資料，開發人員會使用 TPE 來執行特定商務事件資料與實際協調流程之間的對應。 當您建立、儲存、測試和部署追蹤設定檔之後，商務分析師可以擷取已收集的資料，以分析商務程序的特定層面。
