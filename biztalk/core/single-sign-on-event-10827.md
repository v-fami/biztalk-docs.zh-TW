|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10827|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_PSADMIN_INVALID_GROUP_ADAPTER_TYPE|  
|訊息文字|無效的應用程式群組配接器類型。 有效的應用程式類型為 「 密碼同步群組配接器 」。|  
  
## <a name="explanation"></a>說明  
 無效的應用程式群組配接器類型。 有效的應用程式類型為 「 密碼同步群組配接器 」。  
  
## <a name="user-action"></a>使用者動作  
 使用其中一種有效的應用程式類型。
