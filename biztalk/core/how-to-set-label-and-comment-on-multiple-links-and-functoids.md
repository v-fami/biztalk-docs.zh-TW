# <a name="how-to-set-label-and-comment-on-multiple-links-and-functoids"></a>如何在多個連結與運算質設定標籤和註解
您可以設定多個運算質和 (或) 連結的通用標籤和 (或) 註解。 本主題提供如何執行此作業的詳細資料。  
  
 如需如何選取多個運算質和/或連結的資訊，請參閱[如何選取多個連結和運算質](../core/how-to-select-multiple-links-and-functoids.md)。  
  
 選取多個運算質和 (或) 連結之後，您可以執行下列任一作業：  
  
-   設定所選取連結的通用標籤。  
  
-   設定所選取運算質的通用標籤和 (或) 註解。  
  
-   設定所選取運算質和連結的通用標籤。  
  
## <a name="prerequisites"></a>必要條件  
 這些指示需要執行中的 BizTalk 對應工具。  
  
## <a name="to-set-a-common-label-for-selected-links"></a>若要設定所選取連結的通用標籤  
 在多個連結上設定通用標籤是一個不可部分完成的作業。 您可以復原或重做此作業。  
  
 若要設定所選取連結的通用標籤：  
  
1.  在對應工具方格頁上，選取想要的連結。  
  
2.  在**屬性**視窗中，輸入適用的標籤在**標籤**屬性。  
  
## <a name="to-set-a-common-label-andor-comment-for-selected-functoids"></a>若要設定所選取運算質的通用標籤和 (或) 註解  
 為所選取的運算質設定通用標籤和通用註解，是兩項不同的不可部分完成作業。 您可以復原或重做這些作業。  
  
 若要設定所選取運算質的通用標籤/註解：  
  
1.  在對應工具方格頁上，選取想要的運算質。  
  
2.  在**屬性**視窗中，輸入適用的標籤及/或註解中**標籤**和**註解**屬性。  
  
## <a name="to-set-a-common-label-for-selected-functoids-and-links"></a>若要設定所選取運算質和連結的通用標籤  
 您可以設定所選取運算質和連結的通用標籤。 **標籤**是唯一運算質和連結的通用的屬性。 您可以復原或重做此作業。  
  
 若要設定所選取運算質和連結的通用標籤：  
  
1.  在對應工具方格頁上，選取想要的運算質和連結。  
  
2.  在**屬性**視窗中，輸入適用的標籤在**標籤**屬性。
