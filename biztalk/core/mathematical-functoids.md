# <a name="mathematical-functoids"></a>數學運算質

## <a name="overview"></a>概觀
**數學**運算質可用以執行各種基本的數學運算。  
  
 輸入參數數目**數學**運算質不同運算質到運算質，並在合理的狀況下，這些運算質都接受大量輸入。 例如，**除法**運算質需要兩個輸入的參數 （被除數和除數），同時**加法**運算質接受介於 2 到 100 的輸入參數，導致其加總。  
  
> [!IMPORTANT]
>  因為 Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]使用.NET Framework 中，部分所產生的結果的基礎功能**數學**運算質可能與舊版 BizTalk Server 中的對等運算質不同。 請徹底測試您的對應，以確保得到預期的結果。  

## <a name="functoids-list"></a>運算質清單  
 **數學**運算質為： 

* 絕對值
* 加
* 除
* Integer
* 最大值
* 最小值
* 餘數
* 乘
* 捨入
* 平方根
* 減

在這些運算質的詳細資訊是[!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
