# <a name="how-to-delete-schemas"></a>如何刪除結構描述
有時候，您可能不只想將結構描述自其 BizTalk 專案中排除，而是想將它自硬碟中完全移除。 相對於排除結構描述，這稱為刪除結構描述。 本主題描述刪除結構描述所需的步驟。  
  
### <a name="to-delete-a-schema"></a>刪除結構描述  
  
1.  在 方案總管 中，以滑鼠右鍵按一下您想要刪除，然後按一下 結構描述**刪除**。  
  
2.  在確認對話方塊中按一下 **[確定]**。  
  
> [!NOTE]
>  刪除的結構描述會自檔案系統中永遠移除，因此請謹慎進行並定期備份您的檔案。
