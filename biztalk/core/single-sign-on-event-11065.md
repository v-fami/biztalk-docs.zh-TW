# <a name="single-sign-on-event-11065"></a>單一登入： 事件 11065
## <a name="details"></a>詳細資料  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  產品名稱   |                 企業單一登入                  |
| 產品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件識別碼     |                           11065                            |
|  事件來源   |                           ENTSSO                           |
|    元件    |                            不適用                             |
|  符號名稱  |            SSO_INFO_PASSWORD_SYNC_MIIS_STARTING            |
|  訊息文字   |    Windows 的密碼同步 (從 MIIS) 正在啟動。%r    |
  
## <a name="explanation"></a>說明  
 這是資訊訊息，指出 Windows 的密碼同步 (從 MIIS) 正在啟動。  
  
## <a name="user-action"></a>使用者動作  
 使用者不必採取任何動作。
