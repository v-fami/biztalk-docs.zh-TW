# <a name="how-to-navigate-with-the-keyboard"></a>如何使用鍵盤巡覽
您可以不用滑鼠，而是使用鍵盤來巡覽整個設計介面。 下表顯示您可以使用的按鍵。  
  
|索引鍵|效果|  
|---------|------------|  
|DOWN ARROW|將游標移到下一個連接線或下面的圖形，包括**啟動**和**結束**圖形。|  
|向上鍵|將游標移到上一個連接線或是上面的圖形，包括**啟動**和**結束**圖形。|  
|PAGE UP|將游標焦點變更到階段的上一個開始圖形。 這個按鍵也具有向上捲動頁面，以便顯示前面管線部分的效果。|  
|PAGE DOWN|將游標焦點變更到階段的下一個開始圖形。 這個按鍵也具有向下捲動頁面，以便顯示後面管線部分的效果。|  
|HOME|焦點變更**啟動**管線的圖形。|  
|END|焦點變更**結束**管線的圖形。|
