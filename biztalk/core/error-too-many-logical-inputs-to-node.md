# <a name="error---too-many-logical-inputs-to-node"></a>錯誤-節點有太多邏輯輸入
**錯誤碼**  
  
 btm1006  
  
 **說明**  
  
 沒有連接到指定的輸入連結數目與目的結構描述節點的邏輯連結數目越大**迴圈**連接到指定節點的祖系節點的運算質。 前一個與後一個類型的連結數目應該一致。  
  
 **使用者動作**  
  
 重做的連結數目連接至指定的節點和**迴圈**」 運算質連接到上階節點，使其相符。
