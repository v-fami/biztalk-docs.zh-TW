# <a name="soap-adapter"></a>SOAP 配接器
Microsoft BizTalk Server Web 服務在接收和傳送 Web 服務要求時會使用 SOAP 配接器。 SOAP 配接器可以讓您發佈協調流程做為 Web 服務與取用外部 Web 服務。
