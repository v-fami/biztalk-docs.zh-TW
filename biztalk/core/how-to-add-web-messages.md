# <a name="how-to-add-web-messages"></a>如何新增 Web 訊息
您在 [協調流程檢視] 視窗中建立 Web 訊息變數。 您必須將 Web 訊息的訊息類型設定為 Web 訊息類型。  
  
### <a name="to-add-a-web-message"></a>若要新增 Web 訊息  
  
1.  開啟，請在協調流程與**檢視**功能表上，按一下 **其他視窗**，然後按一下 **協調流程檢視**。  
  
2.  以滑鼠右鍵按一下**訊息**節點，然後按一下**新訊息**。  
  
3.  Message_*x*之下**訊息**節點。  
  
4.  在**屬性**視窗中，從**訊息類型**下拉式清單方塊中，選取 Web 訊息類型。  
  
5.  將下拉式清單方塊捲動至您要使用的 Web 訊息類型，然後按一下該 Web 訊息類型選取它。  
  
     下圖顯示**協調流程檢視**和選取 Message_1 視窗和 屬性 視窗中，選取 Web 訊息類型的 Message_1。
