# <a name="managing-aggregations"></a>管理彙總
彙總是資料集衍生一般代表特性或資料集本身描述的累計作業。 本節說明如何管理不同的彙總項目，例如索引、活動與即時彙總 (RTA)。  
  
 如需彙總的一般資訊，請參閱[BAM 入口網站中的彙總](../core/aggregations-in-the-bam-portal.md)和[BAM 入口網站頁面上的彙總](../core/aggregations-on-the-bam-portal-page.md)。
