# <a name="btadinstallmode"></a>BTAD_InstallMode
BTAD_InstallMode 會描述 BizTalk 應用程式部署的安裝模式。  
  
## <a name="remarks"></a>備註  
 下表描述 BTAD_InstallMode 的可能值。  
  
|Value|說明|  
|-----------|-----------------|  
|Install|建立或更新 BizTalkHostInstance (本機電腦)|  
|匯入|建立或更新 ConfigurationDb (群組的 BizTalk 管理資料庫)|  
|Uninstall|從 BizTalkHostInstance (本機電腦) 刪除|
