# <a name="nodes-that-provide-wildcard-capabilities"></a>提供萬用字元功能的節點
BizTalk 編輯器提供對 XSD 概念的直接對應，這些 XSD 概念會提供項目與屬性的萬用字元功能。 **Any 項目**節點會直接對應到 XML 結構描述定義 (XSD) 語言**任何**項目和**Any 屬性**節點會直接對應到 XSD **anyAttribute**項目。 接下來將提供這些節點的其他資訊。
