# <a name="appendix-c-using-component-interfaces"></a>附錄 c： 使用元件介面
本章節中的主題說明如何建立新的元件介面，以及如何修改現有元件介面，以便搭配 Microsoft BizTalk Adapter for PeopleSoft Enterprise 使用。 這些主題也會說明如何套用安全性至這些元件介面，以及如何測試安全性。  
  
 您可以執行下列工作：  
  
- 搭配您的應用程式使用 PeopleSoft 提供的元件介面。  
  
   元件介面亦稱為 Enterprise Integration Points (EIP)。  
  
- 修改現有的元件介面。  
  
- 建立新元件介面。  
  
  使用您的元件介面之前，必須先套用及測試安全性。  
  
> [!NOTE]
>  本章節的目的在於提供有用的補充資訊，並不能取代 PeopleSoft 文件。 如需 PeopleSoft 元件介面完整與最新資訊，請參閱 PeopleSoft Online Library。
