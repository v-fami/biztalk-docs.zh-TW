# <a name="biztalk-project-properties-window"></a>BizTalk 專案屬性視窗
[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 中的 BizTalk 專案的專案屬性視窗包含下列屬性：  
  
## <a name="bpel-compliance"></a>BPEL 規範  
 指定是否產生 Business Process Execution Language (BPEL) 規範輸出。  
  
## <a name="project-file"></a>專案檔  
 包含關於專案建置、組態與其他資訊之檔案的名稱。  
  
## <a name="project-folder"></a>專案資料夾  
 專案檔的位置。
