# <a name="error---schema-file-name-not-valid"></a>錯誤-無效的結構描述檔案名稱
**說明**  
  
 指定的結構描述檔案名稱是無效的。  
  
 **使用者動作**  
  
 確認結構描述檔案位於指定的位置，且可供存取。
