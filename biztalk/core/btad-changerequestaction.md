# <a name="btadchangerequestaction"></a>BTAD_ChangeRequestAction
BTAD_ChangeRequestAction 包含表示變更要求動作的值。  
  
## <a name="remarks"></a>備註  
 下表描述可能的 BTAD_ChangeRequestAction 值。  
  
|Value|說明|  
|-----------|-----------------|  
|建立|匯入，但不指定覆寫|  
|Update|匯入或安裝，並指定覆寫|  
|Delete|Uninstall|
