# <a name="how-to-configure-the-message-assignment-shape"></a>如何設定訊息指派圖形
![](../core/media/ebiz-orch-assign.gif "ebiz_orch_assign")  
訊息指派圖形  
  
> [!NOTE]
>  A**訊息指派**圖形只能存在於**建構訊息**圖形。 如果您的設計介面上，拖曳其他地方 「 訊息指派新**建構訊息**圖形將會建立。  
  
### <a name="to-configure-a-message-assignment-shape"></a>若要設定訊息指派圖形  
  
1.  如果看不到 BizTalk 運算式編輯器，以滑鼠右鍵按一下**訊息指派**圖形，並按一下**編輯運算式**或在 屬性 視窗中，按一下**省略**(**...**) 按鈕**運算式**屬性。  
  
2.  在「BizTalk 運算式編輯器」中，建立運算式以指派值。 如需詳細資訊，請參閱[運算式的需求與限制](../core/requirements-and-limitations-for-expressions.md)。
