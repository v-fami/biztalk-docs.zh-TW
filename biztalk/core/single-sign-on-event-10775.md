# <a name="single-sign-on-event-10775"></a>單一登入： 事件 10775
## <a name="details"></a>詳細資料  
  
|                 |                                                                                                                          |
|-----------------|--------------------------------------------------------------------------------------------------------------------------|
|  產品名稱   |                                                企業單一登入                                                 |
| 產品版本 |                                [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                |
|    事件識別碼     |                                                          10775                                                           |
|  事件來源   |                                                          ENTSSO                                                          |
|    元件    |                                                           不適用                                                            |
|  符號名稱  |                                                 ENTSSO_E_APP_ADMIN_SAME                                                  |
|  訊息文字   | 此應用程式的應用程式系統管理員帳戶指定為 SSO 分支機構系統管理員帳戶。 |
  
## <a name="explanation"></a>說明  
 若要使用 SSO 分支機構系統管理員帳戶做為 Applictation 系統管理員帳戶已標示應用程式。 如此一來，您無法設定或變更應用程式系統管理員帳戶。  
  
## <a name="user-action"></a>使用者動作  
 請連絡系統管理員以尋求協助。
