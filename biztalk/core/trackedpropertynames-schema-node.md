# <a name="trackedpropertynames-schema-node"></a>TrackedPropertyNames (結構描述節點)
繫結檔案的 TrackedPropertyNames 節點包含一些項目，這些項目會指定要追蹤父結構描述中的哪些屬性。  
  
## <a name="nodes-in-the-trackedpropertyname-node"></a>TrackedPropertyName 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|string|元素|xs:string|指定將要追蹤的屬性名稱。|不需要|預設值：空白|
