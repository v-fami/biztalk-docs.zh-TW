# <a name="enlisted-parties-role-node"></a>已登錄的合作對象 (角色節點)
繫結檔案的 [已登錄的合作對象] 節點是所有 [已登錄的合作對象] 節點的父節點，其中包含隨同繫結檔案所匯出之角色有關聯之已登錄合作對象的特定資訊。  
  
## <a name="nodes-in-the-enlisted-parties-node"></a>已登錄的合作對象節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[已登錄合作對象](../core/enlisted-party-enlisted-parties-node.md)|記錄|EnlistedParty (ComplexType)|指定與某個角色關聯之已登錄合作對象的相關資訊。|非必要|預設值：無|
