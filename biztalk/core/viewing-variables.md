# <a name="viewing-variables"></a>檢視變數
您可以使用兩種模式來檢視變數：  
  
-   追蹤已封存的資料時 - 離線  
  
-   針對即時資料執行互動式偵錯時  
  
 在處理已封存的資料時，您可看到部分變數的狀態為：  
  
-   變數尚未在範圍中  
  
-   目前作用中  
  
-   您可以按一下以取得詳細資訊的值  
  
-   已傳遞至範圍外的值；您不能再存取任何詳細資訊
