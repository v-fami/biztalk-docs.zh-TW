# <a name="managing-user-mappings"></a>管理使用者對應
本節提供有關如何建立與設定「企業單一登入」對應的資訊。  
  
 系統管理員可以使用 ssomanage 公用程式管理對應，而應用程式使用者可以使用 ssoclient 公用程式管理他們的對應。  
  
> [!NOTE]
>  應用程式系統管理員可以管理與分支機構應用程式 (為其系統管理員) 關聯的對應，而 SSO 分支機構系統管理員和 SSO 系統管理員可以管理與分支機構應用程式關聯的所有對應。
