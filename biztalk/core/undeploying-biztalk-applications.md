# <a name="undeploying-biztalk-applications"></a>解除部署 BizTalk 應用程式
您必須解除部署 BizTalk 應用程式，才能將其從您的 BizTalk 群組和主機電腦移除。 本節描述解除部署應用程式所牽涉到的下列工作：  
  
-   **刪除 BizTalk 應用程式。** 這樣便會從 BizTalk 群組的 BizTalk 資料庫中移除其資料。 當您這樣做時，[BizTalk 管理主控台] 便不會再顯示該應用程式。  
  
-   **解除安裝 BizTalk 應用程式。**  這樣會從本機檔案系統移除與應用程式關聯的檔案、設定及其成品。  
  
-   **正在移除其他檔案和設定。** 您也可能需要移除與應用程式關聯的某些其他檔案和設定。
