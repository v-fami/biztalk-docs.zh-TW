# <a name="how-to-configure-the-loop-shape"></a>如何設定迴圈圖形
![](../core/media/ebiz-orch-loop.gif "ebiz_orch_loop")  
迴圈圖形  
  
### <a name="to-configure-a-loop-shape"></a>若要設定迴圈圖形  
  
1.  如果看不到 BizTalk 運算式編輯器，以滑鼠右鍵按一下**迴圈**圖形，並按一下**編輯布林值運算式**，或在 屬性 視窗中，按一下**省略**(**...**) 按鈕**運算式**屬性。  
  
2.  在 [BizTalk 運算式編輯器]中，建立重複項目條件的布林值運算式。 如需詳細資訊，請參閱[運算式的需求與限制](../core/requirements-and-limitations-for-expressions.md)。
