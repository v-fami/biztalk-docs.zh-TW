# <a name="delimited-record-considerations"></a>分隔記錄的考量
有許多考量，您應該謹記在心時使用分隔**記錄**節點在結構描述內。 這包括附屬節點與其分隔符號的相對順序、組合訊息的一般檔案表示法時可選擇省略分隔符號的情況，以及與混合分隔和位置記錄有關的限制。 本節提供這些考量的相關資訊。
