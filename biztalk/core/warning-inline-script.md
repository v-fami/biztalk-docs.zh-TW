# <a name="warning---inline-script"></a>警告-內嵌指令碼
**錯誤碼**  
  
 btm1020  
  
 **說明**  
  
 提供給指定的內嵌指令碼**指令碼處理**運算質包含語法警告。  
  
 **使用者動作**  
  
 選取指定**指令碼處理**運算質，按一下省略符號 (**...**) 相關聯的按鈕**指令碼**屬性在 microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]屬性 視窗中，然後在**設定運算質指令碼**對話方塊中，正確語法警告。
