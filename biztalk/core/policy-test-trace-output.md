# <a name="policy-test-trace-output"></a>原則測試追蹤輸出
當您測試原則時，原則測試程式會在版本資訊視窗中顯示輸出。 本節描述追蹤內所包含的活動。  
  
> [!NOTE]
>  若已經判斷提示衍生的類別，但直接針對基底類別的成員撰寫規則，則會判斷提示基底類別的執行個體，而且針對該執行個體評估其條件。
