# <a name="creating-web-ports"></a>建立 Web 連接埠
Web 連接埠是專門針對使用 (呼叫) Web 服務所設定的連接埠。 Web 連接埠可包含多項作業，代表混合單向 (僅限要求) 和雙向 (要求-回應) 的 Web 方法。 Web 連接埠中的每一項作業代表 Web 服務的一種方法。  
  
 您必須先新增 Web 參考，才能建立 Web 連接埠。  
  
> [!NOTE]
>  您無法刪除、剪下、複製、貼上或修改 Web 連接埠類型和 Web 訊息類型的組態。
