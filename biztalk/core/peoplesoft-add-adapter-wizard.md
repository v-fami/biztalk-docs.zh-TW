# <a name="peoplesoft-add-adapter-wizard"></a>PeopleSoft 新增配接器精靈
使用 [新增配接器精靈] 從 PeopleSoft 系統存取中繼資料資訊。  
  
|使用|動作|  
|--------------|----------------|  
|選取配接器清單|從清單中選取配接器。|  
|SQL Server|從清單中選取 SQL Server，然後按一下**下一步**。|  
|資料庫|連接到與 SQL Server 相同的資料庫。|  
|通訊埠|選取已設定的傳送埠，然後按一下 **下一步**。|
