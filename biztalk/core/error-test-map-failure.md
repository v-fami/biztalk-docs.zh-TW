# <a name="error---test-map-failure"></a>錯誤-測試對應失敗
**錯誤碼**  
  
 btm1054  
  
 **說明**  
  
 所指定對應的「測試對應」作業失敗。  
  
 **使用者動作**  
  
 使用伴隨的錯誤訊息，判斷失敗的 「 測試對應 」 作業的原因、 地圖、 其相關聯的來源和目的結構描述，還是兩者進行必要的變更。
