# <a name="single-sign-on-event-10569"></a>單一登入： 事件 10569
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10569|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|SSO_WARN_NO_UPDATE_BY_APP_ADMIN|  
|訊息文字|無法變更應用程式 Administrator.%r 應用程式系統管理員帳戶<br /><br /> 應用程式名稱: %1 %r<br /><br /> 應用程式系統管理員: %2 %r<br /><br /> 錯誤碼： %3|  
  
## <a name="explanation"></a>說明  
 由應用程式系統管理員無法變更應用程式系統管理員帳戶。  
  
## <a name="user-action"></a>使用者動作  
 您必須是一個 SSO 分支機構應用程式系統管理員或更高的應用程式 Administratos 帳戶。
