# <a name="managing-the-master-secret"></a>管理主要密碼
主要密碼是用以加密所有儲存在 SSO 資料庫中之資訊的金鑰。 若主要密碼伺服器失敗且密碼遺失，您將無法擷取儲存在 SSO 資料庫中的資訊。 因此，請務必在產生主要密碼之後立即備份。
