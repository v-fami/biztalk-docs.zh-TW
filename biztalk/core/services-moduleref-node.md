# <a name="services-moduleref-node"></a>服務 (ModuleRef 節點)
繫結檔案的 [服務] 節點包含隨著繫結檔案匯出之每項服務的特定資訊。  
  
## <a name="nodes-in-the-services-node"></a>服務節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[服務](../core/service-services-node.md)|記錄|ServiceRef (ComplexType)|描述此服務之節點的容器節點。|不需要|預設值：無|
