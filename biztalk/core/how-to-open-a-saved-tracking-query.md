# <a name="how-to-open-a-saved-tracking-query"></a>如何開啟儲存的追蹤查詢
您可以開啟 （目前執行的服務執行個體和訊息） 的即時追蹤資料的訊息或服務執行個體查詢。 這是由 BizTalk Server 管理主控台中 從的 群組中樞 頁面上，使用 查詢 索引標籤。  
  
## <a name="prerequisites"></a>必要條件  
 您必須以 BizTalk Server 系統管理員群組的成員身分登入，才能執行這個程序。  
  
### <a name="to-open-a-saved-query"></a>若要開啟儲存的查詢  
  
1. 按一下 **開始**，按一下**所有程式**，按一下  [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]，然後按一下**BizTalk Server 管理**。  
  
2. 在 **群組中樞**頁面的 **新的查詢**索引標籤上，按一下 **開啟查詢**。  
  
3. 按 的查詢，然後按一下**開啟**。  
  
4. 如果您開啟查詢，第一次顯示安全性警告，選擇性地檢查**不要再顯示此訊息**以防止它再次發生。 按一下 **[是]** 繼續。
