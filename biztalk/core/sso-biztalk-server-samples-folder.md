# <a name="sso-biztalk-server-samples-folder"></a>SSO （BizTalk Server Samples 資料夾）
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 在其軟體開發套件 (SDK) 中包含了兩個「企業單一登入」(SSO) 範例。 本節提供有關這些 SSO 範例示範的功能、建置和執行這些範例的指示，以及您可預期之結果的詳細資訊。  

## <a name="in-this-section"></a>本節內容  

- [HTTPSSO （BizTalk Server 範例）](../core/httpsso-biztalk-server-sample.md)。 示範如何搭配 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] HTTP 配接器來使用「企業單一登入」。  

- [管理 （BizTalk Server 範例）](../core/manage-biztalk-server-sample.md)。 示範如何建構 .xml 檔案，該檔案可以與 ssomanage.exe 命令列公用程式搭配使用，執行系統管理作業。
