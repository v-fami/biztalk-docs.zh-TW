# <a name="how-to-modify-vocabulary-definitions"></a>如何修改詞彙定義
您可以使用「詞彙定義精靈」變更顯示名稱或變更與顯示名稱關聯的繫結，以修改未發佈的詞彙版本中的詞彙定義。  
  
> [!NOTE]
>  並非所有項目均可修改。  
  
 本主題包含下列工作的程序：  
  
-   修改詞彙定義  
  
-   新增事實到詞彙或詞彙定義  
  
### <a name="to-modify-a-vocabulary-definition"></a>修改詞彙定義  
  
1.  詞彙定義，以滑鼠右鍵按一下，然後按一下 **修改**。  
  
2.  使用「詞彙定義精靈」以建立所需的新詞彙定義。  
  
### <a name="to-add-a-fact-to-a-vocabulary-or-vocabulary-definition"></a>新增事實到詞彙或詞彙定義  
  
1.  在 [事實總管] 視窗中，按一下**詞彙**索引標籤，然後選取您想要更新的未發佈的詞彙版本。  
  
2.  按一下**資料庫**， **XML 結構描述**，或 **.NET 類別**。  
  
3.  巡覽資料來源階層到您需要的事實。  
  
    > [!NOTE]
    >  事實可以是資料庫資料表或資料庫資料表資料行、XML 文件項目或屬性，或 .NET 類別或類別成員。  
  
4.  按一下事實並按住滑鼠鍵。  
  
5.  拖曳事實**詞彙** 索引標籤。  
  
     **詞彙**索引標籤隨即開啟。  
  
6.  拖曳事實到您要修改的詞彙版本。
