# <a name="error---xsl-transformation-failure"></a>錯誤-XSL 轉換失敗
**錯誤碼**  
  
 btm1050  
  
 **說明**  
  
 轉換處理時發生指定的 XSLT 錯誤。  
  
 **使用者動作**  
  
 根據指出的 XSLT 錯誤，修改對應和/或來源與目的結構描述，以更正錯誤原因。
