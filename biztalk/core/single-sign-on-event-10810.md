# <a name="single-sign-on-event-10810"></a>單一登入： 事件 10810
## <a name="details"></a>詳細資料  
  
|                 |                                                                                  |
|-----------------|----------------------------------------------------------------------------------|
|  產品名稱   |                            企業單一登入                             |
| 產品版本 |            [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]            |
|    事件識別碼     |                                      10810                                       |
|  事件來源   |                                      ENTSSO                                      |
|    元件    |                                       不適用                                        |
|  符號名稱  |                        ENTSSO_E_HISSO_INVALID_CREDENTIALS                        |
|  訊息文字   | 指定的認證無效，或與 SSO 資料庫中的認證不相符。 |
  
## <a name="explanation"></a>說明  
 指定的認證無效，或與 SSO 資料庫中的認證不相符。  
  
## <a name="user-action"></a>使用者動作  
 設定 SSO 資料庫中的資訊，以符合外部系統的資訊。
