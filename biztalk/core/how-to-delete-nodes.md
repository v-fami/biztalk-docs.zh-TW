# <a name="how-to-delete-nodes"></a>如何刪除節點
您可能會需要刪除結構描述樹狀結構中的節點。 本主題提供此工作的逐步解說。  
  
### <a name="to-delete-a-node"></a>刪除節點  
  
1.  以滑鼠右鍵按一下您想要刪除，然後按一下節點**刪除**。  
  
2.  按一下**是**確認對話方塊中。  
  
> [!CAUTION]
>  當您刪除**記錄**節點，其所有子節點也會一併刪除。  
  
> [!CAUTION]
>  若在結構描述樹狀結構中有其他地方參考至您刪除的節點，也會刪除參考的節點。  
  
> [!NOTE]
>  若節點變成灰色，表示它是唯讀，不能將它刪除。
