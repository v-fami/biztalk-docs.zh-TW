# <a name="checklist-undeploy-a-biztalk-application"></a>檢查清單： 解除部署 BizTalk 應用程式
|步驟|參考|  
|----------|---------------|  
|從 BizTalk 群組刪除 BizTalk 應用程式。 這會將它從 BizTalk 管理主控台和 BizTalk Server 資料庫移除。|[如何從 BizTalk 群組刪除 BizTalk 應用程式](../core/how-to-delete-a-biztalk-application-from-the-biztalk-group.md)|  
|從已安裝應用程式的任何電腦解除安裝應用程式。|[如何解除安裝 BizTalk 應用程式](../core/how-to-uninstall-a-biztalk-application.md)|  
|必要時，從全域組件快取 (GAC) 中移除包含在應用程式中的組件。|[如何從 GAC 解除安裝組件](http://msdn.microsoft.com/library/464706a8-f902-4d05-a724-19169facd2b4)|  
|從本機電腦移除其他檔案和設定 (於先前步驟中未自動移除)。|[如何移除其他檔案和設定 BizTalk 應用程式](../core/how-to-remove-other-files-and-settings-for-a-biztalk-application.md)|
