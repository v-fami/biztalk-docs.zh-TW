# <a name="admin-wmi-biztalk-server-samples-folder"></a>管理 WMI （BizTalk Server 範例資料夾）
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 的軟體開發套件 (SDK) 中包含數個 Microsoft Windows Management Instrumentation (WMI) 管理範例。 本節提供每個 WMI 管理範例所示範功能的詳細資訊、用來建置和執行範例的指示，以及您可以預期的結果。  
  
> [!WARNING]
>  如果在部署後不需要部署指令碼，就應該將其移除。 必須保留的系統管理指令碼和其他指令碼都應該由 ACL 保護，而且予以密切監控。  
  
## <a name="in-this-section"></a>本節內容  
  
-   [啟用接收位置 （BizTalk Server 範例）](../core/enable-receive-location-biztalk-server-sample.md)。 示範如何啟用接收位置及設定該接收位置的輸入傳輸 URL。  
  
-   [登錄協調流程 （BizTalk Server 範例）](../core/enlist-orchestration-biztalk-server-sample.md)。 示範如何將 BizTalk Server 協調流程登錄到主控件。  
  
-   [列舉接收位置 （BizTalk Server 範例）](../core/enumerate-receive-locations-biztalk-server-sample.md)。 示範如何擷取一或多個接收位置的詳細資訊。  
  
-   [移除接收埠 （BizTalk Server 範例）](../core/remove-receive-port-biztalk-server-sample.md)。 示範如何移除一或多個接收埠。  
  
-   [移除傳送埠 （BizTalk Server 範例）](../core/remove-send-port-biztalk-server-sample.md)。 示範如何取消登錄一或多個傳送埠，並且將其移除。  
  
-   [設定 Send Handler 屬性 （BizTalk Server 範例）](../core/set-send-handler-property-biztalk-server-sample.md)。 示範如何設定 Simple Mail Transfer Protocol (SMTP) 傳送處理常式的 XML 組態資訊。  
  
-   [啟動傳送埠 （BizTalk Server 範例）](../core/start-send-port-biztalk-server-sample.md)。 示範如何在使用 FILE 配接器時啟動傳送埠及設定主要傳輸位址。  
  
-   [停止協調流程 （BizTalk Server 範例）](../core/stop-orchestration-biztalk-server-sample.md)。 示範如何停止 BizTalk Server 協調流程，以及選擇性地將它取消登錄。
