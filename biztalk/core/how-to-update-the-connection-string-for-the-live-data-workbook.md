# <a name="how-to-update-the-connection-string-for-the-live-data-workbook"></a>如何更新即時資料活頁簿的連接字串
當您將 BAM 主要匯入資料庫移到另一台伺服器時，BAM 即時資料活頁簿中的連接字串必須更新為指向新的伺服器。 您可使用 BAM 的 Excel 增益集執行這項更新。  
  
### <a name="to-update-the-live-data-workbook-to-point-to-a-new-server"></a>將即時資料活頁簿更新為指向新的伺服器  
  
1.  按一下**啟動**，指向 **所有程式**，指向  **Microsoft Office**，然後按一下  **Microsoft Office Excel**。  
  
2.  按一下**檔案**功能表，然後再按一下**開啟**。 瀏覽至您的 BAM 即時的資料活頁簿，然後按一下**開啟**。  
  
3.  按一下**BAM**功能表**增益集**索引標籤，然後再按一下**BAM DB 連線**開啟**選取 BAM 資料庫** 對話方塊。  
  
4.  在**SQL Server**文字方塊中，輸入 SQL server 的 BAM 主要匯入資料庫現在的名稱。  
  
5.  選取從 BAM 主要匯入資料庫**資料庫名稱**下拉式清單。  
  
6.  按一下 **[確定]** ，關閉對話方塊。  
  
7.  儲存活頁簿。
