# <a name="single-sign-on-event-10849"></a>單一登入： 事件 10849
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10849|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_DIRECT_SYNC_NOT_ALLOWED_CREATE|  
|訊息文字|無法指定 'direct 密碼同步處理' 旗標建立應用程式。|  
  
## <a name="explanation"></a>說明  
 無法指定 'direct 密碼同步處理' 旗標建立應用程式。  
  
## <a name="user-action"></a>使用者動作  
 建立應用程式，直接密碼同步旗標。 然後加入和建立欄位，讓應用程式，並指定直接密碼同步旗標。
