# <a name="deploying-biztalk-applications"></a>部署 BizTalk 應用程式
本節提供下列有關部署 BizTalk 應用程式的資訊：  
  
-   將應用程式及其成品匯出至 Windows Installer (.msi) 檔案  
  
-   將 .msi 檔案匯入其他 BizTalk 群組，以便在新群組中建立應用程式及其成品  
  
-   將應用程式安裝在將會予以執行的電腦上  
  
-   將 BizTalk 組件安裝到全域組件快取 (GAC)  
  
-   監控應用程式的部署及測試的進度  
  
-   將應用程式部署到商務夥伴  
  
-   編輯繫結檔案以自訂不同部署環境的繫結  
  
 您必須在不同的應用程式部署案例中執行的工作檢查清單，請參閱[BizTalk 應用程式部署和管理檢查清單](../core/biztalk-application-deployment-and-management-checklists.md)。 如需 BizTalk 應用程式和其部署的背景資訊，請參閱[瞭解 BizTalk 應用程式部署和管理](../core/understanding-biztalk-application-deployment-and-management.md)。 如需建立應用程式，並修改藉由新增或移除成品或加入另一個應用程式的參考資訊，請參閱[建立和修改 BizTalk 應用程式](../core/creating-and-modifying-biztalk-applications.md)。 如需部署 BizTalk 組件的相關資訊，請參閱[從到 BizTalk 應用程式的 Visual Studio 部署 BizTalk 組件](../core/deploying-biztalk-assemblies-from-visual-studio-into-a-biztalk-application.md)。
