# <a name="engine-performance-and-capacity"></a>引擎效能和容量
當您規劃 BizTalk Server 解決方案時，引擎效能和容量是相當重要的考量。 引擎效能是在一段時間處理文件的能力，而引擎容量是您的 BizTalk Server 系統維護已接收但未處理之文件積存的能力。
