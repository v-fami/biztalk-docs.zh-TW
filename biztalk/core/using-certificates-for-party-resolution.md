# <a name="using-certificates-for-party-resolution"></a>使用憑證進行合作對象解析
A*合作對象*是與協調流程互動的 BizTalk Server 外部的實體。 當 BizTalk Server 接收訊息時，它會使用公開金鑰憑證來決定是誰傳送訊息，並將傳送者解析為 BizTalk Server 環境中的已知合作對象。  
  
 本節提供如何設定 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 管線、接收位置、連接埠和 BizTalk Server 環境，以進行合作對象解析的相關資訊。  
  
 如需訊息驗證的詳細資訊，請參閱[驗證訊息的傳送者](../core/authenticating-the-sender-of-a-message.md)。
