# <a name="components-of-the-bam-portal"></a>BAM 入口網站的元件
BAM 入口網站是劃分為三個功能區域 (對應至入口網站的三個頁面) 的 ASP.NET 網頁。 本節將描述這些頁面： 活動搜尋 頁面、 彙總 頁面上及警示管理員 頁面。 您可以使用這些頁面來建立活動、搜尋現有活動或修改現有活動。
