# <a name="keyboard-shortcuts-general-to-the-design-surface"></a>鍵盤快速鍵一般至設計介面
下表描述設計介面之 [處理區域] 及 [連接埠介面] 區域通用的鍵盤巡覽方式。  
  
|索引鍵|效果|  
|---------|------------|  
|TAB|將選取範圍和焦點移至右邊下一個設計介面區域 (亦即 [處理區域] 或 [連接埠介面]) 中未選取的圖形。<br /><br /> 範例：選取範圍和焦點目前位於左 [連接埠介面] 中的連接埠。 按下 TAB 鍵時，選取範圍和焦點會移至 [處理區域] 中非作用中焦點所在的圖形。|  
|SHIFT + TAB|與 TAB 相同，但方向相反。|  
|PAGE DOWN|向下捲動頁面，以顯示協調流程較後面的部分。 選取範圍和焦點不會變更。|  
|PAGE UP|與 PAGE DOWN 鍵相同，但方向相反。 選取範圍和焦點不會變更。|  
|SHIFT + PAGE DOWN|依據固定的遞增值，向左水平捲動整個設計介面。 選取範圍和焦點不會變更。|  
|SHIFT + PAGE UP|與 SHIFT + PAGE DOWN 相同，只是設計介面會向右捲動。 選取範圍和焦點不會變更。|  
|ENTER|執行選取範圍的預設動作，通常是指將焦點轉移至 [屬性] 方格。|  
|SPACE|選取目前焦點所在的圖形。 若焦點位於圖形連接器上則無作用。|  
|CTRL + SHIFT + F1|在選取之圖形的「智慧標籤」進行展開和摺疊的切換。 之後便可使用向上鍵和向下鍵在「智慧標籤」的項目間移動。 展開時，ENTER 鍵或空格鍵可執行「智慧標籤」項目的相關動作 (以及摺疊「智慧標籤」)。|  
|DELETE|刪除目前的選項。|  
|CTRL + 滑鼠滾輪上移|顯示設計介面的縮小檢視。|  
|CTRL + 滑鼠滾輪下移|顯示設計介面的放大檢視。|
