# <a name="how-to-add-or-remove-messages-in-orchestrations"></a>如何新增或移除協調流程中的訊息
訊息是協調流程中的基本通訊單位。 它包含一或多個部分，以及描述訊息屬性和訊息內容的內容資料。  
  
### <a name="to-add-a-message"></a>若要新增訊息  
  
1.  在協調流程檢視] 視窗中，以滑鼠右鍵按一下**訊息**，然後按一下 [**新訊息**。 **訊息**資料夾，則會展開摺疊，並加入新的訊息。  
  
2.  指定訊息名稱。  
  
3.  指定訊息類型。  
  
### <a name="to-remove-a-message"></a>若要移除訊息  
  
-   在協調流程檢視] 視窗中，以滑鼠右鍵按一下您想要移除，然後按一下 [的訊息**刪除**。
