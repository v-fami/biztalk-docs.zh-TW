# <a name="biztalk-flat-file-schema-wizard-walkthrough"></a>BizTalk 一般檔案結構描述精靈逐步解說
下列逐步解說為您顯示如何使用「BizTalk 一般檔案結構描述精靈」從一般檔案文件執行個體建立一般檔案結構描述。  
  
 若要從交換執行個體建立一般檔案結構描述，您應該先遵循逐步解說中描述的步驟，但只需從 [選取文件資料] 畫面中的交換執行個體中選取描述個別文件結構的部分，而不需要選取整個交換執行個體，來建立文件結構描述。 然後，您可定義信封結構描述。 如需有關定義信封結構描述和文件結構描述和信封結構描述設定使用的管線元件的詳細資訊，請參閱[信封結構描述](../core/envelope-schemas.md)和[如何設定 XML 組合器管線元件](../core/how-to-configure-the-xml-assembler-pipeline-component.md)。
