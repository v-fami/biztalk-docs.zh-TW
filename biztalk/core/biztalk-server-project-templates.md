# <a name="biztalk-server-project-templates"></a>BizTalk Server 專案範本

## <a name="overview"></a>概觀
當您安裝[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，安裝可新增至 BizTalk Projects 資料夾**新專案** 對話方塊。 BizTalk Projects 資料夾包含用來建立空白的 BizTalk Server 專案、 BizTalk Server BPEL 匯入專案和 BizTalk Server 應用程式專案範本。

## <a name="available-templates"></a>可用的範本
下表描述這些專案範本。  
  
|使用|動作|  
|--------------|----------------|  
|**空白的 BizTalk Server 專案**|建立新的空白 BizTalk Server 專案。|  
|**BizTalk Server BPEL 匯入專案**|Business Process Execution Language (BPEL)、 Web 服務描述語言 (WSDL) 或 XML 結構描述定義語言 (XSD) 檔案匯入 BizTalk 專案。|  
|**BizTalk Server 應用程式專案 (.btaproj)**|從開始[!INCLUDE[bts2016](../includes/bts2016-md.md)][功能組件 1](../core/configure-the-feature-pack.md)，或更新版本。 <br/><br/>用來自動部署和更新使用 Visual Studio Team Services (VSTS) 的應用程式。 專案包含`BizTalkServerInventory.json`在部署期間使用 VSTS 的檔案。 |
