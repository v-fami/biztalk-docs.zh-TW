# <a name="single-sign-on-event-10788"></a>單一登入： 事件 10788
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10788|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_DTC_IMPORT_FAILED1|  
|訊息文字|無法匯入 DTC 交易。 請檢查 MSDTC 已正確設定為支援遠端作業。 請參閱文件以取得詳細資料。|  
  
## <a name="explanation"></a>說明  
 無法匯入 DTC 交易。  
  
## <a name="user-action"></a>使用者動作  
 請檢查 MSDTC 已正確設定為支援遠端作業，並在指定的電腦，如需詳細資訊，請參閱事件日誌。  
  
 如需 MSDTC 問題的說明，請參閱[疑難排解 MSDTC 問題的](../core/troubleshooting-problems-with-msdtc.md)。
