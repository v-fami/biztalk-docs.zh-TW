# <a name="how-to-select-multiple-links-and-functoids"></a>如何選取多個連結與運算質
當您想要在對應中執行類似的運算質和/或連結的群組上的作業時，您可以選取該運算質和/或連結的群組，所有在相同的時間。 本主題提供如何執行此作業的相關資訊。  
  
## <a name="prerequisites"></a>必要條件  
 這些指示需要執行中的 BizTalk 對應工具。  
  
## <a name="why-do-you-need-to-bulk-select-linksfunctoids"></a>為什麼會需要大量選取連結/運算質？  
 大量選取連結和 (或) 運算質，可讓您執行下列其中一個動作：  
  
-   複製/剪下選取範圍並貼到同一個格線頁或是同一個對應中的其他格線頁。  
  
    > [!NOTE]
    >  如需如何剪下、 複製和貼上連結和運算質的資訊，請參閱[如何剪下、 複製和貼上連結和運算質](../core/how-to-copy-cut-and-paste-links-and-functoids.md)。  
  
-   在格線頁之間移動選取範圍  
  
    > [!NOTE]
    >  如需如何將關係從一個格線頁移至另一個資訊，請參閱[如何移動關係格線頁之間](../core/how-to-move-a-relationship-between-grid-pages.md)。  
  
-   編輯選取範圍中運算質和連結的通用屬性  
  
    > [!NOTE]
    >  如需如何設定註解和運算質和連結的標籤資訊，請參閱[如何標示連結](../core/how-to-label-a-link.md)或[如何加上標籤與註解的運算質](../core/how-to-label-and-comment-a-functoid.md)。  
  
-   以一個步驟刪除多個連結和 (或) 運算質  
  
    > [!NOTE]
    >  如需如何刪除運算質的資訊，請參閱[如何刪除運算質](../core/how-to-delete-functoids.md)。  
  
## <a name="to-select-multiple-links-and-functoids"></a>若要選取多個連結和運算質  
 您可以任何下列方式大量選取運算質和 (或) 連結：  
  
-   按一下連結或運算質。 按住 CTRL 鍵，然後按一下您想要選取的其他連結和 (或) 運算質。  
  
     下圖顯示選取的運算質和連結 (已反白顯示)。  
  
     ![大量選取運算質和連結](../core/media/bulkselect-functois-links.gif "BulkSelect_Functois （& s) 的連結")  
  
-   在對應介面上拖曳滑鼠。 選取矩形下的運算質會反白顯示。  
  
     ![運算質的矩形選取範圍](../core/media/bulkselect-selectionrectangle.gif "BulkSelect_SelectionRectangle")  
  
-   您可以結合運用拖曳和 CTRL 鍵。 在對應介面上拖曳滑鼠，選取該範圍內的運算質和 (或) 連結。 按住 CTRL 鍵，然後按一下選取範圍外的運算質和 (或) 連結。  
  
## <a name="to-select-all-links-and-functoids-on-the-grid-page"></a>若要選取格線頁上的所有連結和運算質  
 您可以使用下列任一方式選取 [對應工具] 格線頁上的所有運算質和連結：  
  
-   以滑鼠右鍵按一下格線頁上的任何位置，然後按一下**全選**。  
  
-   按一下格線頁的任何位置，然後按鍵盤的 CTRL+A。  
  
-   按一下格線頁的任何位置。 從 Visual Studio 功能表中，按一下 **編輯**，然後按一下 **全選**。
