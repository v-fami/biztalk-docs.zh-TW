# <a name="business-activity-monitoring-biztalk-server-samples-folder"></a>商務活動監控 （BizTalk Server 範例資料夾）
Microsoft BizTalk Server 的軟體開發套件 (SDK) 中包含數個商務活動監控 (BAM) 範例。 本節將提供每個 BAM 範例所展示功能的詳細資訊、用於建置和執行範例的指示，以及您可以預期的結果。  
  
## <a name="in-this-section"></a>本節內容  
  
-   [BAM API （BizTalk Server 範例）](../core/bam-api-biztalk-server-sample.md)。 示範如何進行下列工作：  
  
    -   商務分析師會使用精靈來定義有意義的資料，檢查彙總，以及查詢活動資料。  
  
    -   資訊技術 (IT) 系統管理員則建立 SQL、Data Transformation Services (DTS) 和線上分析處理 (OLAP) 基礎結構。  
  
    -   開發人員會進行商務實作。  
  
-   [BAM 端對端 （BizTalk Server 範例）](../core/bam-end-to-end-biztalk-server-sample.md)。 示範如何使用 BAM 來使多個元件中的事件相互關聯。  
  
-   [從協調流程運算式 （BizTalk Server 範例） 的 BAM API](../core/bam-api-from-an-orchestration-expression-biztalk-server-sample.md)。 示範如何進行下列工作：  
  
    -   從 BizTalk Server 協調流程使用 BAM API。  
  
    -   將訊息內的重複項目當做個別活動執行個體進行追蹤。  
  
    -   建立自協調流程中使用追蹤設定檔所追蹤 BAM 資料和自 API 所追蹤資料之間的關係。
