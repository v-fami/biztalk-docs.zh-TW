# <a name="compiler-directives-and-links"></a>編譯器指示詞和連結
您可以設定以下兩個以逐一連結為基礎的編譯器指示詞：  
  
-   從輸入執行個體訊息中擷取何種資料，並複製為輸出執行個體訊息中相關的項目或屬性值。  
  
    > [!NOTE]
    >  此處的資料包含複製項目名稱或屬性本身的選項，而不是任何與項目或屬性相關聯的值。 項目和屬性名稱一般不會被視為 XML 執行個體訊息中附帶資料的一部分。  
  
-   如何執行來源與目的結構描述之間的節點比對。  
  
 本節提供這些指示詞可用選項的詳細說明。
