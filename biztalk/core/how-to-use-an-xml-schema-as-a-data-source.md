# <a name="how-to-use-an-xml-schema-as-a-data-source"></a>如何使用 XML 結構描述做為資料來源
### <a name="to-specify-an-xml-schema-as-a-data-source"></a>指定 XML 結構描述作為資料來源  
  
1.  在 事實總管 中，按一下**XML 結構描述** 索引標籤。  
  
2.  以滑鼠右鍵按一下**結構描述**資料夾，然後再按一下**瀏覽**。  
  
3.  在**結構描述檔案**對話方塊中，選取一個 XSD 檔案，，然後按一下**開啟**。
