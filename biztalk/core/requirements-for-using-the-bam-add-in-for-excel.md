# <a name="requirements-for-using-the-bam-add-in-for-excel"></a>使用 Excel 的 BAM 增益集的需求
商務分析師會使用 Excel 的 BAM 增益集來定義要追蹤的事件和資料，以及使用者所能存取之追蹤資料檢視的邏輯群組和彙總。  
  
## <a name="bam-add-in-for-excel-requirements"></a>Excel 的 BAM 增益集需求  
 您可以在任何電腦上執行 Excel 的 BAM 增益集，不論這台電腦是否安裝了 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。 不過，您必須先安裝下表所列的檔案，才能讓 Excel 的 BAM 增益集正常運作。 請參閱 Microsoft Excel 文件以了解這些檔案所放置的位置。  
  
-   Excel.exe  
  
-   Vbe7.dll  
  
-   Stdole2.tlb  
  
-   Mso.dll  
  
-   Fm20.dll  
  
-   Scrrun.dll  
  
-   Mscomctl.ocx  
  
-   Msado15.dll
