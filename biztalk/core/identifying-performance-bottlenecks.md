# <a name="identifying-performance-bottlenecks"></a>識別效能瓶頸
理想情況下，當系統以將近滿載的條件執行 (充分利用可用的資源) 時，便可達到維持輸送量的目標，從而降低整體擁有成本 (TCO)。
