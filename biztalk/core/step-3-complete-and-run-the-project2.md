# <a name="step-3-complete-and-run-the-project"></a>步驟 3： 完成及執行專案
在新增組件金鑰及繫結連接埠之後，接著為測試協調流程。  
  
 如果您使用相同的物件，則可以使用 Sample XML BeginDoc 進行測試。 如果沒有，您可以將[步驟 4： 建立範例 XML BeginDoc](../core/step-4-create-a-sample-xml-begindoc1.md)因為這是剛開始整個程序的觸發程序。  
  
### <a name="to-complete-and-run-the-project"></a>完成及執行專案  
  
1.  在 [方案總管] 中，以滑鼠右鍵按一下您的專案，然後**建置**。  
  
2.  以滑鼠右鍵按一下您的專案，然後選取**部署**。  
  
3.  在 BizTalk Server 管理主控台中，選取**繫結**，**登錄**和**啟動**來測試您的協調流程。
