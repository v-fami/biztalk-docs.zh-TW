# <a name="receiveportcollection-node"></a>ReceivePortCollection 節點
繫結檔案的 [ReceivePortCollection] 節點是所有 [ReceivePort] 節點的父節點，這些節點包含隨同繫結檔案一起匯出之接收埠的特定相關資訊。  
  
## <a name="nodes-in-the-receiveportcollection-node"></a>ReceivePortCollection 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[接收埠](../core/receiveport-receiveportcollection-node.md)|記錄|ReceivePort (ComplexType)|指定隨同繫結檔案一起匯出之接收埠的相關資訊。|不需要|預設值：無|
