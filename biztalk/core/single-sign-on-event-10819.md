# <a name="single-sign-on-event-10819"></a>單一登入： 事件 10819
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10819|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_MAPPING_CONFLICT|  
|訊息文字|外部帳戶未更新，因為有對應衝突。|  
  
## <a name="explanation"></a>說明  
 外部帳戶未更新，因為有對應衝突。  
  
## <a name="user-action"></a>使用者動作  
 如果這是預期中的行為，則不需要任何動作，此訊息僅供參考。 如果應該允許對應衝突，則據此設定應用程式。  
  
 如需對應衝突的詳細資訊，請參閱**密碼同步配接器屬性： 選項** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
