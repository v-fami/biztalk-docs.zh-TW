# <a name="required-ports-for-biztalk-server"></a>BizTalk Server 的必要連接埠
如需保護 BizTalk Server 部署安全的完整資訊，請參閱[範例 BizTalk Server 架構](../core/sample-biztalk-server-architectures.md)。  
  
 本節提供需要在防火牆上開啟的連接埠的相關資訊，使 BizTalk Server 元件可以相互通訊。  
  
 若除了使用的 BizTalk Server 之外，生產環境中還有其他伺服器，您可能需要開啟其他連接埠。 如需 Windows Server System™ 的連接埠需求的詳細資訊，請參閱[http://go.microsoft.com/fwlink/?LinkId=25713](http://go.microsoft.com/fwlink/?LinkId=25713)。
