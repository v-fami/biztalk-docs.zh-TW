## <a name="explanation"></a>說明  
 SSO 可以建立最大 SQL 資料庫名稱長度為 123 個字元，以允許預設的記錄檔名稱。  
  
## <a name="user-action"></a>使用者動作  
 使用符合這些準則的名稱。
