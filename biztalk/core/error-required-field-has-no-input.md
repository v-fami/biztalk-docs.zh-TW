# <a name="error---required-field-has-no-input"></a>錯誤-必要的欄位有沒有輸入
**錯誤碼**  
  
 btm1028  
  
 **說明**  
  
 目的結構描述中的指定節點已指定為必要，但沒有連入連結、常數值或預設值，因此將不會包含在此對應所提供的輸出執行個體訊息中。  
  
 **使用者動作**  
  
 視需要而定，將目的結構描述中的指定節點變更為選擇性，或為它提供連入連結、常數值或預設值。
