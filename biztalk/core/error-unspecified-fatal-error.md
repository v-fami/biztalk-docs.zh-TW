# <a name="error---unspecified-fatal-error"></a>錯誤-未指定的嚴重錯誤
**錯誤碼**  
  
 BEC1001, BEC2004, BEC3001  
  
 **說明**  
  
 編譯結構描述時發生未指定的嚴重錯誤。  
  
 **使用者動作**  
  
 確認結構描述檔案有效且未損毀。
