# <a name="consuming-web-services"></a>使用 Web 服務
使用 Web 服務可讓您將現有 Web 服務加入至商務程序。 您可以將數個 Web 服務彙總成為一個協調流程。  
  
 您可以透過 Web 連接埠，從協調流程使用 (呼叫) Web 服務。 若要從協調流程取用 Web 服務，您必須建立 Web 連接埠，並建構 Web 訊息。  
  
 您可以搭配已使用的 Web 服務來使用 SOAP 標頭，變更已使用 Web 服務的 URI，並且為已使用的 Web 服務動態設定 URI。  
  
 如需設定 SOAP 接收處理常式，請參閱[如何設定 SOAP 接收處理常式](../core/how-to-configure-a-soap-receive-handler.md)。 如需設定 SOAP 接收位置，請參閱[如何設定 SOAP 接收位置](../core/how-to-configure-a-soap-receive-location.md)。
