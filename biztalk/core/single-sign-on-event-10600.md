# <a name="single-sign-on-event-10600"></a>單一登入： 事件 10600
## <a name="details"></a>詳細資料  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  產品名稱   |                 企業單一登入                  |
| 產品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件識別碼     |                           10600                            |
|  事件來源   |                           ENTSSO                           |
|    元件    |                            不適用                             |
|  符號名稱  |            SSO_ERROR_CS_CALLBACK_ACCESS_DENIED             |
|  訊息文字   |            組態存放區伺服器存取 denied.%r            |
  
## <a name="explanation"></a>說明  
 訊息已傳送至組態存放區伺服器，但不是被接受。 原因可能是由數項不同的原因，例如不正確的通訊協定或安全性權限不足。
