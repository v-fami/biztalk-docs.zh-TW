# <a name="parties"></a>合作對象
合作對象或組織單位代表 BizTalk Server 外部與協調流程互動的實體。 在 BizTalk Server 中，與您交換訊息的每個組織都以合作對象來表示。 藉由在角色中登錄，您可以定義合作對象互動的方式。
