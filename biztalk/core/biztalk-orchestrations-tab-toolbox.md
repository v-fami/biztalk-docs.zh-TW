# <a name="biztalk-orchestrations-tab-toolbox"></a>BizTalk 協調流程 索引標籤工具箱
當您開啟使用協調流程設計師的專案 ([!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]協調流程)，也會顯示 [工具箱] **BizTalk 協調流程** 索引標籤。呼叫這個索引標籤上的工具*圖形*，為可見和可供使用時**BizTalk 協調流程** 索引標籤就會展開。  
  
 若要使用 BizTalk 協調流程**協調流程**索引標籤上，將它拖曳至設計介面的作用中的 BizTalk 協調流程。 您可以拖曳**連接埠**圖形和**角色連結**圖形拖曳到左側連接埠介面或右 連接埠介面。 到中央的流程領域中，您可以將其他所有圖形 — 描述在協調流程的程序流程的圖形。  
  
 如需使用 [工具箱] 的詳細資訊和**工具箱**索引標籤上，請參閱中的 「 管理索引標籤和項目工具箱中的"[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]文件。
