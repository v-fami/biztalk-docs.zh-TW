# <a name="what-is-the-soap-adapter"></a>什麼是 SOAP 配接器？
Web 服務是一些程式，其介面符合為 Web 服務描述語言 (WSDL) 設定的標準。 您可以使用 Microsoft BizTalk Server 協調流程來建立和使用 Web 服務將容易管理而且直覺的方式結合不同但相關的商務功能。 如需 Web 服務和協調流程的詳細資訊，請參閱[使用 Web Services](../core/using-web-services.md)。  
  
 「BizTalk Web 服務發佈精靈」包含在 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 中，可加速發佈 Web 服務的開發程序。 建立用來公佈為 Web 服務的協調流程之後，「BizTalk Web 服務發佈精靈」會為您產生一個 Web 服務專案。 如需 BizTalk Web 服務發佈精靈的詳細資訊，請參閱[協調流程發佈為 Web 服務](../core/publishing-an-orchestration-as-a-web-service.md)。 請注意，執行「BizTalk Web 服務發佈精靈」之前，必須啟用 BizTalk Web 服務。 如需啟用 BizTalk Web 服務的詳細資訊，請參閱[啟用 Web 服務](../core/enabling-web-services.md)。  
  
 SOAP 配接器包含兩個配接器 — 傳送配接器和接收配接器。
