# <a name="orchestration-designer-keyboard-shortcuts"></a>協調流程設計師鍵盤快速鍵
「協調流程設計師」設計介面提供可透過鍵盤使用的完整協助工具。 當您開啟的設計介面中，您可以執行 Visual Studio 中，選取從開始功能表選項時，會啟動此協助工具**新增**選項**檔案**功能表。  
  
 設計介面中也支援協助工具，例如，您可以按向上鍵和向下鍵來選取不同的圖形及圖形部分。 在設計介面上選取圖形時，它會反白顯示在設計介面和 [協調流程檢視] 視窗 （如果適用），同時**屬性**方格也會顯示所選取圖形的相關資訊。  
  
 「協調流程設計師」透過提供一些鍵盤快速鍵來支援協助工具。 依照您目前所使用的不同設計介面部分，可使用的鍵盤快速鍵也會不同。  
  
 您也可以使用 Visual Studio 中許多現有可用的快速鍵，就如同在 [BizTalk 運算式編輯器] 中使用 Microsoft IntelliSense 一樣。
