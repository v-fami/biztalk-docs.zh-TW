# <a name="single-sign-on-event-10852"></a>單一登入： 事件 10852
## <a name="details"></a>詳細資料  
  
|                 |                                                                                               |
|-----------------|-----------------------------------------------------------------------------------------------|
|  產品名稱   |                                   企業單一登入                                   |
| 產品版本 |                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                   |
|    事件識別碼     |                                             10852                                             |
|  事件來源   |                                            ENTSSO                                             |
|    元件    |                                              不適用                                              |
|  符號名稱  |                         ENTSSO_E_CREATE_FIELD_NO_DIRECT_PASSWORD_SYNC                         |
|  訊息文字   | 無法建立欄位，因為這個應用程式已設定「直接密碼同步」旗標。 |
  
## <a name="explanation"></a>說明  
 當應用程式已設定直接密碼同步旗標時，您無法建立任何欄位。  
  
## <a name="user-action"></a>使用者動作  
 使用者不必採取任何動作。 不支援這個選項。
