# <a name="single-sign-on-event-11033"></a>單一登入： 事件 11033
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|11033|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|SSO_INFO_PS_WIN_CHANGE_APP_DISABLED|  
|訊息文字|Windows 密碼變更。 已偵測到此 Windows 帳戶的對應，但忽略，因為應用程式已 disabled.%r<br /><br /> 追蹤識別碼: %1 %r<br /><br /> Windows 帳戶: %2 %r<br /><br /> 應用程式: %3 %r<br /><br /> 外部帳戶: %4 %r<br /><br /> 用戶端使用者： %5|  
  
## <a name="explanation"></a>說明  
 已偵測到此 Windows 帳戶的對應但忽略，因為已停用應用程式。  
  
## <a name="user-action"></a>使用者動作  
 若要啟用應用程式，請參閱[如何啟用分支機構應用程式](../core/how-to-enable-an-affiliate-application.md)。
