# <a name="nil-value-functoid"></a>Nil 值運算質

## <a name="description"></a>Description
**「 Nil 值**運算質設定目的地的項目**Nil**。 當您想在輸出訊息中包含空白項目時，便可使用這種運算質。 它可以結合其他運算質，以產生**Nil**值根據特定條件。
