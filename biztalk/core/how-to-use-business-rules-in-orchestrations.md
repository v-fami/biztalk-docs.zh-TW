# <a name="how-to-use-business-rules-in-orchestrations"></a>如何在協調流程中使用商務規則
您可以建立商務規則原則的執行個體，並在協調流程中執行它。 若要這麼做，請在不可部分完成的交易範圍中新增「呼叫規則」圖形，並在此圖形上設定原則。
