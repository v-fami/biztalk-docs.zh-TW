# <a name="silent-configuration-of-the-mqseries-adapter"></a>MQSeries 配接器的無訊息組態
您可從命令列使用「MQSAgent COM+ 組態精靈」，以執行無訊息組態。 您也可使用命令列版本的精靈來移除 MQSAgent。 精靈會從 XML 檔案讀取組態資訊。
