# <a name="error---multiple-loop-paths"></a>錯誤-多個迴圈路徑
**錯誤碼**  
  
 btm1030  
  
 **說明**  
  
 在目的結構描述中指定的節點，有多個來源迴圈路徑。  
  
 **使用者動作**  
  
 使用**迴圈**的迴圈會執行來源結構描述中，明確指定節點的運算質。
