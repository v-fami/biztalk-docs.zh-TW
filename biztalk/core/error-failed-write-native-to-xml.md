# <a name="error---failed-write-native-to-xml"></a>錯誤-無法寫入原生至 XML
**錯誤碼**  
  
 btm1043  
  
 **說明**  
  
 寫入指定的 XML 檔案時發生檔案寫入錯誤。  
  
 **使用者動作**  
  
 確認您有足夠的磁碟空間與權限可建立指定的檔案。
