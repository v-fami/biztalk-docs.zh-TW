# <a name="sendports-distributionlist-node"></a>SendPorts (DistributionList 節點)
繫結檔案之 DistributionList 節點的 SendPorts 節點是通訊群組清單中傳送埠參考的容器節點。  
  
> [!NOTE]
>  通訊群組清單在 BizTalk 系統管理員中稱為傳送埠群組。  
  
## <a name="nodes-in-the-sendports-node"></a>SendPorts 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[SendPortRef](../core/sendportref-sendports-node.md)|記錄|SendPortRef (ComplexType)|傳送埠參考的容器節點 (此傳送埠為通訊群組清單所產生)。|不需要|預設值：無|
