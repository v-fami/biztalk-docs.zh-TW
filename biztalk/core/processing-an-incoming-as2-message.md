 <span data-ttu-id="42c94-116">若 MDN 將以同步方式傳回，則 MDN 就會透過雙向接收位置的傳送埠傳回。</span><span class="sxs-lookup"><span data-stu-id="42c94-116">If the MDN will be returned synchronously, the MDN is returned over the send port of the two-way receive location.</span></span> <span data-ttu-id="42c94-117">這個 MDN 是做為 HTTP 回應，例如 200OK 表示成功接收訊息。</span><span class="sxs-lookup"><span data-stu-id="42c94-117">This MDN serves as the HTTP response, for example, a 200OK indicating successful reception of the message.</span></span>
