# <a name="moving-biztalk-server-databases"></a>移動 BizTalk Server 資料庫
在任何作用中的資料中心中，有時必須將資料庫移動到不同的硬體。 例如，當資料庫大小比其硬體大，或是硬體升級或以較新的裝備替換時，您就應該需要移動資料庫。 本節中的程序描述如何移動最常被移動的 BizTalk Server 資料庫。
