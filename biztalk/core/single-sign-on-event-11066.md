# <a name="single-sign-on-event-11066"></a>單一登入： 事件 11066
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|11066|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|SSO_INFO_PASSWORD_SYNC_MIIS_STARTED|  
|訊息文字|已 started.%r 的 （來自 MIIS) 的 Windows 密碼同步。|  
  
## <a name="explanation"></a>說明  
 這是資訊訊息，指出已啟動的 （來自 MIIS) 的 Windows 密碼同步處理。  
  
## <a name="user-action"></a>使用者動作  
 使用者不必採取任何動作。
