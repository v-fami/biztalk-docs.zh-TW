# <a name="single-sign-on-event-10721"></a>單一登入： 事件 10721
## <a name="details"></a>詳細資料  

|                 |                                                                                 |
|-----------------|---------------------------------------------------------------------------------|
|  產品名稱   |                            企業單一登入                            |
| 產品版本 |           [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]            |
|    事件識別碼     |                                      10721                                      |
|  事件來源   |                                     ENTSSO                                      |
|    元件    |                                       N\A                                       |
|  符號名稱  |                        SSO_INFO_REPLAY_FILE_CLOSED_FULL                         |
|  訊息文字   | 重新執行檔案已關閉，因為它已滿。%r<br /><br /> 檔案名稱： %1 |

## <a name="explanation"></a>說明  
 這個資訊事件表示重新執行檔案已關閉，因為它已滿。 必要時，會建立新的重新執行檔案。
