# <a name="edi-schemas"></a>EDI 結構描述
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] EDI 和 AS2 使用下列類型的結構描述來處理內送訊息和產生外寄訊息：  
  
-   定義訊息內文的文件結構描述  
  
-   定義訊息信封的標頭服務結構描述，以及定義技術通知和功能通知之信封和內文的通知控制結構描述。  
  
-   定義所保留批次之信封的 XML (批次) 結構描述。  
  
-   定義 EDI 內容屬性的屬性結構描述。 如需詳細資訊，請參閱[EDI 內容屬性](../core/edi-context-properties.md)。
