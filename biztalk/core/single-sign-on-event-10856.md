# <a name="single-sign-on-event-10856"></a>單一登入： 事件 10856
## <a name="details"></a>詳細資料  
  
|                 |                                                                |
|-----------------|----------------------------------------------------------------|
|  產品名稱   |                   企業單一登入                    |
| 產品版本 |   [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    事件識別碼     |                             10856                              |
|  事件來源   |                             ENTSSO                             |
|    元件    |                              不適用                               |
|  符號名稱  |               ENTSSO_E_MAPPING_CREATE_RESTRICTED               |
|  訊息文字   | 應用程式使用者不可建立這個應用程式的對應。 |
  
## <a name="explanation"></a>說明  
 此應用程式已設定應用程式使用者群組的成員不允許建立對應的方式。  
  
## <a name="user-action"></a>使用者動作  
 您應該要求具有足夠的權限的人員，若要建立這些對應。
