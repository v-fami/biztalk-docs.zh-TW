# <a name="single-sign-on-event-10829"></a>單一登入： 事件 10829
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10829|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|ENTSSO_E_PSADMIN_INVALID_ADAPTER_TYPE_2|  
|訊息文字|無效的應用程式配接器的類型。 有效的應用程式類型為 「 密碼同步配接器 」 或 「 密碼同步群組配接器 」。|  
  
## <a name="explanation"></a>說明  
 無效的應用程式配接器的類型。 有效的應用程式類型為 「 密碼同步配接器 」 或 「 密碼同步群組配接器 」。  
  
## <a name="user-action"></a>使用者動作  
 使用其中一種有效的應用程式類型。
