# <a name="admin-explorerom-biztalk-server-samples-folder"></a>系統管理員 ExplorerOM （BizTalk Server 範例資料夾）
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 在其軟體開發套件 (SDK) 的 Admin\ExplorerOM 資料夾中包含了範例。 本節會詳細介紹每個 BizTalk 總管物件模型管理範例所示範的功能、用來建置和執行範例的指示，以及可預期結果。  
  
> [!WARNING]
>  如果在部署後不需要部署指令碼，就應該將其移除。 必須保留的系統管理指令碼和其他指令碼都應該由 ACL 保護，而且予以密切監控。  
  
> [!NOTE]
>  Microsoft.BizTalk.ExplorerOM.dll 同時支援 32 位元和 64 位元程序。  
  
## <a name="in-this-section"></a>本節內容  
  
-   [（BizTalk Server 範例） ApplicationManager](../core/applicationmanager-biztalk-server-sample.md)。 示範如何啟動或停止 BizTalk 應用程式。  
  
-   [（BizTalk Server 範例） BrowsingArtifacts](../core/browsingartifacts-biztalk-server-sample.md)。 示範如何列舉 BizTalk 成品和屬性。  
  
-   [CBR （BizTalk Server 範例）](../core/cbr-biztalk-server-sample.md)。 示範如何加入和設定新的內容為基礎的路由的 BizTalk 訊息的傳送埠。  
  
-   [DeleteParty （BizTalk Server 範例）](../core/deleteparty-biztalk-server-sample.md)。 示範如何刪除指定的合作對象。  
  
-   [OrchestrationBinding （BizTalk Server 範例）](../core/orchestrationbinding-biztalk-server-sample.md)。 示範如何設定及管理協調流程。  
  
-   [（BizTalk Server 範例） PartnerManagement](../core/partnermanagement-biztalk-server-sample.md)。 示範如何建立和刪除合作對象。 也會示範與角色登錄和取消登錄合作對象。  
  
-   [ReceiveLocations （BizTalk Server 範例）](../core/receivelocations-biztalk-server-sample.md)。 示範如何建立和管理接收埠的接收位置。  
  
-   [接收埠 （BizTalk Server 範例）](../core/receiveports-biztalk-server-sample.md)。 示範建立、 列舉和刪除接收埠。  
  
-   [傳送埠群組 （BizTalk Server 範例）](../core/sendportgroups-biztalk-server-sample.md)。 示範如何列舉和管理傳送埠群組。  
  
-   [SendPorts （BizTalk Server 範例）](../core/sendports-biztalk-server-sample.md)。 示範如何列舉和管理傳送埠。  
  
-   [UnenlistParties （BizTalk Server 範例）](../core/unenlistparties-biztalk-server-sample.md)。 示範如何取消登錄與已部署 BizTalk 組件關聯的所有合作對象。
