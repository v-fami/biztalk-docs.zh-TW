**錯誤碼**  
  
 btm1084  
  
 **使用者動作**  
  
 將條件加入至 「 Nil 值 」 運算質 （從邏輯運算質或特定布林值新增輸入），其他詳細資料 核取此連結[如何新增 Nil 值運算質至對應](http://go.microsoft.com/fwlink/?LinkId=196675)。
