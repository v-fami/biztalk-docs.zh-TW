# <a name="creating-a-net-application-to-test-a-published-web-service"></a>建立.NET 應用程式來測試已發佈的 Web 服務
若要測試已發佈的 Web 服務，您可以建立 ASP.NET Web 用戶端應用程式，該應用程式會使用您已發佈的 Web 服務。 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 說明集合內有實用的逐步解說，說明如何建立 ASP.NET Web 用戶端應用程式。 您可以使用逐步解說來測試已發佈的 Web 服務。  
  
 逐步解說會建立 ASP.NET Web 用戶端應用程式、新增 Web 參考，並提供範例程式碼，說明如何存取已發佈的 Web 服務。 逐步解說會引導您以偵錯模式執行應用程式。  
  
 如資訊和程序的相關建立 XML Web 服務用戶端專案，請參閱 「 逐步解說:: 存取 XML Web 服務使用 Visual Basic 或 Visual C#"中[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]協助處[http://go.microsoft.com/fwlink/?LinkId=62263](http://go.microsoft.com/fwlink/?LinkId=62263)。 您可以用 BizTalk 已發佈的 Web 服務取代此範例中使用的 Web 服務 TempConvert1。
