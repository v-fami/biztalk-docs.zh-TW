# <a name="sendportref-sendports-node"></a>SendPortRef (SendPorts 節點)
繫結檔案之 SendPorts 節點的 SendPortRef 節點會指定通訊群組清單所參考之傳送埠的名稱。  
  
> [!NOTE]
>  通訊群組清單在 BizTalk 系統管理員中稱為傳送埠群組。  
  
## <a name="nodes-in-the-sendportref-node"></a>SendPortRef 節點中的節點  
 下表列出可以為繫結檔案中的這個節點設定的屬性：  
  
|**名稱**|**節點類型**|**資料類型**|**說明**|**限制**|**註解**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|名稱|Attribute|xs:string|指定通訊群組清單所參考之傳送埠的名稱。|不需要|預設值：空白|
