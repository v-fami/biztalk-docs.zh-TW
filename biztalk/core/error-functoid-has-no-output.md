# <a name="error---functoid-has-no-output"></a>錯誤-運算質有沒有輸出
**錯誤碼**  
  
 btm1013  
  
 **說明**  
  
 所指定的運算質沒有輸出連結，因此無作用。  
  
 **使用者動作**  
  
 針對指定的運算質建立適當的輸出連結。
