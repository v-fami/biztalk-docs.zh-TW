# <a name="soap-headers-with-published-web-services"></a>SOAP 標頭與已發佈的 Web 服務
您可以將 SOAP 標頭加入至已發佈的 Web 服務。 在 BizTalk Web 服務發佈精靈上**Web 服務屬性**頁面上，您可以檢查**加入 SOAP 標頭**方塊將 SOAP 標頭新增至已發行的 Web 服務。
