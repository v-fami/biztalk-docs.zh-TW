# <a name="how-to-search-for-activities"></a>如何搜尋活動
使用 BAM Web 入口網站上的 [商務活動搜尋] 以搜尋儲存在 BAM 資料庫中的追蹤資料 (也稱為執行個體)。  
  
 搜尋會針對目前執行中程序的即時資料以及儲存資料來執行查詢。  
  
### <a name="to-search-the-bam-database"></a>搜尋 BAM 資料庫  
  
1.  在 Internet Explorer 中**位址**方塊中，輸入 BAM Web 入口網站的 URL。  
  
2.  從**我的檢視**瀏覽窗格中，按一下檢視，即可展開檢視來顯示工作清單。  
  
3.  按一下**活動搜尋**工作以顯示活動的清單...  
  
4.  按一下某個活動，將活動搜尋頁面載入至內容框架中。  
  
5.  在**查詢**產生器選擇搜尋的查詢參數。  
  
6.  在**欄位選擇器**選取代表要在搜尋中傳回資料項目的資料行。  
  
7.  按一下**執行查詢**按鈕右上方的 [內容] 框架。  
  
     結果會顯示在**結果**活動搜尋內容頁面的區域。  
  
> [!NOTE]
>  您可以選取最多 20 個資料行。  
  
> [!NOTE]
>  這個版本不支援邏輯運算子的群組功能。
