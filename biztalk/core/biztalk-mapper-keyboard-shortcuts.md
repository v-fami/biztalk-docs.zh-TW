# <a name="biztalk-mapper-keyboard-shortcuts"></a>BizTalk 對應工具鍵盤快速鍵
如同任何標準的 Microsoft Windows 應用程式，您可以使用鍵盤快速鍵在 BizTalk 對應工具來存取命令否則可使用功能表。 下表描述可用的鍵盤快速鍵。  
  
> [!NOTE]
>  以逗點分隔的按鍵組合代表必須依照輸入的順序。 例如，"CTRL + I，CTRL + R"表示，您必須先輸入 CTRL + I，然後再輸入 CTRL + R。 按鍵的組合，例如"ALT + B，I，F"表示您必須先輸入 ALT + B，但可輸入"I"及"F"而不需要按住 ALT 鍵。  
  
> [!IMPORTANT]
>  為讓鍵盤快速鍵能正確運作，必須選取想要的項目。
