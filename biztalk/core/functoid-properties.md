# <a name="functoid-properties"></a>運算質屬性

## <a name="overview"></a>概觀
運算質具有可在 [[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 屬性] 視窗中檢視和修改的屬性。 有些運算質屬性僅供特定運算質使用，其他運算質無法使用。 一個運算質屬性，**輸入參數**幾乎所有運算質，特別重要，可能會最常使用。 如需有關**輸入參數**屬性，請參閱[運算質輸入參數](../core/functoid-input-parameters.md)。  
  
 所有的運算質屬性都歸類為**一般**屬性。 如需運算質屬性的完整清單，請參閱**一般運算質屬性** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)]。
