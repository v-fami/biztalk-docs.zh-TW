## <a name="explanation"></a>說明  
 此錯誤表示找不到結構描述。  
  
## <a name="user-action"></a>使用者動作  
 若要解決這個錯誤，將部署文件結構描述：  
  
1.  啟動**Microsoft Visual Studio**。  
  
2.  建立或開啟現有的專案  
  
3.  將結構描述加入至專案。  
  
4.  部署 Visual Studio 中的專案。
