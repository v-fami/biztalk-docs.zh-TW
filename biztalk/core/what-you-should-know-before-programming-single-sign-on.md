# <a name="what-you-should-know-before-programming-single-sign-on"></a>單一登入程式設計前應該知道什麼
為能有效地使用此文件，您應該熟悉以下項目：  
  
-   Microsoft[!INCLUDE[btsPlatformsComApis](../includes/btsplatformscomapis-md.md)]作業系統。 您尤其應該熟悉 Windows 安全性功能。  
  
-   「企業單一登入」的管理功能，特別是如何使用使用者介面以執行管理動作。  
  
 根據所使用的 API 和開發工具的不同，您也應該熟悉以下介面：  
  
-   COM  
  
-   .NET Framework 和通用語言執行階段  
  
-   Windows 網路，特別是如何傳送和接收票證
