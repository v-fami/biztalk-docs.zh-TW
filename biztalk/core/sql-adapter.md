# <a name="sql-adapter"></a>SQL 配接器
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 不再支援 SQL 配接器。 我們建議您使用 [!INCLUDE[adaptersql](../includes/adaptersql-md.md)] 來達成相同的功能。  
  
 不過，[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 支援 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 隨附的 SQL 配接器元件，如 EDI 控制訊息 SQL 接收管線、或 AS2 重送控制訊息接收管線。  
  
 如需詳細資訊，請參閱[Microsoft BizTalk Adapter Pack 說明](http://go.microsoft.com/fwlink/?LinkId=187919)。
