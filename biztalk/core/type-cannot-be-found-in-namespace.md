## <a name="explanation"></a>說明  
 此錯誤表示所建立的成品參考指定的命名空間中找不到型別。  
  
## <a name="user-action"></a>使用者動作  
 加入包含找不到，類型的參考，然後再次執行精靈，（如果您擁有想要使用的 WCF 服務）。 否則，請連絡服務提供者。
