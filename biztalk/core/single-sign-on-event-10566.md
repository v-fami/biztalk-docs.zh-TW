# <a name="single-sign-on-event-10566"></a>單一登入： 事件 10566
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|10566|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|SSO_WARN_CANNOT_UPDATE_APP_FLAGS|  
|訊息文字|您無法更新某些指定的旗標，為此應用程式。 將予以 ignored.%r<br /><br /> 應用程式名稱: %1 %r<br /><br /> 指定旗標遮罩： %2|  
  
## <a name="explanation"></a>說明  
 無法變更某些應用程式中的旗標。 （例如，它不是允許變更應用程式類型從個別群組。）中的警告文字指定此應用程式的旗標。  
  
## <a name="user-action"></a>使用者動作  
 使用者不必採取任何動作。
