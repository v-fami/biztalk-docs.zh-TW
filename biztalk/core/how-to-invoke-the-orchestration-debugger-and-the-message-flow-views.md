# <a name="how-to-invoke-the-orchestration-debugger-and-the-message-flow-views"></a>如何叫用協調流程偵錯工具和訊息流程檢視
在 BizTalk Server 管理主控台的 [群組中樞] 頁面中，[查詢結果] 窗格會顯示查詢的輸出。 從內容功能表中，您可以顯示 [協調流程偵錯工具] 或 [訊息流程] 檢視。  
  
## <a name="prerequisites"></a>必要條件  
 您必須以 BizTalk Server 系統管理員群組的成員身分登入，才能執行這個程序。  
  
### <a name="to-display-the-orchestration-debugger-or-message-flow-views"></a>若要顯示協調流程偵錯工具或訊息流程檢視  
  
1.  若要顯示訊息流程檢視，以滑鼠右鍵按一下中的協調流程執行個體相關聯的資料格**查詢結果**清單並選取**訊息流程**。  
  
2.  若要顯示協調流程偵錯工具，以滑鼠右鍵按一下與協調流程執行個體中相關聯的資料格**查詢結果**清單並選取**協調流程偵錯工具**。
