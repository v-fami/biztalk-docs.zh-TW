# <a name="configuring-ports-for-an-edi-solution"></a>設定 EDI 解決方案的連接埠
若要接收和傳送 EDI 訊息與通知，請建立下列接收埠與傳送埠：  
  
-   接收 EDI 交換和通知的單向靜態 FILE 接收埠，或接收 EDI 交換和傳送通知的雙向要求-回應 FILE 接收埠。  
  
-   傳送 EDI 交換或通知的單向 FILE 傳送埠。
