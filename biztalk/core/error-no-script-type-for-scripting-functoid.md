# <a name="error---no-script-type-for-scripting-functoid"></a>錯誤-指令碼處理運算質沒有指令碼類型
**錯誤碼**  
  
 btm1003  
  
 **說明**  
  
 **指令碼類型**屬性所指定**指令碼處理**運算質未設定。  
  
 **使用者動作**  
  
 選取指定**指令碼處理**運算質，按一下省略符號 (**...**) 相關聯的按鈕**指令碼**屬性在 microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]屬性 視窗中，然後在中設定適當的值**設定運算質指令碼**對話方塊中，包括**指令碼類型**屬性。
