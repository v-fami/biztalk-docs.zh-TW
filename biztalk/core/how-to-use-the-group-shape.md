# <a name="how-to-use-the-group-shape"></a>如何使用 「 群組 」 圖形
![](../core/media/ebiz-orch-group.gif "ebiz_orch_group")  
[群組] 圖形  
  
## <a name="annotations-and-placeholders"></a>註解和預留位置  
 您可以使用**群組**圖形做為預留位置的功能尚未加入，或者您可以使用它來變更附註解會在其中發生的動作。  
  
 上的 name 屬性**群組**圖形可以是最多 512 個字元。 您可以輸入註解**名稱**屬性和摺疊圖形時，會顯示完整名稱。 當圖形展開時，名稱則會被截斷。
