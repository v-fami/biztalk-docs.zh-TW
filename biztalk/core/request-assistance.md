# <a name="request-assistance"></a>要求協助
當您需要協助從您的系統管理員或其他支援人員與您[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]實作中，您可以使用**協助**底部的按鈕**結果**詳細資料若要以提出協助要求的頁面。  
  
## <a name="how-to-request-assistance"></a>如何要求協助  
 按一下**協助**按鈕會開啟**要求技術協助 網頁**頁面 對話方塊。 若要要求協助，您輸入描述問題中的主旨**主旨**文字方塊。 在**問題描述**文字方塊中描述您的系統管理員需要解決問題的詳細資料所遭遇的問題。 當您滿意問題的說明，請按一下**傳送報表**按鈕以傳送協助要求。 您的要求會放置於伺服器應用程式事件記錄檔，且您的系統管理員可以擷取使用的訊息和服務執行個體追蹤，從 [群組概觀] 頁面...
