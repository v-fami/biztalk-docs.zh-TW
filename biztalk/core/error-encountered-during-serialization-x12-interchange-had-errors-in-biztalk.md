# <a name="error-encountered-during-serialization-the-x12-interchange-had-the-following-errors"></a>在序列化期間發生錯誤。 X12 交換發生下列錯誤
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]|  
|產品版本|[!INCLUDE[btsEDIVersion](../includes/btsediversion-md.md)]|  
|事件識別碼|-|  
|事件來源|[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]EDI|  
|元件|EDI 引擎|  
|符號名稱|X12InterchangeSendError|  
|訊息文字|在序列化期間發生錯誤。 X12 交換傳送者識別碼 '{1}'，接收者識別碼 '{2}' 識別碼為 '{0}'，發生下列錯誤|  
  
## <a name="explanation"></a>說明  
 這個錯誤/警告/資訊事件表示 EDI 傳送管線發生錯誤，當序列化 X12 外寄交換，因為敘述的錯誤。  
  
## <a name="user-action"></a>使用者動作  
 若要解決這個錯誤，找出交換中的錯誤，，然後決定產品說明中的問題解決方法需要使用錯誤訊息中的資訊。
