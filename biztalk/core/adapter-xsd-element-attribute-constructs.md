# <a name="adapter-xsd-element-attribute-constructs"></a>配接器 XSD 項目屬性建構
下表列出配接器架構支援的 XSD 項目屬性建構。 垂直軸包含支援的項目類型，水平軸則包含每個項目的有效屬性類型。  
  
> [!NOTE]
>  項目和屬性都必須具有唯一名稱。 重複的名稱會造成編譯錯誤。  
  
> [!NOTE]
>  不支援具有列舉 Facet 的固定屬性。  
  
> [!NOTE]
>  組態 XSD 應該具有項目稱為**uri**的型別 xsd: string。
