# <a name="how-to-label-a-link"></a>如何標示連結
標籤有助於記錄對應中的運算質或連結用途。 您可以使用**標籤**屬性命名連結。 當您的對應包含大型的結構描述結構，識別運算質的輸入和輸出連結變得困難時，標籤就很有用。  
  
> [!NOTE]
>  如需標籤和註解的運算質的如何資訊，請參閱[如何加上標籤與註解的運算質](../core/how-to-label-and-comment-a-functoid.md)。  
  
 下圖顯示連結標籤。  
  
 ![標示連結](../core/media/new-labelling-link.gif "New_Labelling_link")  
  
## <a name="prerequisites"></a>必要條件  
 這些指示需要執行中的 BizTalk 對應工具。  
  
### <a name="to-add-a-label-to-a-link"></a>新增連結的標籤  
  
1.  選取您要加上標籤的連結。  
  
2.  在**屬性**視窗中，提供新的名稱在**標籤**欄位。  
  
    > [!IMPORTANT]
    >  允許的字元數目上限是 256。 如果指定超過 256 個字元的字串，則接受前 256 個字元，其餘部分會被捨棄。  
  
     ![連結標籤屬性](../core/media/new-to-label-link.gif "New_To_Label_Link")
