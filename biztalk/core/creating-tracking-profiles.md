# <a name="creating-tracking-profiles"></a>建立追蹤設定檔
您可以建立新的追蹤設定檔，也可以修改現有的設定檔，加強對組織特定商務程序的管理與監控。 追蹤設定檔編輯器 (TPE) 可以讓您定義要收集的資料，以符合商務分析師的需求。 您建立或修改的設定檔可以很簡單，也可以很複雜，需視您的商務需求而定。  
  
 身為開發人員，您可以根據 BAM 活動定義建立新的設定檔。 已部署的活動定義可能已經具有為它所定義的設定檔。 如果沒有，請執行下列工作以建立追蹤設定檔：  
  
-   選取部署伺服器與資料庫  
  
-   從 BizTalk 管理資料庫中選取已部署的 BAM 活動定義  
  
-   定義要從協調流程擷取的資料  
  
-   連接活動 (如果商務程序的實際實作橫跨一個以上的協調流程)
