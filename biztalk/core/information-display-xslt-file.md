# <a name="information---display-xslt-file"></a>資訊-顯示 XSLT 檔案
**錯誤碼**  
  
 btm1055  
  
 **說明**  
  
 產生的 XSLT 已寫入至指定的檔案，此 XSLT 檔案會用來執行從執行個體訊息 (符合來源結構描述) 至執行個體訊息 (符合目的結構描述) 的指定對應。  
  
 **使用者動作**  
  
 您可以使用提供的連結來存取結果 XSLT，並使用它在其他工具 (例如 XSLT 偵錯工具) 中執行各種工作。
