# <a name="single-sign-on-event-10747"></a>單一登入： 事件 10747
## <a name="details"></a>詳細資料  

|                 |                                                                                                                                        |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------|
|  產品名稱   |                                                       企業單一登入                                                        |
| 產品版本 |                                       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                       |
|    事件識別碼     |                                                                 10747                                                                  |
|  事件來源   |                                                                  N\A                                                                   |
|    元件    |                                                                  N\A                                                                   |
|  符號名稱  |                                                     SSO_ERROR_UNKNOWN_NOTIFICATION                                                     |
|  訊息文字   | 未知的通知類型是 received.%r<br /><br /> 追蹤識別碼: %1 %r<br /><br /> 配接器: %2 %r<br /><br /> 通知類型： %3 |

## <a name="explanation"></a>說明  
 這個錯誤事件表示 SSO 服務偵測到無效的通知類型發生內部錯誤。  

## <a name="user-action"></a>使用者動作  
 若要解決這個錯誤，執行下列作業：  

- 請檢查系統和應用程式事件記錄檔相關聯的事件。  

  如需詳細資訊，請參閱下列資源：  

- [使用 SSO](../core/using-sso.md)
