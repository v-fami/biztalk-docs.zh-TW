# <a name="wcf-customisolated-adapter"></a>WCF-CustomIsolated 配接器
Wcf-customisolated 配接器是外掛式配接器來設定透過 HTTP 傳輸接收位置的任何繫結和行為設定 Windows Communication Foundation (WCF) 適用於[!INCLUDE[btsCoName](../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。 WCF-CustomIsolated 配接器可以用於標準 WCF 配接器所不支援的實例。 WCF-CustomIsolated 配接器也可以讓您使用 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 的 WCF 擴充性點。  
  
 WCF-CustomIsolated 配接器是用於外掛式主控件中執行的接收位置。 若要使用 WCF-CustomIsolated 配接器，您必須使用 BizTalk WCF 服務發佈精靈，在 Microsoft Internet Information Services (IIS) 中建立對應這個接收位置的虛擬目錄。
