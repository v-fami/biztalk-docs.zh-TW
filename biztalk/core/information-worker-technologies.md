# <a name="information-worker-technologies"></a>資訊工作者技術
Microsoft BizTalk Server 引擎提供自動化的商務程序跨越多個應用程式的支援。 程序建立後，使用程序的資訊工作者 (商務人士，不是開發人員) 會有其他的需求。 例如，他們可能需要監控不同的業務相關程序，或以不同方式與交易夥伴工作。  
  
 BizTalk Server 會提供數個解決這些問題的元件。 這些元件群組於本節描述的「商務活動監控」中。
