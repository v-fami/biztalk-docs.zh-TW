# <a name="service-oriented-solution"></a>服務導向解決方案
服務導向解決方案顯示如何將 BizTalk 應用程式以服務的方式呈現，可做為 Web 服務並可供舊有應用程式存取的形式來使用。 解決方案也顯示如何與做為 Web 服務的後端程序通訊，以及如何從多個後端系統彙總回應。  
  
 以下幾節將提供解決方案的概觀、模式與設計選擇的詳細說明，以及建置與執行解決方案的相關資訊。  
  
> [!NOTE]
>  在這幾節中所記載的模式和指導，用意在於說明特定商務解決方案的處理方式。 模式是簡單的機制，其用意是要套用至特定的問題，而且通常都會與其他的模式合併。 這些模式不是要用來插入到應用程式中。 此範例程式碼係以「現況」提供，而且不適用於實際執行用途。 此範例的目的不在於說明模式，因此並不包括額外的程式碼，像是例外狀況處理、記錄、安全性及驗證。 Microsoft 不支援將範例程式碼以「現況」部署到實際執行環境。
