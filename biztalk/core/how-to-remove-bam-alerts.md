# <a name="how-to-remove-bam-alerts"></a>如何移除 BAM 警示
系統管理員可以使用**移除警示**命令來移除指定的檢視中的所有警示。  
  
### <a name="to-remove-bam-alerts"></a>移除 BAM 警示  
  
1. 開啟命令提示字元，如下所示： 按一下**開始**，按一下**執行**，型別**cmd**，然後按一下 **確定**。  
  
2. 瀏覽到 [!INCLUDE[btsBiztalkServerPath](../includes/btsbiztalkserverpath-md.md)]Tracking。  
  
3. 型別**bm 移除警示-檢視：\<檢視表名稱\>**。  
  
4. 按 ENTER 鍵。
