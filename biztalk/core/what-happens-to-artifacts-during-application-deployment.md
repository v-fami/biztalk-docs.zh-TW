# <a name="what-happens-to-artifacts-during-application-deployment"></a>應用程式部署期間成品所發生的狀況
本章節的主題說明當您新增和移除成品、匯出和匯入應用程式和成品及安裝和解除安裝應用程式時，與應用程式相關的成品所發生的狀況。 本章節涵蓋的主題如下：  
  
-   如何從 BizTalk Server 資料庫新增和移除成品。  
  
-   如何從本機檔案系統複製及刪除成品  
  
-   如何從 Windows 登錄中新增或移除成品 (.NET 組件和 COM 元件)  
  
-   如何在全域組件快取中安裝或解除安裝成品 (BizTalk 組件和 .NET 組件)  
  
-   如何將憑證加入到本機憑證存放區
