# <a name="how-to-create-multi-part-mappings"></a>如何建立多部分對應
若您有多個需共同使用的對應，則必須將它們結合為一個協調流程一起測試。 **測試對應**命令適用於只有一個對應一次。  
  
 如需有關設計和使用協調流程的詳細資訊，請參閱[協調流程使用協調流程設計師建立](../core/creating-orchestrations-using-orchestration-designer.md)。
