# <a name="how-to-select-multiple-functoids"></a>如何選取多個運算質
在處理大量對應時，您可能想要同時在多個運算質上執行作業。 在這種情況下，您可以選取所有的運算質，然後再執行作業。 例如，當您將一組連接的運算質及其連接的連結從一個格線頁移到另一頁時，這種方法就很實用。  
  
### <a name="to-select-multiple-functoids-at-the-same-time"></a>同時選取多個運算質  
  
1.  在 BizTalk 對應工具的格線頁中，按一下第一個運算質以選取。 運算質會反白顯示。  
  
2.  在 BizTalk 對應工具，在相同的格線頁中，按住 CTRL 或 SHIFT 鍵，，然後按一下另一個運算質也加以選取。 此運算質會反白顯示太。  
  
3.  視需要重複步驟 2 以選取所有需要的運算質。  
  
    > [!NOTE]
    >  格線頁上的多個選取作業同時適用於運算質和連結，在將一組連接的運算質及其連接的連結從一個格線頁移到另一頁上時，這是必要的作法。
