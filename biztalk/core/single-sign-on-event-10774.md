# <a name="single-sign-on-event-10774"></a>單一登入： 事件 10774
## <a name="details"></a>詳細資料  
  
|                 |                                                                                      |
|-----------------|--------------------------------------------------------------------------------------|
|  產品名稱   |                              企業單一登入                               |
| 產品版本 |              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]              |
|    事件識別碼     |                                        10774                                         |
|  事件來源   |                                        ENTSSO                                        |
|    元件    |                                         不適用                                          |
|  符號名稱  |                          ENTSSO_E_SSO_NOT_IN_NEW_SSO_ADMIN                           |
|  訊息文字   | SSO 服務帳戶不是指定的 SSO 系統管理員帳戶的成員。 |
  
## <a name="explanation"></a>說明  
 若要變更 SSO 系統管理員帳戶，您必須是該帳戶的成員。  
  
## <a name="user-action"></a>使用者動作  
 有系統管理員協助您連絡 SSO 系統管理員帳戶的成員。
