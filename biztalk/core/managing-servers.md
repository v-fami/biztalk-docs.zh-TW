# <a name="managing-servers"></a>管理伺服器
在 [BizTalk Server 管理] 主控台中的 [伺服器] 節點會列出所有屬於 BizTalk Server 群組的伺服器。 您可以使用本節中的程序，從群組新增或移除伺服器，或從某個群組移動伺服器到另一個群組。
