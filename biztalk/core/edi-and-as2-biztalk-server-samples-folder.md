# <a name="edi-and-as2-biztalk-server-samples-folder"></a>EDI 和 AS2 (BizTalk Server 範例資料夾)
Microsoft [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] 在其軟體開發套件 (SDK) 中包含下列 EDI 和 AS2 範例。 本節提供這些 EDI 範例示範的功能、建置和執行這些範例的指示，以及可預期結果的詳細資訊。  
  
## <a name="in-this-section"></a>本節內容  
  
-   [AS2 over File （BizTalk Server 範例）](../core/as2-over-file-biztalk-server-sample.md)。 示範如何透過 FILE 接收位置來接收 AS2 訊息。  
  
-   [訊息豐富 」 範例 （BizTalk Server 範例）](../core/message-enrichment-sample-biztalk-server-sample.md)。 示範如何將交換標頭附加至 EDI 文件的交易集訊息。  
  
## <a name="additional-sample-code"></a>其他範例程式碼  
  
-   [透過 FILE 傳送埠傳送 AS2 訊息](../core/sending-an-as2-message-over-a-file-send-port.md)。 本主題會說明如何撰寫 AS2 訊息的標頭，以及如何升級 AS2 標頭內容屬性。  
  
-   [撰寫 AS2 內容屬性的輸出合作對象解析](../core/writing-as2-context-properties-for-outbound-party-resolution.md)。 本主題會說明如何將 AS2To 屬性或 UserHttpHeaders 屬性 (具有 AS2 屬性) 寫入至內容，以便能夠執行輸出 AS2 合作對象解析。
