# <a name="bam-and-tracking-assistance"></a>BAM 和追蹤協助
您可以要求透過 BAM 入口網站的協助，依序按一下**協助**活動狀態 頁面上的按鈕。  
  
 協助要求會放置在應用程式事件日誌中。 要求的來源為 BAM Web 服務，並且事件識別碼為 1000。  
  
 訊息內文是在 [描述] 欄位中，並且包含要求的主旨、問題描述文字、BizTalk 服務執行個體識別碼、訊息識別碼、BizTalk 管理資料庫伺服器以及 BizTalk 管理資料庫名稱。
