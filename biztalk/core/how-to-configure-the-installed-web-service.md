# <a name="how-to-configure-the-installed-web-service"></a>如何設定已安裝的 Web 服務
安裝 Web 服務檔案之後，您必須設定 BizTalk Server 從 Web 服務接收訊息。  
  
### <a name="to-configure-the-web-service"></a>若要設定 Web 服務  
  
1.  在 BizTalk Server 管理主控台中，展開  **BizTalk 群組**，依序展開**應用程式**，然後以滑鼠右鍵按一下您想要設定 Web 服務應用程式。  
  
2.  指向**匯入**，然後按一下 **繫結**。  
  
3.  在 [發佈] 資料夾中，選取 BindingInfo.xml 檔案，然後按一下**確定**。  
  
4.  啟動協調流程並啟用接收位置。
