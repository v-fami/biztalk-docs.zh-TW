# <a name="how-to-validate-schemas-in-visual-studio"></a>如何驗證在 Visual Studio 中的結構描述
建構結構描述之後，您可以使用隔離方式驗證結構描述本身，以檢查您的工作。  
  
 本主題提供此驗證工作的逐步說明。  
  
### <a name="to-validate-a-schema-in-isolation"></a>以隔離方式驗證結構描述  
  
1.  在**方案總管] 中**，以滑鼠右鍵按一下您要驗證，然後按一下 [結構描述**驗證結構描述**。  
  
2.  在 [輸出] 視窗中檢視結果。 成功和錯誤訊息會顯示在此視窗中。  
  
> [!NOTE]
>  若 [輸出] 視窗未開啟和顯示結構描述驗證是否成功或失敗的資訊，您可以手動將它開啟。 如需有關管理此視窗的詳細資訊，請參閱[管理其他 Visual Studio 視窗](../core/how-to-manage-other-visual-studio-windows.md)。
