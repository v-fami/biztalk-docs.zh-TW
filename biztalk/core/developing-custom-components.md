# <a name="developing-custom-components"></a>開發自訂元件
您可以在 BizTalk Server 中開發和使用自訂元件，來修改或擴充 BizTalk Server 的特定基礎結構項目。  主要，這包括： 傳送或接收配接器元件會根據配接器架構，且用來處理特定傳輸通訊協定。管線元件用在任何傳送或接收管線階段。與運算質在對應中使用。  您可以透過自訂程式碼，以數種方式擴充 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]。 下列主題說明執行此作業的方法。
