# <a name="password-sync-adapters"></a>密碼同步配接器
A*密碼同步配接器*是從非 Windows 系統的密碼變更會傳播的元件。 雖然密碼同步配接器與傳統單一登入應用程式相似，但有數個差異：  
  
-   密碼同步配接器會使用特定介面進行管理。  
  
-   密碼同步配接器是使用組態存放區中的特定 XML 格式加以描述。  
  
-   Host Integration Server 會使用特定功能，在組態存放區中組織配接器。
