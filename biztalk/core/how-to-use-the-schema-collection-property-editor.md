# <a name="how-to-use-the-schema-collection-property-editor"></a>如何使用結構描述集合屬性編輯器
您可以使用結構描述集合屬性編輯器選取管線元件的結構描述。  
  
### <a name="to-use-the-schema-collection-property-editor"></a>使用結構描述集合屬性編輯器  
  
1.  在結構描述集合屬性編輯器 」 中，在**可用的結構描述**方塊中，選取應可供管線元件的結構描述。 使用 CTRL 鍵以選取一個以上的結構描述。  
  
2.  按一下 **[加入]**。  
  
     選取的結構描述現在會出現在**新增的結構描述**方塊。  
  
3.  按一下 **[確定]**。
