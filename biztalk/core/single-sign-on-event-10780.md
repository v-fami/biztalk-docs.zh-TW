# <a name="single-sign-on-event-10780"></a>單一登入： 事件 10780
## <a name="details"></a>詳細資料  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  產品名稱   |                 企業單一登入                  |
| 產品版本 | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    事件識別碼     |                           10780                            |
|  事件來源   |                           ENTSSO                           |
|    元件    |                            不適用                             |
|  符號名稱  |                  ENTSSO_E_INVALID_ACCOUNT                  |
|  訊息文字   |      帳戶名稱無效或不存在。      |
  
## <a name="explanation"></a>說明  
 帳戶名稱無效或不存在。  
  
## <a name="user-action"></a>使用者動作  
 檢查名稱和帳戶的拼字，並再試一次。
