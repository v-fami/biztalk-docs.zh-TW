# <a name="working-with-the-orchestration-debugger-view"></a>使用協調流程偵錯工具檢視
您可以存取協調流程偵錯工具檢視，以滑鼠右鍵按一下結果清單中的資料格，然後選取 協調流程偵錯工具。  
  
 「協調流程偵錯工具」會顯示單一執行個體的技術處理活動。 檢視會顯示繪製的協調流程影像。  
  
 從「協調流程偵錯工具」檢視，可移至「訊息流程」檢視。
