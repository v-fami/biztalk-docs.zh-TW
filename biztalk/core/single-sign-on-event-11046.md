# <a name="single-sign-on-event-11046"></a>單一登入： 事件 11046
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|企業單一登入|  
|產品版本|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|事件識別碼|11046|  
|事件來源|ENTSSO|  
|元件|不適用|  
|符號名稱|SSO_WARN_EXISTING_MAPPING_EXTERNAL|  
|訊息文字|無法建立對應，因為指定的外部帳戶已有對應存在。%r<br /><br /> 追蹤識別碼: %1 %r<br /><br /> 應用程式名稱: %2 %r<br /><br /> 外部帳戶： %3|  
  
## <a name="explanation"></a>說明  
 指定的外部帳戶的對應已經存在，而且不允許重複的項目。  
  
## <a name="user-action"></a>使用者動作  
 使用者不必採取任何動作。
