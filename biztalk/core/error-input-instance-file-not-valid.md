# <a name="error---input-instance-file-not-valid"></a>錯誤-輸入執行個體檔案無效
**說明**  
  
 指定的值**輸入執行個體檔案名稱**結構描述檔案的屬性不是有效的檔案名稱。  
  
 **使用者動作**  
  
 以滑鼠右鍵按一下相關的結構描述中的檔案 Microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]方案總管，然後按一下**屬性**開啟**屬性頁**此結構描述 對話方塊。 在此對話方塊，請在**一般**索引標籤上，按一下省略符號 (**...**) 中的值欄位按鈕**輸入執行個體檔案名稱**屬性，然後使用**選取輸入檔案**對話方塊來指定包含輸入執行個體訊息的檔案. 您也可以直接在值欄位中輸入檔案名稱**輸入執行個體檔案名稱**屬性。
