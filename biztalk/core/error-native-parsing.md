# <a name="error---native-parsing"></a>錯誤-原生剖析
**錯誤碼**  
  
 btm1041  
  
 **說明**  
  
 無法剖析針對「測試對應」作業指定的原生輸入執行個體訊息檔案，且發生指出的剖析錯誤。  
  
 **使用者動作**  
  
 視需要使用指出的剖析錯誤，更正指定檔案中，與對應關聯的來源結構描述或原生輸入執行個體訊息，或兩者。
