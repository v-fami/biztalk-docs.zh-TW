# <a name="how-to-set-pipeline-component-properties"></a>如何設定管線元件屬性
每個元件都有必須在使用前先設定的屬性。  
  
### <a name="to-set-pipeline-component-properties"></a>設定管線元件屬性  
  
-   在 [屬性] 視窗中**管線元件屬性**區段中，按一下**值**格**屬性**資料列，然後輸入新的值。 如果**值**儲存格包含箭號按鈕，按一下按鈕，選擇清單。 如果**值**資料格包含省略符號 (**...**) 按鈕，再按一下該按鈕可以使用結構描述集合屬性編輯器來設定屬性。 如需詳細資訊，請參閱[如何使用結構描述集合屬性編輯器](../core/how-to-use-the-schema-collection-property-editor.md)。
