# <a name="single-sign-on-event-10820"></a>單一登入： 事件 10820
## <a name="details"></a>詳細資料  
  
|                 |                                                                                              |
|-----------------|----------------------------------------------------------------------------------------------|
|  產品名稱   |                                  企業單一登入                                   |
| 產品版本 |                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                  |
|    事件識別碼     |                                            10820                                             |
|  事件來源   |                                            ENTSSO                                            |
|    元件    |                                             不適用                                              |
|  符號名稱  |                                ENTSSO_E_REQUIRE_OLD_PASSWORD                                 |
|  訊息文字   | 變更外部帳戶的密碼時，配接器必須提供舊密碼。 |
  
## <a name="explanation"></a>說明  
 此應用程式設定密碼同步配接器，才能提供舊密碼的方式。  
  
## <a name="user-action"></a>使用者動作  
 請檢查您的密碼同步配接器的組態。
