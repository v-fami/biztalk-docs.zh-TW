# <a name="how-to-open-the-functoid-toolbox"></a>如何開啟運算質工具箱
在建立對應並選取來源與目的結構描述之後，就可以將運算質放置在格線上。 運算質會顯示在 [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] 工具箱中，每個運算質類別都有一個工具箱索引標籤。  
  
### <a name="to-open-the-functoid-toolbox"></a>開啟運算質工具箱  
  
-   在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，BizTalk 對應開啟它的作用中 [編輯器] 視窗上**檢視**功能表上，按一下 **工具箱**。  
  
    > [!NOTE]
    >  您也可以使用**Ctrl + Alt + X**或**工具箱**工具**標準**顯示運算質工具箱工具列。  
  
    > [!IMPORTANT]
    >  若在工具箱看不到運算質索引標籤，請在主要編輯視窗中選取要編輯的 BizTalk 對應，以確定您已經開啟 [BizTalk 對應工具]。 如果仍看不到工具箱，請參閱[疑難排解 BizTalk Server 工具](../core/troubleshooting-biztalk-server-tools.md)。  
  
    > [!IMPORTANT]
    >  有時，當您修改工具箱的預設設定時，您在工具箱中看不到部分/全部運算質。 然後，在[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]，按一下 **工具**功能表，然後再按一下**選擇工具箱項目**。 在對話方塊中，移至**BizTalk 對應工具運算質** 索引標籤。請確定所有運算質已選取，然後按一下**確定**。  
    >   
    >  如果您仍然無法看到所有運算質，請參閱[疑難排解 BizTalk Server 工具](../core/troubleshooting-biztalk-server-tools.md)。
