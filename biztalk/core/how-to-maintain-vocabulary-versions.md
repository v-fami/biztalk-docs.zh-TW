# <a name="how-to-maintain-vocabulary-versions"></a>如何維護詞彙版本
當您已新增詞彙定義到詞彙版本時，可以將版本儲存至規則存放區以供進一步開發之用，或是您可以發佈版本以建立定義正確且不變的一組資料繫結詞彙，讓使用者在開發原則時可用以新增至規則。 請注意，現有規則將仍然指向舊版本。  
  
 本主題包含下列工作的程序：  
  
-   儲存詞彙版本  
  
-   發佈詞彙版本  
  
-   建立詞彙更新版本  
  
-   建立空的詞彙新版本  
  
### <a name="to-save-a-vocabulary-version"></a>儲存詞彙版本  
  
-   以滑鼠右鍵按一下版本，然後按一下**儲存**。  
  
### <a name="to-publish-a-vocabulary-version"></a>發佈詞彙版本  
  
-   以滑鼠右鍵按一下版本，然後按一下**發行**。  
  
### <a name="to-create-an-updated-version-of-a-vocabulary"></a>建立詞彙更新版本  
  
1.  以滑鼠右鍵按一下現有的版本，然後按一下**複製**。  
  
2.  以滑鼠右鍵按一下詞彙資料夾，然後**貼上**。  
  
     具有與複製版本相同項目的新版本已建立。  
  
### <a name="to-create-an-empty-new-version-of-a-vocabulary"></a>建立空的詞彙新版本  
  
-   以滑鼠右鍵按一下詞彙資料夾，然後**新增版本**。  
  
    > [!NOTE]
    >  您只能使用已發佈詞彙的詞彙定義。
