# <a name="interceptor-operations"></a>攔截器作業
本節包含一般攔截器組態檔作業的資訊。 這些作業適用於 Windows Workflow Foundation (WF) 和 Windows Communication Framework (WCF) 攔截器，以及任何以 Common Interceptor Framework 為基礎建置的其他攔截器。 WF 和 WCF 攔截器都會透過定義額外的網域專屬作業，以擷取和操作攔截器資料的方式，延伸這些運算子提供的一般功能。  
  
> [!NOTE]
>  作業名稱區分大小寫。
