# <a name="error---third-and-fourth-parameters-of-the-database-lookup-functoid-must-be-constants"></a>錯誤 - 資料庫尋查運算質的第三和第四個參數必須為常數
**錯誤碼**  
  
 btm1078  
  
 **說明**  
  
 第三個和第四個參數**資料庫尋查**運算質的資料表和資料行的名稱。 這些引數必須為常數值。  
  
 **使用者動作**  
  
 編輯運算質輸入的參數**設定運算質輸入**對話方塊。 按一下格線上的運算質。 在 [屬性] 視窗中，選取**輸入參數**屬性按一下省略符號 (**...**) 按鈕。 **設定運算質輸入**對話方塊隨即出現。
