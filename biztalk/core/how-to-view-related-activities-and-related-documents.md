# <a name="how-to-view-related-activities-and-related-documents"></a>如何檢視相關活動和相關文件
在 BAM Web 入口網站中，您可以透過 [商務活動搜尋結果] 頁面檢視相關活動和文件。 您可以從一個相關活動巡覽至另一個；例如，如果您搜尋到特定訂單，就能查看訂單上的多批出貨。 您也可以檢查相關文件；例如，就上述訂單而言，您可以查看相關的銷貨單。  
  
 如需搜尋活動的步驟，請參閱[如何搜尋活動](../core/how-to-search-for-activities.md)  
  
### <a name="to-view-related-activities"></a>檢視相關活動  
  
1.  在 BAM Web 入口網站中，搜尋特定活動。  
  
2.  在 [活動狀態] 頁面上，按一下加號 （+） 中的圖示**相關活動**內容框架的區域。  
  
     相關活動的清單便會出現。  
  
3.  按一下清單中的相關活動。  
  
### <a name="to-view-related-documents"></a>檢視相關文件  
  
1.  在 BAM Web 入口網站中，搜尋特定活動。  
  
2.  在 [活動狀態] 頁面上，按一下加號 （+） 中的圖示**相關文件**內容框架的區域。  
  
     相關文件的清單便會出現。
