# <a name="error---generic-native-serialization"></a>錯誤-一般原生序列化
**錯誤碼**  
  
 btm1049  
  
 **說明**  
  
 由於發生未指定的錯誤，無法將轉換 (由對應所指定) 產生的 XML 輸出執行個體訊息，轉換為目的結構描述指定的原生格式。  
  
 **使用者動作**  
  
 在 [BizTalk 編輯器] 中開啟目的結構描述，並使用**驗證結構描述**，**驗證執行個體**，和**產生執行個體**可用的命令，當您以滑鼠右鍵按一下在 [方案總管] 中，以隔離問題的結構描述。
