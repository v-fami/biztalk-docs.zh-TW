# <a name="error---property-schema-structure-not-valid"></a>錯誤-無效的屬性結構描述結構
**錯誤碼**  
  
 BEC2007  
  
 **說明**  
  
 此屬性結構描述具有無效的結構。 屬性結構描述的是限制為一或多個**欄位項目**節點的子系**結構描述**節點。  
  
 **使用者動作**  
  
 更正這個屬性結構描述的結構，讓**結構描述**節點只包含**欄位項目**做為子系的節點。
