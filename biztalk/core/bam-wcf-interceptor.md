# <a name="bam-wcf-interceptor"></a>BAM WCF 攔截器
BAM WCF 攔截器提供了在 WCF 服務內追蹤資料的完整支援。  
  
 您可以使用 BAM WCF 攔截器執行下列作業：  
  
-   明確將工作流程應用程式資料傳送到 BAM，而不需要重新編譯您的 WCF 應用程式。  
  
-   從 WCF 服務呼叫的參數和訊息中追蹤資訊。  
  
-   從 WCF 服務呼叫的訊息中追蹤資訊。  
  
-   參與來自用戶端的交易，或由 WCF 執行階段針對交易的服務呼叫而在內部啟始的交易。
