# <a name="how-to-set-receive-pipelines-for-tibco-enterprise-message-service"></a>如何設定 TIBCO Enterprise Message Service 的接收管線
Microsoft BizTalk Adapter for TIBCO Enterprise Message Service 需要您設定接收處理常式與接收管線。  
  
### <a name="to-set-the-pipeline"></a>若要設定管線  
  
1.  在**接收處理常式**清單中，選取**BizTalkServerApplication**。  
  
2.  在**接收管線**清單中，選取想要的管線。  
  
3.  按一下 **[確定]**。
