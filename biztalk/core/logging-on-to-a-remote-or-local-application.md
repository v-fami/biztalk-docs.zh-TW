# <a name="logging-on-to-a-remote-or-local-application"></a>登入的遠端或本機應用程式
完成為使用者設定分支機構憑證的作業之後，您可以使用「企業單一登入」服務 (ENTSSO) 提供應用程式的存取權。 若是本機使用者，您可以使用 ENTSSO 來擷取非 Windows 應用程式的憑證。 相反地，若是遠端使用者，則可以使用 ENTSSO 來擷取本機應用程式的憑證。
