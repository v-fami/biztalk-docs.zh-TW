# <a name="how-to-import-bindings-from-a-binding-file"></a>如何從繫結檔案匯入繫結
您可以使用繫結檔案匯入 BizTalk 應用程式的繫結至另一個現有的 BizTalk 應用程式。 您也可以匯入所有繫結群組的群組或應用程式組件的繫結。  
  
-   如需匯入繫結時，請記住的各種考量的資訊，請參閱[匯入繫結](http://go.microsoft.com/fwlink/?LinkId=155012)(http://go.microsoft.com/fwlink/?LinkId=155012)。  
  
-   如需匯入繫結至 BizTalk 應用程式的指示，請參閱[如何匯入到 BizTalk 應用程式的繫結](http://go.microsoft.com/fwlink/?LinkId=155013)(http://go.microsoft.com/fwlink/?LinkId=155013)。
