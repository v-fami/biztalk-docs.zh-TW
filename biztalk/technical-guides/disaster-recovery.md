# <a name="disaster-recovery"></a>災害復原
嚴重損壞修復程序所運用的步驟，以繼續作業失敗的系統，改善系統可用性。 災害復原與容錯功能的嚴重損壞修復通常需要手動介入來還原失敗的系統，而容錯的設計可以繼續操作不需要手動介入，如果系統發生失敗條件。
