# <a name="implementing-automated-testing"></a>實作自動化的測試
BizTalk Server 解決方案通常會部署在 「 任務關鍵性 」 案例中，藉此 BizTalk 解決方案是企業的核心元件。 在這些情況下，持續的效能與穩定性的 BizTalk 解決方案是重要的商務需求，如果 BizTalk 解決方案失敗，相關聯的停機時間成本皆屬於顯著。 在這種情況下，它最重要的是方案將放入實際執行之前經過徹底測試 BizTalk 解決方案。 相較於未測試或不足以測試解決方案所產生的停機時間成本與正確測試方案相關聯的成本都很小。 因此，BizTalk Server 解決方案的測試是說是任何 BizTalk Server 方案部署的最重要階段。  
  
 本章節描述在生產環境中執行方案之前，先測試 BizTalk 解決方案的適當方法。
