# <a name="configuring-the-source-system"></a>設定來源系統
基於 BizTalk Server 記錄傳送的目的，它並不重要如果來源系統位於單一[!INCLUDE[btsSQLServerNoVersion](../includes/btssqlservernoversion-md.md)]執行個體，或如果它分散裝載於 Windows Server 叢集中的多個執行個體。 沒有任何額外的考量以外，才能成功執行 「 備份 BizTalk Server 」 工作。 若要設定這項作業，請參閱[如何設定 「 備份 BizTalk Server 」 工作](http://go.microsoft.com/fwlink/?LinkID=154072)(<http://go.microsoft.com/fwlink/?LinkID=154072>) 在 BizTalk Server 說明中。 您已設定 「 備份 BizTalk Server 」 工作之後，繼續進行本主題[如何設定目的系統](../technical-guides/how-to-configure-the-destination-system.md)。
