# <a name="configuring-the-source-system"></a>設定來源系統
基於 BizTalk Server 記錄傳送，並不重要如果來源系統位於單一[!INCLUDE[btsSQLServerNoVersion](../includes/btssqlservernoversion-md.md)]執行個體，或如果它會分散於多個執行個體裝載於 Windows Server 叢集。 沒有以外成功執行 「 備份 BizTalk Server 」 工作所需的其他考量。 若要設定此作業時，請參閱[如何設定 「 備份 BizTalk Server 」 工作](http://go.microsoft.com/fwlink/?LinkID=154072)(http://go.microsoft.com/fwlink/?LinkID=154072) 在 BizTalk Server 說明中。 您已設定 「 備份 BizTalk Server 」 工作之後，繼續進行本主題[如何設定目的系統](../technical-guides/how-to-configure-the-destination-system.md)。
