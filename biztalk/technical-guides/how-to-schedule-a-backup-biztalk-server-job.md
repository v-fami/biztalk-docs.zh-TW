# <a name="how-to-schedule-a-backup-biztalk-server-job"></a>如何排程備份 BizTalk Server 作業
「備份 BizTalk Server」工作會依照 SQL Server Agent 服務的排程執行。 如果您想要建立較頻繁或較不頻繁的備份，您可以使用 SQL Server Enterprise Manager 來變更「備份 BizTalk Server」工作的排程。 如需如何排程備份 BizTalk Server 」 工作的詳細資訊，請參閱[如何排程 「 備份 BizTalk Server 」 工作](http://go.microsoft.com/fwlink/?LinkId=154674)(<http://go.microsoft.com/fwlink/?LinkId=154674>) 中[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]幫助。
