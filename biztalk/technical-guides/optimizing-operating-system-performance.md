# <a name="optimizing-operating-system-performance"></a>作業系統效能最佳化
本節提供在生產環境 BizTalk Server 環境中使用的 BizTalk Server 電腦的最佳化效能的建議。 安裝和設定 BizTalk Server 後，會套用此章節所列的最佳化。  
  
> [!IMPORTANT]  
>  本節中的建議事項的一些需要 Windows Server 登錄的修改。 不正確使用登錄編輯程式可能會造成問題，需要重新安裝作業系統。 您必須自行負擔使用「登錄編輯程式」的風險。 如需如何備份、 還原和修改登錄的詳細資訊，請參閱 Microsoft 知識庫文章[進階使用者的 Windows 登錄資訊](http://go.microsoft.com/fwlink/?LinkId=62729)(http://go.microsoft.com/fwlink/?LinkId=62729)。
