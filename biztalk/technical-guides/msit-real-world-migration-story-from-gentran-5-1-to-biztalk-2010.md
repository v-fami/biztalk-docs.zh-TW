# <a name="msit-real-world-migration-story-from-gentran-51-to-biztalk-2010"></a>MSIT： 真實世界移轉劇本 Gentran 5.1 從 BizTalk 2010
BizTalk 技術文章  
  
 **MSIT： 真實世界移轉劇本 Gentran 5.1 從 BizTalk 2010**  
  
 **作者：** Amit Kumar Dua、 Microsoft &#124; Banupriya Thirumaran Microsoft  
  
 **審稿者：** Mandi Ohlinger，Microsoft &#124; Nitin Mehrotra Microsoft  
  
 **參與者：** Nikhil Tayal，Microsoft &#124; Anil 錢德 Lingam Microsoft  
  
 **發行日期：** 2014 年 10 月  
  
 **適用於：** BizTalk Server 2010，Gentran 伺服器  
  
 **摘要：** Microsoft IT (MSIT) 整合平台會使用 BizTalk Server 和 Gentran。 直到最近，Microsoft IT 會有 Gentran 足跡良好與否。 BizTalk server 支援雲端運算和 AS2/EDI 功能之後，沒有機會 Gentran 取代 Microsoft BizTalk Server 2010。  Gentran 支援 Windows Server 2003 和 SQL server 2005。亦即延伸支援即將結束。 沒有任何以支援較新的平台，包括 Windows Server 2008/2012年和 SQL Server 2008/2014 Gentran 藍圖。  
  
 MSIT 會視為這有機會：  
  
-   取得用於較新的技術平台  
  
-   移除可支援性條件約束  
  
-   簡化的整合平台的整體架構  
  
-   請更符合成本效益且健全的整合平台  
  
 BizTalk Server 2010 是經過證實的整合平台、 具有許多功能和功能，並支援新的平台技術。  
  
 這份技術白皮書討論的策略和從 Gentran 5.1 移轉至 BizTalk Server 所採取的步驟。
