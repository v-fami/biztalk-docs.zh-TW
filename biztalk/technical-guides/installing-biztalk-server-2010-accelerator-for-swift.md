# <a name="installing-biztalk-server-2010-accelerator-for-swift"></a>安裝 BizTalk Server 2010 Accelerator for SWIFT
![標誌](../technical-guides/media/bts-10-installaccelerator-logo.gif "BTS_10_InstallAccelerator_Logo")  
  
 BizTalk 技術文章  
  
 **安裝 BizTalk Server 2010 Accelerator for SWIFT**  
  
 **作者：** Maria Quian，Microsoft Corporation  
  
 **技術校閱：** 安德列斯 Naranjo，Mandi Ohlinger  
  
 **發行日期：** 2012 年 6 月  
  
 **適用於：** BizTalk Server 2010 Accelerator for SWIFT  
  
 **摘要：** 成功安裝的 Microsoft BizTalk Server 2010 Accelerator for SWIFT (A4SWIFT) 的基礎是了解 A4SWIFT 功能以及每項功能所依存的 BizTalk 元件。 每個 A4SWIFT 功能取決於其他已正確設定的 BizTalk 元件。 會影響建置 A4SWIFT 環境的其他考量包括現有的網路基礎結構和已設定的 BizTalk Server 環境。 本文件將參考，並將現有的文件建置 BizTalk Server 環境。 A4SWIFT 繼承 BizTalk Server 2010 的所有軟硬體需求。  
  
 安裝指南適用於每個 A4SWIFT 相關的元件，但不是的單一編譯文件 elaborates 安裝/設定步驟。 本文件會將它們結合在一起，並識別某些安裝和設定期間遇到的常見問題。  
  
 若要檢閱文件，請下載[安裝 BizTalk Server 2010 Accelerator for SWIFT](http://go.microsoft.com/fwlink/?LinkId=255118) Word 文件。
