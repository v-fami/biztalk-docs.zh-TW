# <a name="how-to-add-a-reference-to-an-application"></a>如何將參考加入至應用程式
本主題提供使用 BizTalk Server 管理主控台，將參考加入至另一個應用程式的相關資訊。 當您想在目前應用程式中使用相同 BizTalk 群組內另一個應用程式的現有成品，便可以加入參考。 如果您想要使用 EDI 特有的結構描述、 管線和協調流程 （BizTalk EDI 應用程式中部署） 從另一個應用程式，例如 BizTalk Application 1 中，您必須加入 BizTalk EDI 應用程式的參考。 請勿將自訂成品加入至「BizTalk EDI 應用程式」。  
  
 如需詳細資訊：  
  
-   將參考加入至另一個應用程式，請參閱[如何加入另一個應用程式的參考](http://go.microsoft.com/fwlink/?LinkID=155011)(http://go.microsoft.com/fwlink/?LinkID=155011)。  
  
-   將參考加入至[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]EDI 應用程式，請參閱[如何將參考加入至 BizTalk Server EDI 應用程式](http://go.microsoft.com/fwlink/?LinkId=155026)(http://go.microsoft.com/fwlink/?LinkId=155026)。
