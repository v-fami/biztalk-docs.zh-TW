# <a name="proactivity-in-biztalk-server"></a>在 BizTalk Server proactivity
BizTalk 技術文章  
  
 **BizTalk Server 的主動性**  
  
 **作者：** Tord 高興 Nordahl (包威 AS)  
  
 **技術校閱：**有 Pereira (Devscope)、 Steef-jan Wiggers (Motion10)、 Erik Thue (包威 AS)、 Alexander Thue (包威 AS)、 Saravana Kumar (BizTalk360)、 Mandi Ohlinger (Microsoft)  
  
 **發行日期：** 2013 年 7 月  
  
 **適用於：** BizTalk Server  
  
 **摘要：**技術白皮書適用於從 BizTalk Server 2004 BizTalk Server 2013 環境。  目標是提供一些如何主動和解決，或避免之前發生, 的潛在問題的線索。 它適用於所有類型的 BizTalk 使用者，包括架構設計人員、 開發人員和系統管理員。 並非所有所述的案例可能會發生在您的環境，這份技術白皮書中未提及的某些情況下可能會發生。  
  
 由於本質和重要性的 BizTalk Server 中，設定和最佳作法可能會因環境。包括網路設定、 架構設計、 版本、 訊息流程和資源。 這份技術白皮書討論一般的安裝程式，並建議值。
