# <a name="supported-configurations"></a>支援的組態
一般而言，支援的組態[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]監視管理組件所述的下列位置：  
  
- [Operations Manager 2007 R2/2012年支援的組態](http://go.microsoft.com/fwlink/?LinkId=90676)  
  
- [Microsoft 支援週期原則](http://go.microsoft.com/fwlink/?LinkId=123820)  
  
  [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 監視管理組件支援下列的 BizTalk Server 版本和其相關聯的地區設定。  
  
|                                             版本                                              | 支援 |
|---------------------------------------------------------------------------------------------------|-----------|
| [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 2013 （英文） |    是    |
|                              所有其他的 BizTalk 版本和地區設定                               |    否     |
  
 BizTalk Server 管理組件，Operations Manager 2007 R2/2012 旨在僅監視[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]2013年。  
  
> [!NOTE]
>  在  [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，電子資料交換 (EDI) 是 BizTalk 應用程式。
