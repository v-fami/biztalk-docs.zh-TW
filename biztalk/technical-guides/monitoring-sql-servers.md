# <a name="monitoring-sql-servers"></a>監視 SQL Server

## <a name="use-sql-management-pack"></a>使用 SQL 管理組件
Microsoft SQL Server 管理組件提供主動式與反應式監視 SQL Server 的企業環境中。 可用性和組態監視，效能資料集合及預設臨界值會建立監視企業層級。 本機和遠端連接檢查，有助於確保資料庫的可用性。  
  
 與 SQL Server 管理組件中內嵌的專業技術，您可以主動管理 SQL Server，並找出問題變嚴重之前。 此管理組件會增加安全性、 可用性和效能的 SQL Server 基礎結構。  
  
 隨附的組件的 Microsoft SQL Server 管理組件指南描述管理組件的內容，並說明如何將其部署。 管理組件的功能包括：  
  
-   監視 SQL Server、 SQL Agent、 報表伺服器、 Notification Services 包含服務的狀態  
  
-   監視資料庫的狀態  
  
-   監視在資料庫中，可設定 %或 MB 的可用空間  
  
-   確保資料庫已正確設定  
  
-   確定用戶端可以連線至 SQL Server  
  
-   監視封鎖的處理序  
  
-   失敗的代理程式監看的工作，以及花了過多的時間來執行的工作  
  
-   監視複寫和警示失敗的健全狀況  
  
-   監視資料庫鏡像狀態
