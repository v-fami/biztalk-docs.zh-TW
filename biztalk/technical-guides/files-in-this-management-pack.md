# <a name="files-in-this-management-pack"></a>本管理組件的檔案
若要監視[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]，您必須先下載[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理組件[System Center 管理組件類別目錄](http://www.microsoft.com/download/details.aspx?id=39617)(<http://www.microsoft.com/download/details.aspx?id=39617>)。 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理組件包含下列檔案：  
  
|[檔案名稱]|顯示名稱|描述|  
|---------------|------------------|-----------------|  
|Microsoft.BizTalkServer2013.Library.mp|BizTalk Server 文件庫|包含泛型類別、 關聯性和其他管理組件建置組塊，可由其他管理組件提供監視所有的 BizTalk 應用程式。|  
|Microsoft.BizTalk.Server.2013.Monitoring.mp|BizTalk Server 監控|包含監視、 規則和檢視，會提供可延伸的方式來控制所有的 BizTalk 應用程式。|  
|Microsoft.BizTalk.Server.2013.Discovery.mp|BizTalk Server 探索|包含用來尋找 BizTalk 應用程式的各種元件的探索。|
