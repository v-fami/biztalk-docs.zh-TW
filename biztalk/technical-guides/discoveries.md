# <a name="discoveries"></a>探索
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理組件提供集中式的管理和 BizTalk Server 相關成品的監視體驗。  
  
## <a name="discovery-of-artifacts"></a>成品的探索  
 與[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理組件，監視代理程式探索的成品從單一執行階段的電腦。 根據預設，管理組件會探索已加入至 BizTalk 群組的第一部機器上的成品。 不過如果您想要探索您所選擇的電腦上的成品然後您必須使用覆寫功能來變更自動探索的設定。 若要使用覆寫來變更自動探索的設定，請參閱[的選擇性設定](../technical-guides/optional-configurations.md)。  
  
 動作和從 SCOM 作業主控台執行的工作特有的成品，系統會探索並監視所有的執行階段電腦上。 若要查看所有 BizTalk Server 管理組件提供的工作，請按一下**工作**Operations Manager 2007 R2/2012 Operations 主控台 [監視] 窗格中。
