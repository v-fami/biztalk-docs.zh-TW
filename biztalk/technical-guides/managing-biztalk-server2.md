# <a name="managing-biztalk-server"></a>管理 BizTalk Server
本節提供管理 BizTalk Server 系統在生產環境中的指引。 下列程序通常會假設您正在使用的 BizTalk 應用程式，在部署之後。 這些程序描述最佳做法與概念，包括下列的例行作業：  
  
- 檢查清單和 BizTalk 應用程式部署到生產環境的程序。  
  
- 檢查清單和更新 BizTalk 應用程式部署之後到生產環境的程序。  
  
- 程序來安裝、 設定和管理數位加密憑證。  
  
- 移動的程序[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]從一個 SQL Server 執行個體之間的資料庫。
