# <a name="how-to-view-management-pack-details"></a>如何檢視管理組件詳細資料
如需監視和相關聯的覆寫值的詳細資訊，檢視監視的知識。  
  
### <a name="to-view-knowledge-for-a-monitor"></a>若要檢視監視的知識  
  
1.  在 Operations 主控台中，按一下**製作** 按鈕。  
  
2.  展開**管理組件物件**，然後按一下 **監視器**。  
  
3.  在 [監視] 窗格中，展開目標，直到達到監視層級。 或者，您可以使用**搜尋**方塊尋找特定監視。  
  
4.  按一下監視，並在 [監視] 窗格中，按一下**檢視知識**。  
  
5.  按一下**產品知識** 索引標籤。
